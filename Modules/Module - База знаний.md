# Module - База знаний

**Пакет:** `ithive.knowledgebase`

## Описание

Электронное хранилище внутренней информации компании: документы, инструкции, памятки, шаблоны, реквизиты. Модуль выполняет функцию информационной поддержки, позволяя систематизировать материалы по тематикам, отделам, видам документов. Совместим с PHP 8.1+, MySQL, PostgreSQL. Поддерживает английский язык и многодепартаментность. Доступен через десктопный браузер и мобильное приложение Битрикс24 (без администрирования).

## Функциональность

- Создание и редактирование статей с вложениями (файлы, изображения, видео, презентации)
- Версионность статей (до 50 версий на статью, восстановление, удаление версий)
- Группировка статей по разделам с вложенностью
- Трёхуровневое управление доступом: модуль → раздел → статья
- Уведомления о новых материалах (по подразделениям)
- Публичные ссылки на статьи для внешних пользователей
- Комментирование статей с файлами и реакциями
- Статистика просмотров (intranet, extranet, mobile, за периоды)
- Поиск и фильтрация статей (по названию, описанию, дате, автору, тегам)
- Теги для категоризации статей
- Два визуальных редактора: упрощённый и расширенный HTML
- Стартовая страница с описанием и последними статьями
- Иконки для разделов (настраиваемые по уровням вложенности)
- Печать статей
- Мобильное приложение Битрикс24
- Установка/обновление/удаление через Marketplace
- Настройка отдельно для каждого сайта (департамента)
