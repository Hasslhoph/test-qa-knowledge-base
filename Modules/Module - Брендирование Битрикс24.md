# Module - Брендирование Битрикс24

**Пакет:** `ithive.changeportaldefaultheme`

## Описание

Модуль для стилизации корпоративного портала Битрикс24 под фирменный стиль компании. Позволяет изменить дизайн более 200 элементов без знания CSS. Поддерживает два интерфейса: Legacy и Зефир. Версия модуля: 6.1.0.

**Ключевые возможности:**
- Настройка страницы авторизации, виджетов, кнопок, ссылок, шапки портала
- Загрузка макетов из Figma через специальные плагины
- Экспорт/импорт настроек в ZIP
- Произвольные CSS-стили
- Многодепартаментность
