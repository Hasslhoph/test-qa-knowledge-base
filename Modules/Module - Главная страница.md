# Module - Главная страница

**Пакет:** `ithive.homepage`

## Описание

Модуль «Главная страница» добавляет на корпоративный портал две основные страницы: «Главная» и «О компании». «Главная» становится стартовой страницей портала, объединяя корпоративные сервисы, новости, виджеты и медиа-галерею. Страница «О компании» выполняет роль визитной карточки организации.
