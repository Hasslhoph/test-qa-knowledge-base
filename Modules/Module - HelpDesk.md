# Module - HelpDesk

**Пакет:** `ithive.helpdesk`

## Описание

Модуль для автоматизации работы внутренних сервисных служб корпоративного портала Битрикс24. Централизованный сбор обращений (тикетов), маршрутизация по веткам, назначение ответственных, SLA, аналитика. Версия: 6.0.0.

## Функциональность

- Универсальный мастер создания обращений (многошаговые формы, 8 типов полей)
- Гибкое назначение ответственных (по категории, подразделению, группе)
- Уровни поддержки SLA (расписание, приоритеты, время реакции, просрочки)
- Анонимные обращения (спецпользователь helpdesk.anonymous)
- Скрытые комментарии (для сотрудников ТП)
- Привязка задач Битрикс24 к обращениям (с v6.0.0)
- Оценка удовлетворённости при закрытии
- Аналитика: рабочий стол ТП, графики нагрузки
- Мобильное приложение Б24
- Многодепартаментность
