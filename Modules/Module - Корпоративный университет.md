# Module - Корпоративный университет

**Пакет:** `ithive.ipr`

## Описание

Модуль «Улей: Корпоративный университет» — инструмент для обучения и адаптации сотрудников. Позволяет осваивать и развивать профессиональные компетенции (hard и soft skills) без отрыва от производственной деятельности.

**Версия документации:** 16.0.0
