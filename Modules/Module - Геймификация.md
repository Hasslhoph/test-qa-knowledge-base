# Module - Геймификация

**Пакет:** `ithive.gamification`

## Описание

Модуль «Геймификация» добавляет на корпоративный портал игровые механики: рейтинги, уровни (ранги), игровую валюту, магазин бонусов. Повышает мотивацию и вовлеченность персонала через соревновательные элементы, систему баллов и поощрений.

**Версия:** 7.3.0 (Ноябрь 2025)

**Технические данные:**
- PHP 8.1+
- MySQL, PostgreSQL
- Поддержка английского языка
- Десктопный браузер
