# Лидеры продаж

**Модуль:** [[Module - Главная страница]]

## Описание

Раздел выполняет роль доски почета — выделяет сотрудников за достижения. Администраторы вручную добавляют сотрудников и указывают их заслуги.

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- **Доска почета** — ручное добавление сотрудников с указанием заслуг
- **Разделы (доски)** — группировка по кварталам, отделам, типам достижений
- **Настройки блока:** вывод подписей (сумма/описание), количество разделов, сортировка, ссылка на страницу
- **Карточка сотрудника:** фото, ФИО, должность, сумма сделок, детальное описание достижений
- **Номер** — место на доске почета (1 место, 2 место и т.д.)

## Предусловия

- Права администратора или пользователя с правами на редактирование раздела
- Включённая административная панель (режим правки)

## Шаги

### Создание раздела

1. Перейдите на страницу «Лидеры продаж» → кнопка «Элементы»
2. Нажмите «Добавить раздел»
3. Укажите название (например, «3 квартал 2025»)
4. Сохраните

### Добавление сотрудника

1. На странице «Лидеры продаж» → «Добавить сотрудника»
2. Вкладка «Параметры»:
   - Активность, Название, Сортировка
   - Номер (место на доске)
   - Пользователь (выбор сотрудника)
   - Сумма (успешных сделок)
   - Фотография (если не загружена — из профиля Битрикс24)
3. Вкладка «Подробно»: Детальное описание (достижения)
4. Вкладка «Раздел»: выберите один раздел
5. Сохраните

### Редактирование/удаление

1. В режиме правки наведите на профиль сотрудника → «Изменить» или «Удалить»
2. Или со страницы «Лидеры продаж» аналогичным способом

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- При удалении раздела удаляются все вложенные сотрудники (профили не стираются с портала)
- В блоке на Главной описание ограничено 2 строками, на странице — без ограничений
- При установке модуля «Геймификация» можно настроить начисление внутренней валюты при добавлении сотрудника
