# Оценка 360 - О продукте

**Модуль:** [[Module - Оценка 360]]

## Описание

Модуль «Оценка 360» автоматизирует оценку профессиональных качеств сотрудников по методу «360 градусов». Обратную связь о работе сотрудника предоставляют его руководитель, коллеги, подчиненные и он сам. Модуль помогает HR получать аналитику о сильных и слабых сторонах сотрудников.

## Ключевые функции

- **Универсальная модель компетенций** — создание системы компетенций (hard skills, soft skills) для всесторонней оценки
- **Гибкий механизм оценки** — положительная/отрицательная шкала, система баллов, оценка с индикаторами или без
- **Разные сценарии оценки** — 90°, 180°, 270°, 360°, 540° (с привлечением клиентов из CRM)
- **Оптимизация нагрузки** — анализ загруженности сотрудников оценками перед запуском
- **Открытые вопросы** — возможность добавить вопросы для получения развёрнутой обратной связи
- **Интеграция с Корпоративным университетом** — итоги оценки становятся основой для программ развития
- **Мобильное приложение** — оценка компетенций через мобильное приложение Битрикс24
- **BI-аналитика** — выгрузка данных для построения дашбордов в BI-конструкторе Битрикс24

## Технические данные

- Совместим с PHP 8.1 и выше
- СУБД: MySQL, PostgreSQL
- Поддерживает английский язык
- Поддерживает многодепартаментность
- Доступен через десктопный браузер и мобильное приложение Битрикс24
- Документация API: https://market.dev.wehive.digital/doxygen/assessment360/html/annotated.html

## Предусловия

- Модуль установлен и активирован на портале Битрикс24
- Настроены права доступа для HR-отдела и администраторов

## Сценарии использования

1. **Автономное использование** — компания проводит оценку, затем самостоятельно организует развитие компетенций
2. **Совместно с Корпоративным университетом** — итоги оценки становятся основой для запуска программы развития на портале

## Граничные случаи

- Оценку рекомендуется проводить не чаще двух раз в год (каждые 6-12 месяцев)
- Нельзя запустить новую оценку, пока не завершена предыдущая для того же сотрудника
- Если установлен модуль «Корпоративный университет», раздел «Компетенции» в Оценке 360 не отображается (управление через Корпоративный университет)
