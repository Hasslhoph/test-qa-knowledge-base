# Геймификация - Действия по рейтингам

**Модуль:** [[Module - Геймификация]]

## Описание

Инструкция по просмотру детальной информации о действиях сотрудников по автоматическим рейтингам. Позволяет понять, почему сотрудник занял то или иное место в рейтинге.

## Ключевые функции

- Просмотр действий доступен только по автоматическим рейтингам с включённым ежесуточным пересчётом
- Не доступен для ручных рейтингов и Общего рейтинга
- **Административная часть** — колонки: Действие, Количество, Значения (ID событий, суммы и т.д.)
- **Публичная часть** — слайдер с действиями сотрудника (без колонки «Значения»)
- Три опции отображения: в разделе «Рейтинги» (текущие участники), в блоке «Призеры предыдущих месяцев», в Ленте событий

## Предусловия

- Права администратора (для административной части)
- Включённая опция отображения действий в настройках рейтинга (для публичной части)

## Шаги (административная часть)

1. Перейдите в административную часть → Настройки → Улей: Геймификация
2. Вкладка «Рейтинги» → выберите автоматический рейтинг
3. Прокрутите к таблицам участников → наведите курсор на строку сотрудника → кликните на синюю стрелку
4. Изучите действия: тип действия, количество, значения (ID)

## Шаги (публичная часть)

1. В настройках рейтинга включите опции отображения детальной информации
2. В разделе «Рейтинги» наведите курсор на строку пользователя → кликните на синюю стрелку
3. Откроется слайдер с действиями сотрудника

## Ожидаемый результат

Пользователь видит детальные действия сотрудника по рейтингу и понимает расчёт позиции.

## Граничные случаи

- Колонка «Значения» выводится только в административной части
- Для ручных рейтингов список действий недоступен
- Для Общего рейтинга список действий недоступен (действия — это действия по всем остальным рейтингам)
- Если у автоматического рейтинга отключён пересчёт — действия не отображаются
