# HelpDesk - HelpDesk в мобильном приложении Б24

**Модуль:** [[Module - HelpDesk]]

## Описание

Использование HelpDesk в мобильном приложении Битрикс24. Требуется предварительная привязка разделов мастера к категориям для корректного отображения ответственных.

## Ключевые функции

### Первоначальная настройка
- Привязка разделов мастера к категориям (как в десктопной версии)
- Путь: `/mobile/helpdesk/` → двойной клик → настройки компонента → «Привязка разделов мастера к категориям»

### Доступные действия в мобильном приложении
- Просмотр обращений
- Создание обращений
- Закрытие, оценка и возобновление обращений
- Добавление наблюдателей
- Комментирование (без скрытых комментариев)

### НЕ доступно в мобильном приложении
- Управление правами доступа
- Переход в административную часть портала
- Скрытые комментарии

## Предусловия

- Привязка разделов мастера к категориям выполнена
- Мобильное приложение Битрикс24 установлено

## Граничные случаи

- Скрытые комментарии доступны только в десктопной версии
- Все административные действия — только через десктоп
