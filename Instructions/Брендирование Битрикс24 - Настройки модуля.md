# Настройки модуля

**Модуль:** [[Module - Брендирование Битрикс24]]

## Описание

Настройки модуля «Брендирование» находятся в административной части портала и позволяют управлять правами доступа к стилизации портала.

## Ключевые функции

### Доступ к настройкам

1. Перейти: `адрес_портала/bitrix/admin`
2. В шапке: «Настройки» → «Улей: Брендирование Битрикс24»
3. Откроется вкладка «Доступ»

### Управление правами

- Выберите нужную группу пользователей
- Установите право «Доступ к настройкам стилизации»
- Сохраните изменения
- Данная группа сможет вносить изменения в дизайн портала

### Многодепартаментность

- При наличии нескольких департаментов права можно задать:
  - Сразу для всех департаментов
  - Для каждого департамента отдельно (выбрать нужный сайт справа)

## Предусловия

- Модуль установлен
- Пользователь — администратор портала

## Граничные случаи

- По умолчанию доступ к настройкам модуля есть только у администраторов
- Права на настройку стилизации можно делегировать другим группам пользователей
- В многодепартаментной конфигурации права настраиваются отдельно для каждого сайта
