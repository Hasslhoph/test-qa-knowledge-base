# База знаний - Упрощенный и расширенный редактор

**Модуль:** [[Module - База знаний]]

## Описание

Два визуальных редактора для создания и редактирования статей: упрощённый (аналог редактора задач Битрикс24) и расширенный HTML-редактор (стандартный редактор Битрикс24 с поддержкой HTML-вёрстки).

## Ключевые функции

- Упрощённый редактор: базовое форматирование, ссылки, таблицы, код, видео, изображения, прикрепление файлов
- Расширенный HTML-редактор: HTML-вёрстка, iFrame для встраивания контента с других сайтов
- Конвертация из упрощённого в расширенный редактор (необратимо)
- Настройка режима редактора по умолчанию в административной части

## Предусловия

- Права на создание/редактирование статей

## Шаги

### Упрощённый визуальный редактор

1. Создайте или откройте статью
2. Используйте панель инструментов для форматирования текста
3. Прикрепляйте файлы через соответствующую кнопку (только в упрощённом редакторе)

### Расширенный HTML-редактор

1. Создайте или откройте статью
2. Используйте стандартные инструменты форматирования и HTML-вёрстку
3. Для вставки iFrame: используйте HTML-код элемента

### Настройка iFrame

1. Перейдите: `адрес_портала/bitrix/admin/security_filter.php?lang=ru`
2. Откройте вкладку «Исключения»
3. В поле «Маски исключения» укажите: `/knowledgebase/`
4. Сохраните изменения

### Конвертация статьи

1. В правом верхнем углу статьи нажмите переключатель конвертации
2. Проверьте содержимое статьи
3. Сохраните изменения (необратимо)

## Ожидаемый результат

Статьи создаются и редактируются в выбранном редакторе.

## Граничные случаи

- Только упрощённый редактор позволяет прикреплять файлы
- Расширенный редактор не имеет опции прикрепления файлов (используйте ссылки на диск)
- Конвертация необратима — после сохранения нельзя вернуться к упрощённому редактору
- Расширенный редактор нельзя конвертировать в упрощённый
- Прикреплённые файлы не пропадают при конвертации, но удалить их после конвертации невозможно (только через диск Битрикс24)
- Для использования iFrame нужно внести Базу знаний в исключения модуля «Проактивная защита»
