# Оценка 360 - Оценка компетенций

**Модуль:** [[Module - Оценка 360]]

## Описание

Процесс оценки компетенций респондентами. Когда оценка запущена, всем оценивающим приходит уведомление. Карточки появляются в разделе «Я оцениваю» со статусом «Ожидание».

## Ключевые функции

### Обзор карточки оцениваемого
1. Имя и должность оцениваемого
2. Сроки проведения оценки (период для оценки, можно менять ответы)
3. Описание шкалы (что означают баллы)
4. Компетенции и индикаторы с инструментами оценки
5. Открытые вопросы (если добавлены)
6. Поле для комментария
7. Возможность пригласить коллег (если включено)

### Оценка компетенций
- По шкале: выставление баллов по шкале
- По индикаторам: выбор проявленных индикаторов
- Зависит от способа оценки, выбранного в настройках компетенции

### Описание компетенций и индикаторов
- Навести курсор на иконку «?» для просмотра описания
- Если включено в настройках — описание выводится целиком в карточке
- При скрытой балльной оценке описание индикаторов выводится в форме выбора ответа

### Ответы на открытые вопросы
- Текстовые поля после списка компетенций
- Обязательные вопросы помечены красной звёздочкой
- Без ответа на обязательный вопрос нельзя сохранить оценку

### Отсутствие оценки
- Можно воздержаться от оценки компетенции (если включено в настройках)
- Ограничение: нельзя оценить меньше 85% компетенций (настраивается)

### Комментарий к компетенциям
- Комментарий к каждой компетенции
- Может быть обязательным (настраивается в модуле)
- По умолчанию обязателен при оценке 5 и ниже, а также 9 и выше

### Сохранение карточки
- Кнопка «Оценить» → статус «Заполнена»
- Кнопка «Отказ» → статус «Отказ»
- Можно изменить решение до окончания срока оценки

## Предусловия

- Оценка запущена и пользователь указан как оценивающий
- Получено уведомление или карточка видна в разделе «Я оцениваю»

## Граничные случаи

- Карточка исчезает из «Я оцениваю» по истечении срока
- Нельзя оценить меньше 85% компетенций (настраивается)
- При скрытой балльной оценке респонденты не видят баллы за индикаторы
