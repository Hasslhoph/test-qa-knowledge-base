# Установка, обновление и удаление модуля

**Модуль:** [[Module - Главная страница]]

## Описание

Инструкция по установке, обновлению и удалению модуля «Главная страница» (ithive.homepage), а также активации купонов.

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- **Установка модуля** через административную часть
- **Обновление модуля** через Marketplace
- **Активация купона** на покупку/продление
- **Активация купона** на неустановленный модуль
- **Активация купона** при истёкшей демоверсии
- **Удаление модуля** через Marketplace

## Предусловия

- Права группы пользователей «Администраторы»

## Шаги

### Установка модуля

1. Перейдите: `адрес_портала/bitrix/admin/update_system_partner.php?addmodule=ithive.homepage`
2. Модуль будет автоматически отмечен галочкой
3. Нажмите «Загрузить» → подтвердите Лицензионное соглашение → «Применить»
4. Дождитесь завершения установки
5. Уведомление об успешной загрузке = установка завершена

### Обновление модуля

1. Marketplace → «Обновление решений» → «Список обновлений»
2. Выберите «Улей: Главная страница» → «Загрузить»
   Или сразу: `адрес_портала/bitrix/admin/update_system_partner.php`

### Активация купона

1. Marketplace → «Обновление решений» → «Активация купона»
2. Введите номер купона → «Активировать купон»
   Или: `адрес_портала/bitrix/admin/update_system.php?lang=ru`

### Активация купона на неустановленный модуль

1. Marketplace → «Установленные решения» → «Купленные решения»
2. Бургер-меню рядом с модулем → «Загрузить»

### Удаление модуля

1. Marketplace → «Установленные решения» → «Доступные решения»
2. Бургер-меню рядом с решением → «Удалить»
3. Подтвердите удаление после обновления страницы

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- Все операции доступны только группе «Администраторы»
- При активации купона для модуля с истёкшей демоверсией: «Обновление решений» → «Установка обновлений» → «Зарегистрировать решение»
