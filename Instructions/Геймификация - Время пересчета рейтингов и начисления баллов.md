# Геймификация - Время пересчета рейтингов и начисления баллов

**Модуль:** [[Module - Геймификация]]

## Описание

Инструкция по изменению времени и интервала автоматического пересчёта рейтингов и начисления баллов за различные события через настройку агентов Битрикс24.

## Ключевые функции

- **Общая настройка времени пересчёта** — единое время для всех рейтингов в настройках модуля (по умолчанию 00:00)
- **Индивидуальная настройка через агентов** — время и интервал для каждого рейтинга отдельно
- **Агенты рейтингов** — `AgentRating("rating_slug", true)` для каждого из 6 рейтингов
- **Агент начисления баллов за призовые места** — `AgentSumPayPoints()` (по умолчанию 1-го числа каждого месяца, интервал 2592000 сек = 1 месяц)
- **Агент начисления баллов «Спасибо»** — `setThxBonuses()` (по умолчанию 1-го числа каждого месяца)
- **Интервал по умолчанию** — 86400 секунд (24 часа) для пересчёта рейтингов

## Предусловия

- Права администратора портала
- Доступ к агентам Битрикс24

## Шаги (индивидуальная настройка пересчёта рейтинга)

1. Перейдите в `/bitrix/admin/` → Настройки → Настройки продукта → Агенты
2. В поле «Модуль» введите `ithive.gamification`
3. Найдите нужный агент рейтинга:
   - `effectivetaskcreators` — Эффективный постановщик задач
   - `effectivetaskresponsibles` — Эффективный исполнитель задач
   - `activeiprlearning` — Активное развитие
   - `salesleaders` — Лидер продаж
   - `effectivecalendarworkers` — Эффективный пользователь календаря
   - `total` — Общий рейтинг
4. Нажмите «Изменить»
5. Укажите «Дата и время следующего запуска» и «Интервал (сек.)»
6. Сохраните

## Шаги (настройка начисления баллов за призовые места)

1. Найдите агент `AgentSumPayPoints()`
2. Измените дату/время и интервал (по умолчанию 2592000 сек = 1 месяц)

## Шаги (настройка начисления баллов «Спасибо»)

1. Найдите агент `setThxBonuses()`
2. Измените дату/время и интервал (по умолчанию 2592000 сек = 1 месяц)

## Ожидаемый результат

Рейтинги пересчитываются и баллы начисляются по заданному расписанию.

## Граничные случаи

- Агенты — базовая технология Битрикс24
- С помощью даты и времени можно задать конкретный день недели
- Интервал задаётся в секундах
- Изменение интервала влияет на периодичность начиная с указанной даты
