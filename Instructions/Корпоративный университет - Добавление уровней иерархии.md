# Добавление уровней иерархии

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Создание дополнительных уровней иерархии для позиций на схеме карьерных траекторий. По умолчанию предусмотрено 10 уровней.

## Ключевые функции

### Уровень в иерархии

- Опция на слайдере создания позиции
- 10 уровень — позиция в самом верху схемы
- Размещение позиции на нужном уровне иерархии

### Добавление уровней

1. Административная часть → «Контент» → «Корпоративный университет» → «Должности»
2. Настройки информационного блока → вкладка «Свойства»
3. Три точки (...) у свойства «Уровень в иерархии»
4. Добавить новые уровни → сохранить
5. Сохранить страницу настроек инфоблока

### Результат

- Новые уровни доступны на слайдере создания/редактирования позиции
