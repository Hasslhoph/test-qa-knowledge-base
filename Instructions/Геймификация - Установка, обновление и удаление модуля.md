# Геймификация - Установка, обновление и удаление модуля

**Модуль:** [[Module - Геймификация]]

## Описание

Инструкция по установке, обновлению и удалению модуля «Геймификация» (ithive.gamification) на корпоративном портале Битрикс24.

## Ключевые функции

- **Установка модуля** — через ссылку `/bitrix/admin/update_system_partner.php?addmodule=ithive.gamification`; принятие лицензионного соглашения; автоматическая загрузка
- **Обновление модуля** — через Marketplace → Обновление решений → Список обновлений; или по прямой ссылке
- **Активация купона** — через Marketplace → Обновление решений → Активация купона; или для неустановленного модуля через Установленные решения → Купленные решения
- **Активация купона для модуля с истёкшей демо-версией** — через «Зарегистрировать решение»
- **Удаление модуля** — через Marketplace → Установленные решения → Доступные решения → Удалить

## Предусловия

- Права группы «Администраторы» (установка, обновление, удаление, активация купона)
- Доступ к административной части портала

## Шаги (установка)

1. Перейдите по ссылке: `адрес_портала/bitrix/admin/update_system_partner.php?addmodule=ithive.gamification`
2. Модуль автоматически отмечен галочкой → нажмите «Загрузить»
3. Примите Лицензионное соглашение и Условия конфиденциальности → «Применить»
4. Дождитесь завершения установки
5. Убедитесь в уведомлении об успешной загрузке

## Шаги (обновление)

1. Marketplace → Обновление решений → Список обновлений
2. Выберите «Улей: Геймификация» → «Загрузить»

## Шаги (удаление)

1. Marketplace → Установленные решения → Доступные решения
2. Бургер-меню рядом с решением → «Удалить»
3. Подтвердите удаление

## Ожидаемый результат

Модуль установлен/обновлён/удалён. При успешной операции появляется соответствующее уведомление.

## Граничные случаи

- Все операции требуют прав администратора
- При активации купона на неустановленный модуль — сначала загрузите модуль из «Купленные решения»
- При истёкшей демо-версии — используйте «Зарегистрировать решение»
