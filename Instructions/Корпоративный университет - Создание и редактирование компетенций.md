# Создание и редактирование компетенций

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Создание, редактирование и копирование компетенций в Корпоративном университете.

## Ключевые функции

### Создание компетенций

1. Раздел «Компетенции» → «Настройки» (⚙️) → «Добавить»
2. Заполнить поля:
   - **Активность** — отображение в публичных интерфейсах
   - **Наименование** — название компетенции (обязательное)
   - **Значок** — иконка из библиотеки Awesome v.6 (2000+ вариантов)
   - **Цвет** — цвет иконки компетенции
   - **Раздел** — категория (можно привязать к нескольким)
   - **Сортировка** — очередность (меньший индекс = первая)
   - **Описание** — основная информация
   - **Картинка** — иллюстрация в карточке компетенции
   - **Развивающие действия** — обязательные и рекомендуемые действия

### Развивающие действия для компетенций

- Мероприятия
- Курсы
- Книги (модуль «Библиотека»)
- Развитие на рабочем месте
- Шаблоны задач

**Обязательные** — автоматически попадают в карточку развития
**Рекомендуемые** — добавляются по желанию

### Копирование компетенции

- Бургер-меню (≡) → «Копировать»

### Редактирование компетенции

- Из списка компетенций → бургер-меню (≡) → «Изменить»
- Из карточки компетенции → «Редактировать» (⚙️)

## Предусловия

- Права: администраторы и группа HR-отдела
- Созданные категории компетенций

## Граничные случаи

- При установленном модуле «Оценка 360» доступен интерфейс для добавления индикаторов
- Если название отредактировано после добавления в карточку развития — название изменится и в карточке
- Новые развивающие действия, добавленные после создания карточки, не появляются автоматически
- В ранних версиях были поля «Уровень», «Компетенции предыдущего/следующего уровня» — их можно деактивировать
