# Настройка уведомлений для отдела продаж

## Описание
Инструкция по настройке уведомлений для сотрудников отдела продаж в Битрикс24.

## Шаги
1. Перейти в Настройки модуля Уведомления
2. Выбрать отдел Продажи
3. Настроить шаблоны уведомлений
4. Сохранить

## Ожидаемый результат
Сотрудники отдела продаж получают уведомления о новых лидах и сделках.
