# Настройки модуля

**Модуль:** [[Module - Управление целями. Каскад]]

## Описание

Основные настройки модуля выполняются в публичной и административной части портала. Доступ только у администраторов модуля.

## Ключевые функции

> Конфигурация модуля через 4 вкладки настроек.

- **Общие** — права доступа, параметры целей и процессов
- **Уведомления** — тексты и каналы отправки уведомлений
- **Права** — администраторы модуля и сотрудники внутреннего контроля
- **Функ. руководитель** — функциональная подчиненность

## Настройки в публичной части

### Вкладка «Общие»

1. **Минимальный вес цели** — минимальный приоритет (по умолчанию 10)
2. **Выделяемое время для задачи** — время на заполнение карточки (в минутах)
3. **Максимальное количество целей в карточке** — лимит целей (каскадированные входят)
4. **Минимальное количество дней до окончания процесса** — запрет запуска при малом остатке
5. **Включить возможность каскадирования целей** — автоматическое каскадирование
6. **Название колонки «Цель / Задача»** — переименование поля
7. **Название колонки «Критерии выполнения»** — переименование поля
8. **Название колонки «Комментарии»** — переименование поля
9. **Исключить стадии согласований карточек целей** — выбор активных стадий
10. **Автоматическая задача новому сотруднику** — задача на составление карточки
11. **Исключить стадии согласований карточек оценок** — выбор активных стадий

### Вкладка «Уведомления»

Каналы отправки:
- На почту
- На портале (мессенджер Битрикс24)

Настраиваемые уведомления:
- Сотруднику о составлении целей
- Администратору об утверждении цели
- Руководителю/директору о согласовании цели
- Функциональному руководителю о согласовании цели
- О завершении периода заполнения карточки цели
- Сотруднику о заполнении самооценки
- Администратору об утверждении самооценки
- Руководителю о согласовании самооценки
- Функциональному руководителю о согласовании самооценки
- О промежуточном заполнении результатов оценки
- Делегату о назначении/удалении
- Сотруднику об утверждении карточки руководителя
- Проверка актуальности целей
- Уведомление о зависимости целей от подразделения

### Вкладка «Права»

- Группа администраторов модуля
- Сотрудники внутреннего контроля

### Вкладка «Функ. руководитель»

- Выбор функциональных руководителей
- Указание подчиненных для каждого

## Настройки в административной части

Путь: `адрес_портала/bitrix/admin/settings.php?lang=ru&mid=ithive.goalsmanagement`

Те же 4 вкладки + дополнительные настройки во вкладке «Общие»:

1. **Настройка доступа к публичной части модуля** — группы, отделы, сотрудники для каждого сайта
2. **Настройки инфоблоков** — хранилища данных (заполняются автоматически, не рекомендуется менять):
   - Управление целями — Каталог целей
   - Управление целями — Настройки процессов
   - Управление целями — Карточка целей сотрудника
   - Управление целями — Карточка оценки сотрудника

## Предусловия

- Роль администратора модуля

## Граничные случаи

- Поля с инфоблоками заполняются автоматически — не рекомендуется менять
- Все настройки административной части дублируют публичную часть
- Для выбора нескольких элементов зажать Ctrl (⌘ Command на Mac)
