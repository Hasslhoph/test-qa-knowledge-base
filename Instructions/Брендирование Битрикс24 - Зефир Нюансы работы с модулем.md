# Зефир. Нюансы работы с модулем

**Модуль:** [[Module - Брендирование Битрикс24]]

## Описание

Статья описывает важные нюансы работы с модулем «Брендирование» в дизайне «Зефир» (после intranet 25.900.0). Рассмотрены два способа работы, условия видимости дизайна и специфичные особенности интерфейса Зефир.

## Ключевые функции

### Два способа работы

1. **Настройка стилей в модуле** — ручная настройка каждого параметра. Элементы, доступные **только через модуль**:
   - Спрятать блок «Скачать приложения»
   - Спрятать блок «ПУЛЬС КОМПАНИИ»
   - Загружаемые элементы: картинка фона, логотип, фавикон
   - Все настройки блока «Контент»
   - Все настройки блока «Другое»
   - Некоторые настройки шапки портала

2. **Создание дизайна в Figma** — использование конструктора [\[S\] Конструктор "Zephir"](https://www.figma.com/design/zLAHet339UCTc0Vr9OE9Yt/-S—%D0%91%D1%80%D0%B5%D0%BD%D0%B4%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5-%D0%9124—%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D1%80%D1%83%D0%BA%D1%82%D0%BE%D1%80-%22Zephir%22?node-id=1-107&t=ogf9m1oHGTFAWDin-0)

### Условия видимости дизайна

- Работает только со **светлыми темами** («Светло-серый узор», «Цветная бумага», «Роса» и др.)
- Светлая тема = чёрный шрифт на портале. Белый шрифт = тёмная тема (стили не выводятся)
- Исключения (видны всегда): раздел «Авторизация», фавикон, логотип

### Специфичные особенности Зефир

- **Настройки виджетов**: общие настройки не применяются (в доработке). Стилизация только через индивидуальные настройки
- **Цвет левого меню**: автоматически подстраивается под цвет фона страницы. Можно настроить отдельно в разделе «Шапка портала» → «Левое меню» — тогда цвет перестанет меняться при смене фона
- **Тема «Светло-серый узор»**: имеет серый фоновый рисунок, который виден даже при изменении цвета фона

### Возврат стандартных настроек

1. **Отключить модуль** — настройки сохраняются
2. **Сбросить все настройки** — полная очистка (рекомендуется предварительно экспортировать)

## Предусловия

- Модуль установлен и активен
- Светлая тема портала у пользователя

## Граничные случаи

- На тёмных темах стили не выводятся (кроме авторизации и загружаемых элементов)
- Общие настройки виджетов не работают — использовать индивидуальные или CSS
- При задании цвета фона левое меню автоматически окрашивается в похожий оттенок
- У темы «Светло-серый узор» фоновый рисунок остаётся видимым при кастомизации цвета
