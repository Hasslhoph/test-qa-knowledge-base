# Создание курса

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Создание и редактирование онлайн-курсов в Корпоративном университете с поддержкой двух визуальных редакторов.

## Ключевые функции

### Два визуальных редактора

1. **Сайты 24** (по умолчанию) — конструктор с готовыми адаптивными шаблонами
2. **HTML-редактор** — требует знаний HTML и CSS

⚠️ Курс редактируется только в том редакторе, в котором был создан

### Создание курса

- Раздел «Онлайн-курсы» → шестерёнка (⚙️) → «Добавить курс»

### Редактирование курса

- Поля:
  - **Название курса** — с ключевыми словами
  - **Краткое описание** — выводится в списке курсов
  - **Категории** — для структурирования и фильтрации
  - **Индекс сортировки** — положение в списке
  - **Время на изучение** — длительность в часах
  - **Дата создания** — метка «Новинка» в течение месяца
  - **Теги** — для поиска
  - **Подробное описание** — полная информация о курсе

### Настройка прав доступа

- Общие права — наследуются от настроек модуля «Обучение»
- Частные права — применяются к конкретному курсу
- Уровни доступа:
  - Доступ закрыт
  - Просмотр
  - Редактирование/удаление/создание
  - Привязывание/отвязывание
  - Полный доступ

## Предусловия

- Права: администраторы, HR-отдел, авторы курсов, пользователи с правами на курс

## Граничные случаи

- При включённом редакторе Сайты24 нельзя создавать курсы через административную часть
- Частные права не могут ограничивать общие, но могут расширять
