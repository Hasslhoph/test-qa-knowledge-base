# Обзор раздела «Компетенции»

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Раздел со списком компетенций, где любой сотрудник может увидеть свой текущий уровень владения той или иной компетенцией.

## Ключевые функции

### Где находится раздел

- Пункт меню верхнего раздела: «Компетенции»

### Уровни владения компетенциями

- **Серая галочка** — пользователь не владеет в достаточной мере
- **Синяя галочка** — пользователь владеет в достаточной мере для работы
- **Синяя академическая шапочка** — пользователь может делиться знаниями с коллегами

### Глубина вложенности разделов

- Вложенность разделов неограниченная
- Рекомендуемый уровень вложенности — не более четырёх уровней
- Использование более четырёх вложенных разделов может сказаться на адаптивности

### Наследование цвета

- Каждый раздел обладает своим цветом (настраивается в административной части)
- Цвет раздела первого уровня передаётся всем последующим разделам и компетенциям
- Без указания цвета — серый цвет по умолчанию

### Обязательная компетенция «Саморазвитие»

- Находится в разделе «Корпоративные компетенции»
- Нельзя изменить, удалить, переместить или деактивировать
- Нужна для запуска:
  - Программы развития со свободным выбором компетенций
  - Саморазвития при выборе книги из Библиотеки
  - Саморазвития при выборе онлайн-курса
  - Назначения онлайн-курсов и тестов

## Предусловия

- Доступно всем пользователям

## Граничные случаи

- При деактивации раздела «Корпоративные компетенции» компетенция «Саморазвитие» остаётся активной
- При удалении раздела компетенция автоматически переносится в раздел выше
