# Мероприятия

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Мероприятия — один из шести типов развивающих действий: тренинги, лекции, презентации. Раздел состоит из вкладок: «Список», «Календарь», «Предложения».

## Ключевые функции

### Вкладка «Список»

- Справочник шаблонов будущих мероприятий
- Связь шаблонов с компетенциями (обязательные / рекомендуемые)
- Шаблон ≠ дата проведения (шаблон — карточка, дата — событие в календаре)

### Вкладка «Календарь»

- Таблица календаря с мероприятиями КУ
- Только запланированные через интерфейс модуля

### Добавление мероприятий

- Права: администраторы, HR-отдел, модераторы
- Поля: Активность, Название, Описание, Дата проведения, Продолжительность, Напоминание, Организатор, Место проведения, Категория, Компетенции

### Предложенные мероприятия

- Любой сотрудник может предложить мероприятие
- Поля «Ссылка» и «Стоимость»
- Заявка отправляется на модерацию

### Модерация предложений

- Уведомление модераторам
- Через гиперссылку в уведомлении или вкладку «Предложения»
- По умолчанию галочка «Активность» отключена

### Категории мероприятий

- Привязка категории к календарю компании
- Три типа календарей: компании, групп, пользователей
- Фильтрация мероприятий в календаре Битрикс24

### Участие в мероприятии

1. Организатор вручную добавил в список участников
2. Есть активная карточка развития с мероприятием — автоматическое участие

## Предусловия

- Права на добавление: администраторы, HR-отдел, модераторы

## Граничные случаи

- При удалении шаблона события календаря НЕ удаляются
- При удалении категории удаляются все вложенные мероприятия
