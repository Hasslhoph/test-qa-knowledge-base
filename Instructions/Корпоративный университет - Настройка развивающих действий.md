# Настройка развивающих действий в карточке развития

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Настройка отображения и уведомлений для 6 типов развивающих действий в карточке развития.

## Ключевые функции

### Где находится настройка

- Административная часть → «Контент» → «Корпоративный университет» → «Типы развивающих действий»

### Настройка отображения

1. **Активность** — отображение в карточке (у онлайн-курсов нельзя отключить)
2. **Подсказка** — текст подсказки при клике на иконку с вопросительным знаком
3. **Описание в заголовке** — заголовок над сроком выполнения
4. **Название кнопки** — переименование кнопки выбора действия

### Настройка уведомлений о дедлайне

1. **Отправлять уведомления** — «Да» / «Не выбрано»
2. **Кол-во дней** — за сколько дней до дедлайна уведомлять
3. **Шаблон уведомления** — текст уведомления

⚠️ Перед настройкой отключить опцию «Использовать общий шаблон уведомлений для всех типов» в настройках модуля
