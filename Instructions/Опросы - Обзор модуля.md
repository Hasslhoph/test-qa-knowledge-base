# Обзор модуля

**Модуль:** [[Module - Опросы]]

## Описание

Обзор интерфейса модуля «Опросы» для обычных и привилегированных пользователей, а также описание статусов опросов.

## Ключевые функции

### Интерфейс для всех пользователей

- Главная страница модуля — список всех доступных опросов
- Фильтр: действующие / завершенные опросы
- Строка поиска по названию опроса
- Кнопка «Голосовать» — для непройденных опросов
- Кнопка «Результаты» — после прохождения опроса (можно скрыть в настройках)

### Интерфейс для привилегированных пользователей (администраторы и редакторы)

- Кнопка «Настройки» (⚙️) — добавление нового опроса и управление существующими
- Кнопка настроек рядом с каждым опросом: Деактивировать, Редактировать, Обнулить голоса, Удалить
- Фильтр «Неактивные» — отображение опросов с отключённой активностью или с будущей датой начала

### Статусы опросов

1. **Активен** — срок начался и не завершился. Пользователи видят опрос, могут голосовать и смотреть результаты (если включено)
2. **Завершен** — срок завершился, голосование закрыто. Обычные пользователи не могут взаимодействовать, привилегированные могут редактировать
3. **Неактивен** — опрос деактивирован или дата начала ещё не наступила. Видно только администраторам и редакторам

## Предусловия

- Модуль установлен
- Пользователь авторизован на портале

## Граничные случаи

- Если в настройках опроса скрыты результаты, после голосования кнопка «Результаты» не отображается у обычных пользователей
- Неактивные опросы видны только привилегированным пользователям
