# Мои результаты тестирования

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Просмотр результатов попыток прохождения тестов курсов.

## Ключевые функции

### Где посмотреть результаты

1. Раздел «Онлайн-курсы» → «Мои результаты тестирования»
2. Ссылка «Мои результаты тестирования» после завершения теста

### Таблица с результатами

| Колонка | Описание |
|---------|----------|
| Курс | Название курса (кликабельная ссылка) |
| Тест | Название теста |
| Результат последнего тестирования | Набранные баллы |
| Последний тест пройден | да/нет |
| Лучший результат | Максимальный балл |
| Пройден | Общий статус (достаточно одной успешной попытки) |
| Попыток | Количество попыток (кликабельная ссылка) |
| Действия | Ссылка для повторного прохождения |

## Предусловия

- Заполненные поля «Автоматическая проверка результатов» и «Для прохождения теста необходимо набрать не менее» при создании теста
