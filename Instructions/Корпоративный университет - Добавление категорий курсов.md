# Добавление категорий курсов

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Управление категориями курсов для фильтрации и структурирования в разделе «Онлайн-курсы».

## Ключевые функции

### Фильтр «Категории»

- Строка поиска с фильтрами в разделе «Онлайн-курсы»
- Помогает находить курсы по тематикам (например, «Продажи», «Онбординг»)

### Добавление категорий

1. «Университет» → «Онлайн-курсы» → «Настройки» (⚙️) → «Управление категориями»
2. «Добавить» → заполнить поля:
   - **Название** (обязательное)
   - **Сортировка** — порядок (меньший индекс = выше)

### Редактирование категорий

- Бургер-меню (≡) → «Изменить»

### Удаление категорий

- Бургер-меню (≡) → «Удалить»
- ⚠️ Курсы НЕ удаляются при удалении категории

### Прикрепление категорий к курсам

- Страница курса → «Настройки» (⚙️) → «Изменить курс» → указать категории
- Категории отображаются на странице описания курса и доступны как фильтры

## Предусловия

- Права: администраторы, HR-отдел
