# Виджеты

**Модуль:** [[Module - Главная страница]]

## Описание

В правой части Главной страницы выводятся виджеты: стандартные виджеты Битрикс24, собственные виджеты Главной, а также виджеты других решений Улья.

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- **Стандартные виджеты Битрикс24:** Важные сообщения, Пульс, Мои задачи, Бизнес-процессы, Дни рождения, Ближайшие события, Популярное
- **Виджеты Главной:** Отсутствия, Ссылки, Социальные сервисы
- **Виджеты решений Улья:** Корпоративный университет, Управление целями (Мои цели), Оценка 360, Геймификация (Бонусная программа), HelpDesk, Карта офиса (Мои бронирования), Опросы, Пульс-опросы
- **Настройка виджетов:** редактирование через режим правки (Мои задачи, Дни рождения, Ссылки, Социальные сервисы, Ближайшие события)
- **Создание своего виджета** через HTML-код
- **Управление блоком:** скрытие/показ, сортировка, переименование

## Предусловия

- Права администратора для настройки и редактирования виджетов
- Включённая административная панель (режим правки)
- Установленные модули Улья для отображения соответствующих виджетов

## Шаги

### Добавление ссылок на виджет «Ссылки»

1. Перейдите в административную часть: `/bitrix/admin`
2. «Контент» > «Главная страница» > «Ссылки»
3. Нажмите «Добавить ссылку»
4. Заполните: Активность, Название, Сортировка, Открывать в новом окне, Ссылка сноски, Иконка (24x24px)
5. Сохраните

### Добавление социальных сервисов

1. Перейдите в административную часть: `/bitrix/admin`
2. «Контент» > «Главная страница» > «Ссылки»
3. Нажмите «Добавить сервис»
4. Заполните: Название, Сортировка, Открывать в новом окне, Ссылка сноски, Иконка (30x30px)
5. Сохраните

### Создание своего виджета

1. В настройках модуля вкладка «Компоненты» → таблица «Правый блок»
2. Нажмите «Добавить виджет»
3. Укажите HTML-код виджета (включая `<script></script>`)
4. Настройте ширину: 280px
5. Укажите название и сортировку
6. Сохраните, затем поставьте галочку «Активность»

### Управление блоком с виджетами

1. Настройки модуля → вкладка «Компоненты» → таблица «Правый блок»
2. Скрыть весь блок: убрать галочку «Правый блок»
3. Скрыть конкретный виджет: убрать галочку «Активность» у него
4. Настроить: Активность, Сортировка, Название, Код виджета

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- Виджеты решений Улья отображаются только если соответствующие модули установлены
- Виджеты могут выглядеть по-разному в зависимости от состояния пользователя (есть/нет карточки целей, оценки и т.д.)
- При изменении названий стандартных виджетов новое название отображается только на Главной, в разделе «Лента» названия не изменятся
