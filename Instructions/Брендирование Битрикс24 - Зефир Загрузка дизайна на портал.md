# Зефир. Загрузка дизайна на портал

**Модуль:** [[Module - Брендирование Битрикс24]]

## Описание

Инструкция по выгрузке стилей из Figma и загрузке их на корпоративный портал для дизайна «Зефир». Отличается от Legacy другим плагином и настройками экспорта.

## Ключевые функции

### Установка плагина

1. Перейти на Яндекс.Диск: https://disk.yandex.ru/d/qoZE90XWIVhucQ
2. Ввести пароль: **wehive**
3. Скачать архив `Hive-branding-zephir.rar` и разархивировать
4. В Figma: `Plugins > Development > Import plugin from manifest`
5. Выбрать файл `manifest`
6. Плагин появится как `Hive-Branding-zephir`

### Выгрузка и загрузка

1. В Figma: `Plugins > Development > Hive-Branding-zephir`
2. Убедиться, что в «Режимы коллекций» выбраны:
   - **Общие:** Light
   - **Авторизация:** Dark + Light
   - **Виджеты:** Light
   - **Шапка портала:** Light
   - **Кнопки:** Light
3. При необходимости включить «Скрыть логотип Б24» (добавляет CSS-свойство в раздел «Дополнительно»)
4. Нажать «Сгенерировать» → «Скачать ZIP»
5. На портале: открыть модуль «Брендирование» → шестерёнка (⚙️)
6. В поле «Импорт настроек» загрузить ZIP-файл
7. Новый дизайн сразу отобразится

## Предусловия

- Дизайн создан в конструкторе Zephir Figma
- Установлен плагин Hive-Branding-zephir
- Пользователь имеет доступ к модулю «Брендирование»

## Граничные случаи

- Параметры в «Режимы коллекций» **нельзя менять** для корректного импорта
- Другие настройки и чекбоксы в окне импорта находятся в доработке — не использовать
- Опция «Скрыть логотип Б24» добавляет CSS в раздел «Дополнительно» — можно убрать вручную
- Плагин работает только в десктопном приложении Figma
