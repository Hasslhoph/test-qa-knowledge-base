# Фотоальбом

**Модуль:** [[Module - Главная страница]]

## Описание

Блок для загрузки и просмотра фотографий корпоративных событий. Сотрудники могут просматривать, комментировать и лайкать фотографии.

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- **2 шаблона на Главной:** sparkle (по умолчанию), .default
- **Создание альбомов** — тематические группы фотографий
- **Загрузка фотографий** с редактированием (поворот, отражение, обрезка, рамки, подписи)
- **Сортировка** фотографий и альбомов
- **Выбор альбомов для отображения** на Главной (до 5)
- **Настройки страницы:** размер миниатюр, сжатие оригиналов, количество фотографий
- **Управление альбомами и фотографиями** через административную часть

## Предусловия

- Права администратора или пользователя с правами на редактирование фотогалереи
- Включённая административная панель (режим правки)

## Шаги

### Создание альбома

1. Перейдите в раздел «Фотогалерея»: `/mainpage/about/gallery/`
2. Нажмите «Добавить альбом»
3. Заполните: Название (обязательное), Дата, Описание
4. Сохраните

### Загрузка фотографий

1. Перейдите в «Фотогалерея» → «Загрузить фото»
2. Выберите альбом (или создайте новый)
3. Нажмите «Загрузить» и выберите файлы
4. Отредактируйте фото при необходимости (поворот, отражение, обрезка, рамка, подпись, описание)
5. Нажмите «Загрузить» и дождитесь завершения

### Настройка блока на Главной

1. В режиме правки наведите на блок фото → шестерёнка (⚙️)
2. Выберите шаблон, настройте сортировку, выберите альбомы для отображения (до 5)
3. Сохраните

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- Для раздела не предусмотрен пункт меню в настройках модуля
- Максимальный размер открытой фотографии по умолчанию: 1280x1280px
- При удалении альбома удаляются все вложенные фотографии
- Загруженные фотографии можно перетаскивать мышкой для изменения порядка
