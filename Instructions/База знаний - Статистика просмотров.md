# База знаний - Статистика просмотров

**Модуль:** [[Module - База знаний]]

## Описание

Автоматический сбор статистики просмотров статей. Статистика доступна всем сотрудникам и включает аналитические срезы по источникам и периодам.

## Ключевые функции

- 5 типов просмотров: Просмотры, Просмотры intranet, Просмотры ext (extranet), Просмотры mobile, Просмотры за 30/90/180/360 дней
- Настройка колонок таблицы (шестерёнка)
- Фильтрация по типу просмотров
- Сортировка по количеству просмотров и диапазону
- Статистика старше 360 дней автоматически удаляется

## Предусловия

- Права на чтение статей (статистика доступна всем сотрудникам)

## Шаги

### Просмотр статистики

1. Нажмите «Настройки» (⚙️) > «Статистика просмотров статей»
2. Откроется страница со списком документов и количеством уникальных просмотров

### Настройка отображения колонок

1. Нажмите шестерёнку (⚙️) в левом верхнем углу таблицы
2. Выберите нужные типы просмотров для отображения

### Фильтрация просмотров

1. В разделе «Просмотры статей» используйте строку поиска
2. Выберите фильтры для отбора документов по типу просмотров
3. Сортируйте статьи по точному количеству просмотров или диапазону

## Ожидаемый результат

Просмотр статистики по статьям с возможностью фильтрации и сортировки.

## Граничные случаи

- В статистику попадают только статьи с одним или более просмотрами
- Статьи, которые открывал только автор, не учитываются (количество просмотров = 0)
- Просмотры автора статьи не учитываются
- Просмотры с одного IP-адреса (повторные) не учитываются
- Уникальный просмотр intranet — первый просмотр конкретным пользователем
- Уникальный просмотр ext — просмотр через публичную ссылку с уникальным IP
- Уникальный просмотр mobile — первый просмотр через мобильное приложение (даже если уже читал в десктопе)
- Статистика старше 360 дней автоматически удаляется
