# Публичный доступ к курсам

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Возможность сделать раздел «Онлайн-курсы» доступным для неавторизованных пользователей.

## Ключевые функции

### Как открыть публичный доступ

1. Административная часть → «Настройки» → «Улей: Корпоративный университет»
2. Вкладка «Общие» → опция «Выводить онлайн-курсы с правами "Все посетители: Просмотр" по публичной ссылке»
3. По умолчанию опция отключена

### Нюансы публичного доступа

1. **Права доступа внутри курса** — должны быть заданы права «Все посетители: Просмотр»
2. **Ссылка** — адрес_портала`/pub/learning/`
3. **Функционал отключен** — при переходе по ссылке без включённой опции — сообщение об отказе
4. **Тесты недоступны** — неавторизованные пользователи не могут проходить тесты

## Предусловия

- Права: администраторы

## Граничные случаи

- Доступен только просмотр глав и уроков
- Для прохождения тестов нужна авторизация (Интранет или Экстранет)
