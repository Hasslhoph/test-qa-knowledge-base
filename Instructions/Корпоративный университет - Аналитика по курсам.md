# Аналитика по курсам

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Раздел «Аналитика» для анализа результатов прохождения тестов. Пять вкладок с различной информацией о попытках.

## Ключевые функции

### Права доступа

| Роль | Доступ |
|------|--------|
| Администраторы | Все тесты, включая свои |
| HR-отдел | Все тесты, включая свои |
| Группа для отчётов | Все тесты, включая свои |
| Руководители по оргструктуре | Свои + подчинённые |
| Руководители в карточках | Свои + владельцы карточек |
| Наблюдатели в карточках | Свои + владельцы карточек |

### Вкладки

1. **Последние попытки** — последние попытки каждого пользователя по каждому тесту
2. **Лучшие попытки** — самые успешные попытки
3. **Все попытки** — каждая попытка каждого сотрудника
4. **Журнал** — суммарное количество попыток по каждому тесту
5. **Тесты** — сводная информация о результатах сдачи каждого теста
6. **Курсы** — статистика по курсам и тестам

### Аналитика просмотров глав и уроков (вкладка «Курсы»)

- Процент просмотренных страниц курса
- Слайдер со списком пользователей и процентом просмотров
- Детальная информация о просмотрах каждого сотрудника

### Редактирование попыток

- Права: администраторы, HR-отдел
- Массовое редактирование: Отвечен, Верно отвечен, Баллов
- Индивидуальное редактирование: Отвечен, Правильно отвечен, Баллов, Ответ

### Удаление попыток

- Массовое и индивидуальное удаление
- Права: администраторы, HR-отдел

## Предусловия

- Права доступа к разделу «Аналитика»

## Граничные случаи

- При большом количестве пользователей (>1000) возможна замедленная работа
- Опция в настройках модуля для ограничения функциональности при большом количестве попыток
