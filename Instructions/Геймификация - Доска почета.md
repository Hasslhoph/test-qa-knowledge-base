# Геймификация - Доска почета

**Модуль:** [[Module - Геймификация]]

## Описание

Раздел, в котором отображаются призеры рейтингов за прошедший месяц. Список призеров доступен всем пользователям корпоративного портала.

## Ключевые функции

- Автоматическое обновление 1-го числа каждого месяца (для автоматических рейтингов)
- Для ручных рейтингов — попадание на доску при ручном начислении баллов за призовые места
- Поиск призеров по фильтрам: за любые периоды и по любым рейтингам
- Кнопка «Сказать спасибо» для каждого сотрудника на доске почета
- Количество призеров настраивается отдельно для каждого рейтинга в административной части

## Предусловия

- Модуль установлен и настроен
- По рейтингам проведены итоги (автоматически или вручную)

## Шаги

1. Перейдите в раздел «Доска почета» в публичной части портала
2. Просмотрите призеров за прошедший месяц
3. При необходимости используйте фильтры для поиска призеров за другие периоды
4. Для благодарности коллеге нажмите «Сказать спасибо» → выберите категорию → добавьте текст → отправьте

## Ожидаемый результат

Пользователь видит призеров рейтингов и может поблагодарить коллег баллами «Спасибо».

## Граничные случаи

- Призерами считаются все пользователи, получившие баллы за участие в рейтинге
- На доску попадают призеры как автоматических, так и ручных рейтингов
- Категории благодарностей должны быть заранее созданы в настройках модуля
