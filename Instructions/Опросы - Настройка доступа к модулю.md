# Настройка доступа к модулю

**Модуль:** [[Module - Опросы]]

## Описание

Настройка прав доступа для группы «Редактор опросов». Помимо администраторов, работать с опросами могут пользователи из этой группы.

## Ключевые функции

### Расположение настройки прав

1. Перейти в административную часть портала: `адрес_портала/bitrix/admin/`
2. В левом меню: «Настройки» → «Настройки продукта» → «Настройки модулей» → «Опросы, голосования»
3. Перейти во вкладку «Доступы»

### Виды прав доступа для группы «Редактор опросов»

| Право | Описание |
|---|---|
| **Закрыт** | Пользователи не могут добавлять и редактировать опросы. Права на просмотр и участие остаются |
| **Просмотр всех данных модуля** | Пользователи не могут добавлять и редактировать опросы. Могут просматривать опросы в административной части портала |
| **Запись** | Полный доступ к управлению опросами через публичную и административную часть портала |

### Добавление пользователя в группу

Выполняется через учётную запись пользователя.

## Предусловия

- Пользователь — администратор портала

## Шаги

1. Перейти в административную часть портала
2. «Настройки» → «Настройки продукта» → «Настройки модулей» → «Опросы, голосования»
3. Вкладка «Доступы»
4. В поле «Редактор опросов» выбрать нужное право доступа
5. Сохранить изменения

## Ожидаемый результат

Группа «Редактор опросов» имеет настроенные права доступа к модулю.
