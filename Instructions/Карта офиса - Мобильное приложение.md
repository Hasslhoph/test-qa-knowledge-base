# Мобильное приложение

**Модуль:** [[Module - Карта офиса + Бронирование]]

## Описание

Инструкция по использованию модуля «Карта офиса» в мобильном приложении Битрикс24.

## Ключевые функции

- Просмотр страницы «Мои бронирования»
- Просмотр страницы «Кто в офисе»
- Подтверждение бронирований
- Отмена бронирований

## Предусловия

- Установлено мобильное приложение Битрикс24
- Модуль установлен и активирован на портале

## Доступные возможности

### Просмотр бронирований
- Слайдер «Мои бронирования» — текущие и прошедшие брони сотрудника
- Слайдер «Кто в офисе» — кто из сотрудников забронировал место

### Подтверждение и отмена
- Подтверждение бронирования через мобильное приложение
- Отмена бронирования через мобильное приложение

## Граничные случаи

- Через мобильное приложение **нельзя** добавлять контент в модуль
- Через мобильное приложение **нельзя** управлять правами доступа
- Через мобильное приложение **нельзя** переходить в административную часть портала
- Все административные действия доступны только через десктопную версию Битрикс24
