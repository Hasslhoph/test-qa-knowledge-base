# Оценка 360 - Добавление индикаторов и привязка к компетенциям

**Модуль:** [[Module - Оценка 360]]

## Описание

Инструкция по добавлению индикаторов (критериев, проявлений) к компетенциям. Индикатор позволяет настроить глубину оценки — любую компетенцию можно оценивать как саму по себе, так и с использованием индикаторов.

## Ключевые функции

### Что такое индикатор
- Индикатор — это то, как проявляется компетенция
- Пример: компетенция «Управление конфликтами» → индикатор «Старается разрешить конфликт в пользу компании»
- Благодаря индикаторам результаты оценки становятся более объективными

### Добавление индикаторов
- Раздел «Компетенции» → бургер-меню (≡) → «Редактировать» → «Добавить»
- Можно выбрать существующие индикаторы из списка (с поиском по названию/описанию)
- Можно создать новые через кнопку «Добавить индикатор»
- Поля нового индикатора: Название (обязательное), Описание, Привязка к компетенциям

### Редактирование индикаторов
- Бургер-меню (≡) → «Редактировать»
- Отвязка от компетенции: крестик в блоке «Выбранные индикаторы»

### Один индикатор — несколько компетенций
- Индикатор может быть привязан к нескольким компетенциям
- Часто возникает при использовании опросов (компетенции как вопросы, индикаторы как варианты ответа)
- Пример: индикаторы «Не проявляет», «Проявляет редко», «Проявляет часто» универсальны для многих компетенций

## Предусловия

- Права администратора или группы HR-отдела
- Создана компетенция, к которой привязываются индикаторы

## Граничные случаи

- Обычные пользователи не имеют доступа к редактированию компетенций
- Ознакомиться с индикаторами можно в карточке конкретной компетенции
- При балльной оценке с одиночным выбором индикаторы становятся вариантами ответа
