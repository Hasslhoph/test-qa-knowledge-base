# Создание категорий компетенций

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Создание и редактирование категорий (разделов) для компетенций. Каждая компетенция должна быть привязана к категории.

## Ключевые функции

### Добавление категорий

1. Раздел «Компетенции» → «Настройки» (⚙️) → «Категории» → «Добавить»
2. Заполнить поля:
   - **Название** — наименование категории
   - **Родительский раздел** — для создания вложенности
   - **Активность** — скрытие категории от обычных пользователей
   - **Сортировка** — порядок категорий (меньший индекс = выше)
   - **Цвет** — HEX код цвета (например, #f10057)

### Редактирование категорий

- Бургер-меню (≡) → «Изменить»

### Копирование категорий

- Бургер-меню (≡) → «Копировать»

### Удаление категорий

- Бургер-меню (≡) → «Удалить»
- ⚠️ Удаляются все вложенные подразделы и компетенции

### Просмотр подкатегорий

- Клик на название категории → список подкатегорий
- Вложенность категорий не ограничена
- Если у подкатегории не указан цвет — наследует цвет родительской

## Предусловия

- Права: администраторы и группа HR-отдела

## Граничные случаи

- Категория отображается в разделе «Компетенции» только после добавления компетенций
- Рекомендуемый уровень вложенности — не более четырёх уровней
