# Настройка блока «Возникла проблема?»

**Модуль:** [[Module - Карта офиса + Бронирование]]

## Описание

Инструкция по настройке блока «Возникла проблема?» на слайдере бронирования, через который сотрудники могут оставить заявку на ремонт, уборку или другое обслуживание рабочего места.

## Ключевые функции

- Показать/скрыть блок на слайдере бронирования
- Настройка ссылок для фраз о неисправностях
- Настройка текста блока
- Интеграция с HelpDesk или чатом техподдержки

## Предусловия

- Доступ к настройкам есть только у администратора портала

## Показать/скрыть блок

1. Включить режим правки (панель управления)
2. Навести курсор на область карты офиса
3. Нажать на серую кнопку с иконкой шестерёнки
4. Раздел «Дополнительные настройки» → галочка «Показывать блок "Возникла проблема?"»
5. Сохранить изменения

## Настройка ссылок

1. Перейти: `адрес_портала/bitrix/admin/index.php?lang=ru`
2. Контент → Структура сайта → Файлы и папки
3. Путь: `/include/ithive.workplaces`
4. Бургер-меню → «Редактировать как PHP»
5. Вместо `javascript:void(0)` вставить ссылки для ID фраз:
   - `HAS_PROBLEM_LINK_1` — Сломалось кресло
   - `HAS_PROBLEM_LINK_2` — Не работает монитор
   - `HAS_PROBLEM_LINK_3` — Заказать уборку

## Настройка текста

В том же файле `/include/ithive.workplaces` заменить строки:
- `HAS_PROBLEM_TITLE` — заголовок блока
- `HAS_PROBLEM_CONTENT` — текст описания
- `HAS_PROBLEM_LINK_1/2/3` — текст фраз-ссылок

## Граничные случаи

- При переустановке модуля PHP-файл будет перезаписан — ссылки и текст нужно указать заново
- Ссылки могут вести на сервис техподдержки, чат с офис-менеджером, форму HelpDesk и т.д.
