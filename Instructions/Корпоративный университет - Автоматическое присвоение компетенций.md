# Автоматическое присвоение компетенций

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Присвоение компетенций и их добавление в профиль сотрудника по результатам прохождения программы развития.

## Ключевые функции

### Условия присвоения

1. Только онлайн-курсы с тестами → успешное прохождение тестов → получение компетенции
2. Развивающие действия с проверкой руководителя → оценка «Полностью достигнута» или «Достигнута с превышением»

### Оценка компетенций

- Оценка выполняется сотрудником и руководителем
- Уведомления раз в месяц о необходимости оценить прогресс
- Индикатор «Требуется оценить» — загорается красным каждые 30 дней

### Процесс оценки

1. Блок «Сотрудник» — самооценка + комментарии
2. Блок «Руководитель» — оценка по объективным данным + обратная связь

### Градации оценок

1. Нет прогресса
2. Частичное достижение, необходимо дальнейшее развитие
3. Полностью достигнута → синяя галочка
4. Достигнута с превышением → сотрудник становится экспертом

### Цвет иконок в карточке развития

- **Полностью закрашенная** — все действия выполнены, оценка получена
- **Серая с цветным ободком** — действия выполнены, оценка ожидается
- **Серая** — компетенция не получена

## Предусловия

- Активная карточка развития
- Функция «Автоматическая проверка результатов» включена (для курсов с тестами)

## Граничные случаи

- Руководитель может оценить компетенцию без участия сотрудника
- Экспертный уровень нельзя получить через программу развития — только вручную
- При оценке «Достигнута с превышением» кнопки оценки становятся неактивными
