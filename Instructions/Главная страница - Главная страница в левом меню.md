# Главная страница в левом меню

**Модуль:** [[Module - Главная страница]]

## Описание

Инструкция по скрытию Главной страницы из левого меню на время наполнения контентом и рекомендации по установке Главной в качестве стартовой страницы портала.

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- Скрытие разделов Главной из левого меню (Новости, Сервисы, Лидеры продаж, Кадровые изменения)
- Скрытие пунктов «Главная» и «О компании» из левого меню
- Установка Главной стартовой страницей через настройку меню
- Использование Главного инструмента «HR-Portal» для приоритетного расположения модулей Улья
- Информирование сотрудников через подсказки модуля «Интерактивные подсказки»

## Предусловия

- Права администратора портала
- Доступ к административной части (`/bitrix/admin/`)
- Настройки модуля: `/bitrix/admin/settings.php?lang=ru&mid=ithive.homepage`

## Шаги

### Как скрыть Главную страницу из левого меню

1. Перейдите в административную часть портала: `/bitrix/admin/`
2. Нажмите «Настройки» > «Улей: Главная страница» (или перейдите по ссылке `/bitrix/admin/settings.php?lang=ru&mid=ithive.homepage`)
3. Во вкладке «Компоненты» > вкладка нужного сайта снимите галочки во всех пунктах колонки «Вывести в левом меню» (Центральный блок)
4. Перейдите во вкладку «Источники данных» > вкладка второго сайта и снимите галочки:
   - Вывести в левом меню пункт «Главная»
   - Вывести в левом меню пункт «О компании»

### Как сделать Главную стартовой страницей

1. Переместите пункт «Главная» на первое место в своем левом меню (вручную или через Главный инструмент)
2. Для новых сотрудников: установите свое меню для всех новых пользователей
3. Или используйте Главный инструмент «HR-Portal» — при его выборе Главная автоматически встает на первое место

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- Разделы пропадут только из левого меню, но будут доступны по прямым ссылкам
- При включении Главного инструмента меняется порядок ВСЕХ пунктов левого меню
- HR-Portal предназначен для компаний с несколькими HRM-решениями Улья
