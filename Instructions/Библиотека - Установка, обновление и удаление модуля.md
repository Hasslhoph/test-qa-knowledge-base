# Установка, обновление и удаление модуля

**Модуль:** [[Module - Библиотека]]

## Описание

Инструкция по установке, обновлению, активации купонов и удалению модуля «Библиотека» на корпоративном портале Битрикс24. Все операции требуют прав администратора.

## Ключевые функции

- Установка модуля через административную часть
- Обновление через раздел Marketplace
- Активация купона на покупку/продление
- Активация купона на неустановленный модуль
- Активация купона для модуля с истёкшей демо-версией
- Удаление модуля через Marketplace

## Предусловия

- Права администратора портала

## Шаги

### Установка модуля

1. Перейдите: `адрес_портала/bitrix/admin/update_system_partner.php?addmodule=ithive.library`
2. Модуль автоматически отмечен галочкой — нажмите «Загрузить»
3. Примите Лицензионное соглашение и Условия конфиденциальности → «Применить»
4. Дождитесь завершения установки
5. Убедитесь в уведомлении об успешной загрузке

### Обновление модуля

1. «Marketplace» → «Обновление решений» → «Список обновлений»
2. Выберите «Улей: Библиотека»
3. Нажмите «Загрузить»
4. Или перейдите напрямую: `адрес_портала/bitrix/admin/update_system_partner.php`

### Активация купона на покупку/продление

1. «Marketplace» → «Обновление решений» → «Активация купона»
2. Введите номер купона
3. Нажмите «Активировать купон»
4. Или перейдите: `адрес_портала/bitrix/admin/update_system.php?lang=ru`

### Активация купона на неустановленный модуль

1. «Marketplace» → «Установленные решения»
2. В «Купленные решения» нажмите бургер-меню рядом с модулем → «Загрузить»

### Активация купона для модуля с истёкшей демо-версией

1. Перейдите в «Обновление решений»
2. Во вкладке «Установка обновлений» нажмите «Зарегистрировать решение»

### Удаление модуля

1. «Marketplace» → «Установленные решения» → «Доступные решения»
2. Откройте бургер-меню слева от решения → «Удалить»
3. Или перейдите: `адрес_портала/bitrix/admin/update_system_partner.php`
4. Подтвердите удаление после обновления страницы
5. Дождитесь сообщения об удалении

## Граничные случаи

- Все операции требуют прав администратора
- При установке модуля автоматически загружаются все доступные обновления
- Для активации купона на неустановленный модуль используется раздел «Установленные решения» → «Купленные решения»
