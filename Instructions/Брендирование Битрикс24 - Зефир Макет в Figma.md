# Зефир. Макет в Figma

**Модуль:** [[Module - Брендирование Битрикс24]]

## Описание

Инструкция по работе с конструктором Figma для дизайна «Зефир». Макет включает шаблоны страницы авторизации и страницы портала с визуальным предпросмотром изменений.

## Ключевые функции

### Копирование макета

- Ссылка: [\[S\] Конструктор "Zephir" (1.0.2)](https://www.figma.com/design/7de0YCzGbVi7ppBYL5O4qZ/-S—%D0%9A%D0%BE%D0%BD%D1%81%D1%82%D1%80%D1%83%D0%BA%D1%82%D0%BE%D1%80-%22Zephir%22—1.0.2-?node-id=32-518&p=f&t=wWiXcVSBE6rb2UMG-0)
- Пароль: **wehive**
- Копирование: «Duplicate to your drafts»

### Шаблоны макета

1. **Default/Page «Login»** — страница авторизации:
   - Блок загрузки изображений (логотип компании, картинка фона)
   - Общий вид страницы авторизации
   - Настройка состояния элементов (default, hover, active)

2. **Light/Page «Portal»** — страница портала:
   - Блок загрузки изображения фона
   - Общий вид страницы со всеми элементами
   - Настройки верхней панели (строка поиска, результаты)
   - Настройки кнопок (3 состояния)

Элементы, недоступные в Figma:
- Спрятать блок «Скачать приложения» / «ПУЛЬС КОМПАНИИ»
- Фавикон
- Все настройки «Контент» и «Другое»
- Некоторые настройки шапки портала

### Редактирование через Variables

- Меню Variables: клик на пустое пространство → Variables → Show panel
- Структура группировки идентична модулю «Брендирование»
- Настройки с подчёркиванием в начале (например `_Логотип компании`) недоступны для импорта

### Загрузка логотипа

- Максимальный размер: 355×355 пикселей
- Замена через `Paste to replace` на элементе `Vector`
- Размер области: элемент `Default` в списке слоёв

### Настройка элементов

- **Цвет**: Variables → нужный раздел → клик на иконку цвета → палитра/HEX-код
- **Скругление и обводка**: Variables → Виджеты → нужный виджет → числовое значение

## Предусловия

- Аккаунт Figma (бесплатный)
- Десктопное приложение Figma для скачивания стилей

## Граничные случаи

- Загруженные изображения (логотип, фон) **нельзя** скачать из Figma вместе со стилями — их нужно загружать отдельно через модуль
- Не рекомендуется удалять/добавлять Variables и элементы шаблона
- Отмена изменений: Ctrl+Z / Command+Z (Mac)
- Шаблоны `Custom / Page "Login"` и `Custom Light / Page "Portal"` находятся в разработке — не используются
