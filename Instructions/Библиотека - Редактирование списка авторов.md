# Редактирование списка авторов

**Модуль:** [[Module - Библиотека]]

## Описание

Инструкция по редактированию списка авторов книг в административной части портала. В этот список также сохраняются авторы, появившиеся при интеграции с издательствами МИФ или Альпина.

## Ключевые функции

- Редактирование наименований авторов
- Добавление новых авторов
- Список включает авторов из интеграций с издательствами

## Предусловия

- Права администратора портала

## Шаги

1. Перейдите в административную часть: `адрес_портала/bitrix/admin`
2. В левом меню: «Контент» → «Библиотека» → «Библиотека»
3. Внизу страницы кликните на ссылку «Настройки информационного блока»
4. Перейдите на вкладку «Свойства»
5. В списке найдите пункт «Автор» и нажмите на три точки справа (...)
6. В открывшемся списке:
   - Скорректируйте наименования авторов
   - Добавьте нового автора через кнопку «Еще»
7. Сохраните изменения

## Граничные случаи

- В Библиотеке не может быть двух одинаковых авторов — дубликаты автоматически объединяются
- Авторы из интеграций с МИФ и Альпина также сохраняются в этот список
- Работать с настройками может только администратор
