# Разделы модуля

**Модуль:** [[Module - Главная страница]]

## Описание

Модуль «Главная страница» включает в себя десять разделов, каждый из которых выполняет свою функцию на корпоративном портале.

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- **Главная** (`/mainpage/`) — основной раздел с баннерами, виджетами, новостями, сервисами и медиа галереей
- **О компании** (`/mainpage/about/about/`) — визитная карточка организации с настраиваемыми блоками
- **Новости компании** (`/news/`) — основной новостной раздел с возможностью дублирования в ленту
- **Новости СМИ** (`/news2/`) — вспомогательный новостной раздел
- **Сервисы и заявки** (`/proposals/`) — бизнес-процессы Битрикс24 в одном месте
- **Фотоальбом** (`/mainpage/about/gallery/`) — галерея фотографий
- **Видеоальбом** (`/mainpage/about/media.php`) — галерея видеозаписей
- **Лидеры продаж** (`/leader_sales/`) — доска почета
- **Кадровые изменения** (`/company/events.php`) — прием, перевод, увольнение
- **Живая лента** (`/stream/`) — дублирование сообщений из Ленты Битрикс24

## Предусловия

- Установленный модуль «Главная страница»
- При установке автоматически добавляются в левое меню: Главная, О компании, Сервисы и заявки
- Остальные разделы выводятся через [[Главная страница - Настройки модуля]]

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- Разделы «Фотоальбом», «Видеоальбом» и «Живая лента» нельзя добавить в левое меню через настройки Главной — это стандартные разделы Битрикс24
- Раздел «О компании» не выводится на Главной странице, для него выделена отдельная страница
- На Главной центральный блок содержит баннеры и основные разделы, правый блок — виджеты
