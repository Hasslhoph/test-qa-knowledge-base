# Геймификация - Основные рейтинги

**Модуль:** [[Module - Геймификация]]

## Описание

По умолчанию в модуле предусмотрено семь автоматических рейтингов. Рейтинги автоматически пересчитываются каждые сутки, а 1-го числа каждого месяца модуль подводит итоги — определяет призеров и начисляет им баллы.

## Ключевые функции

- **Эффективный исполнитель задач** — учитывает выполненные задачи (исполнитель, постановщик — руководитель, есть дедлайн); параметры: замечания, лайки постановщика к комментариям
- **Эффективный постановщик задач** — учитывает задачи где сотрудник постановщик; параметры: объём описания, файлы, чек-лист, лайки
- **Активное развитие** — работает с модулем «Корпоративный университет» ([[Module - Корпоративный университет]]); учитывает мероприятия, наставничество, книги, тесты, курсы, задачи
- **Лидер продаж** — для отдела продаж; учитывает сделки, звонки, письма, встречи, задачи в timeline CRM
- **Эффективный пользователь календаря** — учитывает многопользовательские события; параметры: длина названия/описания, участники, комментарии
- **Главный по новостям** — работает с модулем «Главная страница» ([[Module - Главная страница]]); учитывает просмотры, лайки, комментарии в новостях компании и СМИ
- **Общий рейтинг** — подводит итоги по остальным рейтингам; вес каждого рейтинга настраивается

## Предусловия

- Модуль установлен
- Рейтинги опубликованы в настройках модуля
- Выбраны подразделения, участвующие в рейтингах

## Шаги

1. Перейдите в раздел «Рейтинги» в публичной части портала
2. Выберите нужный рейтинг из списка
3. Отслеживайте свои позиции и позиции коллег
4. 1-го числа каждого месяца проверяйте начисления баллов и доску почета

## Ожидаемый результат

Сотрудники участвуют в рейтингах, отслеживают позиции, получают баллы за призовые места.

## Граничные случаи

- Рейтинги можно скрыть в настройках модуля
- Можно выбрать подразделения, которые будут участвовать в рейтингах
- По умолчанию в списке отображается до 10 участников (настраивается)
- Призеры попадают на [[Геймификация - Доска почета]]
- Для всех параметров можно задать свой вес влияния на позицию в рейтинге
