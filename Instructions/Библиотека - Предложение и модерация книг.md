# Предложение и модерация книг

**Модуль:** [[Module - Библиотека]]

## Описание

Инструкция по предложению книг обычными пользователями и процессу модерации предложенных материалов. Интерфейс Библиотеки различается для обычных пользователей и модераторов.

## Ключевые функции

- **Для обычных пользователей:** виден только список опубликованных книг, доступна кнопка «Предложить»
- **Для модераторов:** доступны два списка — «Опубликовано» и «Предложения»
- Предложение книги через форму с полями: категория, название, описание, ссылка на материал, стоимость
- Уведомления модераторам о новых предложениях
- Модерация через слайдер редактирования книги (галочка «Активность» для публикации)

## Предусловия

- Модуль «Библиотека» установлен и активен
- Опция предложения книг не отключена в настройках модуля

## Шаги

### Предложение книги (пользователь)

1. Нажмите кнопку «Предложить» в Библиотеке
2. Заполните форму:
   - **Категория учебного материала** — одна тематика (например, «Маркетинг»)
   - **Название учебного материала** — обязательное поле
   - **Описание учебного материала** — краткое содержание, причина добавления (обязательное)
   - **Ссылка на страницу учебного материала** — ссылка на покупку/скачивание
   - **Стоимость учебного материала (руб)** — цена за экземпляр
3. Отправьте предложение на модерацию

### Модерация предложений

1. Получите уведомление о новой книге
2. Откройте книгу через гиперссылку в уведомлении ИЛИ через раздел «Предложения» → «Меню» (≡) → «Открыть»
3. Ознакомьтесь с информацией и заполните недостающие поля
4. Для публикации: активируйте галочку «Активность»
5. Для отклонения: оставьте галочку неактивной или удалите книгу

## Граничные случаи

- По умолчанию у всех предложенных книг отключена галочка «Активность»
- Предлагать книги можно неограниченное количество раз
- Опцию предложения книг можно отключить в административной части портала ([[Библиотека - Настройки модуля]])
- Модератор может настроить отображение столбцов через шестерёнку (⚙️) в таблице предложений
