# Сервисы и заявки

**Модуль:** [[Module - Главная страница]]

## Описание

Раздел для вывода бизнес-процессов Битрикс24 в режиме единого окна. Позволяет группировать сервисы по разделам (для сотрудников, для руководителей) и привязывать к ним бизнес-процессы.

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- **Привязка к бизнес-процессам** Битрикс24 (отпуска, счета, командировки и т.д.)
- **Ссылки на внешние ресурсы** — перенаправление на другие разделы портала или сторонние сайты
- **Группировка сервисов** по разделам (для сотрудников, для руководителей, для HR и т.д.)
- **Избранное** — добавление сервисов в избранное (в шаблоне proposals_by_section_sparkle)
- **2 шаблона на Главной:** proposals_by_section_sparkle (по умолчанию), proposals_by_section
- **2 шаблона на странице:** proposals_list (по умолчанию), proposals_list_favourite
- **Таргетирование** — доступ к сервисам для конкретных отделов/групп
- **Создание/редактирование разделов** для группировки сервисов

## Предусловия

- Права привилегированного пользователя с доступом к инфоблоку «Сервисы и заявки»
- Включённая административная панель (режим правки)
- Созданные бизнес-процессы в Битрикс24 (раздел «Автоматизация» > «Процессы в ленте новостей»)

## Шаги

### Создание сервиса

1. На Главной в режиме правки: наведите на блок сервисов → «Добавить сервис»
   Или на странице «Сервисы и заявки»: «Добавить сервис»
2. Вкладка «Параметры»:
   - Активность, Начало/Окончание активности
   - Название (обязательное), Сортировка
   - Открывать в новом окне (да/нет)
   - Отображать на главной (да/нет)
   - Привязка к БП (поиск по наименованию)
   - Ссылка сноски (если вместо БП)
   - Иконка (JPEG, PNG, 48–90px)
3. Вкладка «Разделы»: выберите один раздел
4. Вкладка «Доступ»: настройка доступа для групп
5. Сохраните

### Создание раздела

1. Перейдите в административную часть: `/bitrix/admin/`
2. «Контент» > «Главная страница» > «Сервисы и заявки»
3. Откройте папку нужного сайта
4. Нажмите «Добавить раздел»
5. Вкладка «Раздел»: Раздел активен, Название
6. Вкладка «Доступ»: настройка доступа
7. Сохраните

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- Модуль не создаёт бизнес-процессы с нуля, а использует уже существующие
- При одновременном указании ссылки и привязки к БП открывается только слайдер БП
- Блок «Избранное» отображается только в шаблоне proposals_by_section_sparkle
- При удалении раздела удаляются все вложенные сервисы
