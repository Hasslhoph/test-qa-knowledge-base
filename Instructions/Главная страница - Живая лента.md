# Живая лента

**Модуль:** [[Module - Главная страница]]

## Описание

Блок в нижней части Главной страницы, дублирующий сообщения из стандартной Ленты Битрикс24.

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- **Дублирование сообщений** из Ленты Битрикс24
- **Настройка количества дней** лога
- **Настройка количества записей** для отображения
- **Переход в основную Ленту** по кнопке «Все сообщения»

## Предусловия

- Права администратора для настройки блока
- Включённая административная панель (режим правки)

## Шаги

### Настройка блока на Главной

1. В режиме правки наведите на блок «Живая лента» → шестерёнка (⚙️)
2. Настройте:
   - **Количество дней лога:** максимальное количество дней для вывода записей
   - **Количество отображаемых записей:** максимальное количество записей в блоке
3. Сохраните

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- Это стандартный раздел Битрикс24
- На многосайтовых порталах выводится лента текущего сайта автоматически
