# Геймификация - О продукте

**Модуль:** [[Module - Геймификация]]

## Описание

Входная точка в инструкцию к модулю «Геймификация» (ithive.gamification). Модуль добавляет на корпоративный портал игровые механики для повышения мотивации и вовлеченности персонала.

## Ключевые функции

- Модуль совместим с PHP версии 8.1 и выше
- Поддержка СУБД: MySQL, PostgreSQL
- Поддержка английского языка
- Функционал доступен через современный десктопный браузер
- Актуальная версия модуля: 7.3.0 (Ноябрь 2025)

## Предусловия

- Установленный Битрикс24 (корпоративный портал)
- PHP 8.1+
- MySQL или PostgreSQL

## Шаги

1. Перейдите в раздел инструкции к модулю «Геймификация»
2. Изучите разделы: О продукте, Рейтинги сотрудников, Бонусная программа, Администрирование модуля
3. При необходимости ознакомьтесь с модулем на [сайте компании Улей](https://www.wehive.digital/products/gamification/)

## Ожидаемый результат

Пользователь получает общее представление о модуле «Геймификация», его технических требованиях и возможностях.

## Граничные случаи

- Инструкция актуальна для версии 7.3.0
- Функционал доступен только через десктопный браузер
