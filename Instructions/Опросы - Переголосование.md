# Переголосование

**Модуль:** [[Module - Опросы]]

## Описание

Функция, позволяющая респондентам менять свои ответы в опросе после первоначального голосования.

## Ключевые функции

- По умолчанию респондент проходит опрос только один раз
- Для включения: на слайдер редактирования опроса → галочка «Разрешить менять мнение»
- После прохождения опроса появляется кнопка «Переголосовать»
- При нажатии пользователь возвращается на страницу опроса с видимыми прошлыми ответами
- Изменённые ответы перезаписывают предыдущие
- Менять ответы можно неограниченное количество раз, пока срок опроса не истёк

## Предусловия

- В настройках опроса активирована опция «Разрешить менять мнение»
- Срок действия опроса не истёк

## Граничные случаи

- **Переголосования не учитываются как новые голоса** — если пользователь прошёл опрос и трижды изменил мнение, его голос учитывается только один раз
- Кнопка «Переголосовать» доступна на весь срок действия опроса
