# Настройка уведомлений для текущих сотрудников

**Модуль:** [[Module - Настройка уведомлений]]

## Описание

В разделе «Уведомления для текущих сотрудников» выполняются настройки для всех пользователей, которые уже добавлены на корпоративный портал. Доступ к разделу есть только у пользователей с соответствующими правами доступа.

## Ключевые функции

- **Выбор пользователей** — можно выбрать как одного пользователя, так и нескольких, а также подразделения и рабочие группы
- **Настройка 18 видов уведомлений** для трёх каналов:
  - Веб-версия и десктоп-приложение
  - Электронная почта
  - Push-уведомления (мобильное приложение)
- **Три уровня настройки**: все уведомления, группа уведомлений, конкретное уведомление
- **Четыре значения настроек**: Установить, Не менять, Снять, Недоступно

## Предусловия

- Модуль установлен и включён
- Пользователь имеет права на настройку уведомлений (входит в группу, указанную в настройках модуля)

## Шаги настройки

### Шаг 1. Выбор пользователей

Выберите пользователей, для которых нужно применить новые настройки. Модуль позволяет выбрать:
- Отдельного пользователя
- Нескольких пользователей
- Подразделения
- Рабочие группы

### Шаг 2. Настройка конкретных уведомлений

На странице модуля показаны все виды уведомлений, предусмотренные в Битрикс24:

1. Чат и звонки
2. Сообщения в Ленте
3. Календарь событий
4. CRM
5. Диск
6. Форум
7. Рейтинги, Лайки
8. Открытые линии
9. Интранет
10. Почта
11. Фотогалерея 2.0
12. Группы и проекты
13. Задачи
14. Учет рабочего времени
15. Wiki
16. Email-маркетинг
17. Бизнес-процессы
18. Опросы, голосования

Каждый раздел можно развернуть, чтобы увидеть конкретные уведомления, относящиеся к нему.

### Шаг 3. Выбор значения настройки

Для каждого уведомления выберите одно из четырёх значений:

| Значение | Описание |
|----------|----------|
| **Установить** | Уведомления будут приходить указанным сотрудникам |
| **Не менять** | Настройки уведомлений у выбранных сотрудников не изменятся |
| **Снять** | Уведомления не будут приходить указанным сотрудникам |
| **Недоступно** | В Битрикс24 нет возможности изменить настройки данных уведомлений |

### Шаг 4. Применение настроек

Нажмите кнопку **«Применить»** в нижней части страницы. Настройки уведомлений будут применены для указанных пользователей.

### Сброс настроек

Кнопка **«Сбросить»** возвращает значения абсолютно всех уведомлений к значениям по умолчанию. При этом настройки не будут применены для сотрудников — сброс влияет только на заполнение значений, а не на их применение.

## Граничные случаи

- После сохранения настроек значения в поле «Все уведомления» очищаются и поля отображаются пустыми. При обновлении страницы никакие значения по умолчанию не будут выбраны
- То же самое происходит со значениями для группы настроек — они очищаются после применения
- Настройки по умолчанию берутся из текущих настроек пользователя, который указан в настройках модуля. По умолчанию это пользователь с ID 1. Если пользователь не указан, то настройки по умолчанию берутся из профиля текущего пользователя
- При большом количестве сотрудников рекомендуется включить логирование в настройках модуля для формирования log-файла со списком пользователей, для которых были применены настройки
