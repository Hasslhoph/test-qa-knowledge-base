# Миссия и ценности

**Модуль:** [[Module - Главная страница]]

## Описание

Блок на странице «О компании» для представления основополагающих принципов и ориентиров компании. Миссия выводится сверху, ценности — ниже.

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- **2 типа блоков:** Миссия (полная ширина, только текст) и Ценность (половина страницы, заголовок + картинка + текст)
- **Сортировка** блоков одного типа
- **Картинка для ценности** (58x64px)
- **Создание новых блоков** миссии/ценности
- **Переименование и изменение очередности** через [[Главная страница - Управление блоками]]

## Предусловия

- Права администратора или пользователя с правами на редактирование инфоблока
- Включённая административная панель (режим правки)

## Шаги

### Редактирование блока

1. В режиме правки наведите на блок → «Изменить элемент»
2. Вкладка «Параметры»:
   - Название, Сортировка
   - Вид отображения: «Миссия» или «Ценность»
3. Вкладка «Анонс»:
   - Картинка для анонса (58x64px, только для «Ценность»)
   - Описание для анонса (текст)
4. Сохраните

### Создание нового блока

1. `/bitrix/admin/` → «Контент» > «О компании» > «Миссия и ценности» > «Моя компания»
2. Нажмите «Добавить элемент»
3. Заполните поля аналогично редактированию
4. Сохраните

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- Сортировать между собой можно только блоки одного типа (Миссия или Ценность)
- Блоки типа «Миссия» всегда выводятся над блоками с ценностями
- Название отображается в публичной части только для блоков типа «Ценность»
