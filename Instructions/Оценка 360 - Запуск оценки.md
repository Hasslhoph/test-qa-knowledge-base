# Оценка 360 - Запуск оценки

**Модуль:** [[Module - Оценка 360]]

# Запуск оценки

## Запуск оценки для одного сотрудника

Если оценка запускается на одного сотрудника, выберите его из списка и нажмите «Далее».

![](_images-01/image489.png)

Если оценка данного сотрудника уже проводилась, убедитесь, что все его карточки оценки имеют статус «Оценка завершена». Новую оценку нельзя запустить, пока не закончена предыдущая.

![](_images-01/image220.png)

### Настройка карточки оценки

На странице настройки вы найдете следующие поля для заполнения:

1. **Кого оцениваем.** Сотрудник, чьи компетенции будут оцениваться. В данное поле автоматически подтягивается сотрудник, указанный вами ранее.

2. **Руководители.** В это поле автоматически подставляется руководитель сотрудника, указанный в оргструктуре компании в Битрикс24. В оценке всегда должен участвовать пользователь в данной роли, поэтому поле обязательно для заполнения.

![](_images-01/image161.png)

Если оцениваемый — обычный сотрудник, то роль руководителя будет отдана руководителю его отдела. Если оцениваемый сам находится на руководящей должности, то в роли руководителя будет его вышестоящий руководитель. Кроме того, вы можете указать руководителя самостоятельно. Обратите внимание, что руководитель может быть только один.

![](_images-01/image425.png)

Перед запуском оценки карточку можно отправить на согласование руководителю, который указан в данном поле. Для этого поставьте галочку в нужном чекбоксе. Подробнее о согласовании написано [ниже](#soglasovanie-kartochki-otsenki).

![](_images-01/image341.png)

3. **Коллеги.** В это поле автоматически подставляются все коллеги оцениваемого сотрудника, указанные в оргструктуре компании в Битрикс24. По умолчанию в оценке должен участвовать хотя бы один коллега, поэтому поле обязательно для заполнения. В настройках это условие можно отключить.

![](_images-01/image462.png)

Если оцениваемый — обычный сотрудник, то коллегами будут сотрудники из его отдела. Если оцениваемый сам находится на руководящей должности, то в роли коллег будут другие руководители того же уровня.

![](_images-01/image41.png)

При необходимости вы можете добавить других пользователей в это поле. По умолчанию к оценке одного пользователя можно привлечь максимум 25 коллег. Изменить этот параметр можно в настройках модуля. Если у оцениваемого больше 25-ти коллег, то из них модуль автоматически выберет 25 пользователей с самым ранним ID.

Кроме того, оцениваемый может самостоятельно [пригласить коллег для своей оценки](#priglashenie-kolleg-k-otsenke). Для этого поставьте соответствующую галочку в чекбоксе. По умолчанию лимит для приглашения коллег — 2 человека. В настройках модуля можно изменить это ограничение.

![](_images-01/image310.png)

4. **Подчиненные.** Если сотрудник является руководителем, в это поле автоматически подтянутся подчинённые из его отдела. Вы также можете выбрать подчиненных по своему усмотрению. Настроить количество привлекаемых подчиненных можно в административной части портала. По умолчанию к оценке сотрудника можно привлечь максимум 25 подчиненных. Если у оцениваемого больше 25-ти подчиненных, то из них модуль автоматически выберет 25 пользователей с самым ранним ID.

![](_images-01/image172.png)

5. **Клиенты и контакты CRM.** С помощью данного поля к оценке можно привлечь контактов сотрудника из CRM. Например, его клиентов. Указанным контактам будет отправлена ссылка на участие в оценке. При необходимости это поле можно отключить в настройках модуля.

Выберите контакт из списка и заполните дополнительные поля:

- **Как отправить ссылку: по SMS или письмом почту.** В случае с SMS администратору портала понадобится сначала настроить SMS-шлюз.

- **Текст для E-mail.** Шаблон письма, которое придет на почту клиенту.

- **Текст для СМС.** Шаблон сообщение, которое придет на мобильный клиенту.

Обратите внимание, что контакты из CRM могут оценивать компетенции только из того раздела, который указан как раздел soft-компетенций в настройках модуля.

![](_images-01/image108.png)

6. **Что оцениваем.** В этом блоке нужно выбрать компетенции, которые будут оцениваться. Их можно заполнить автоматически из [нужного шаблона](../04-podgotovka-k-otsenke/06-shablony-kompetentsiy.md) или указать вручную.

При необходимости вы можете выбрать, кто будет оценивать компетенцию:

- Все оценивающие

- Только сотрудник и руководитель

- Сотрудник, руководитель и подчиненные

- Сотрудник, руководитель и коллеги

![](_images-01/image245.png)

Обратите внимание, что в оценке должны принять участие все сотрудники, указанные слева. Их роли важно учитывать при настройке того, кто оценивает компетенции.

1. **Какие вопросы задать.** При необходимости здесь HR-специалист может добавить открытые вопросы для респондентов. Ответы на открытые вопросы не влияют на оценки по компетенциям, но полезны для получения более объемной обратной связи о компетенциях сотрудника. На основе ответов HR-специалист может точнее определить сильные и слабые стороны сотрудника и запланировать дальнейшее развитие.

![](_images-01/image485.png)

Нажмите на кнопку «Выбрать вопросы» чтобы перейти к списку уже существующих вопросов, доступных для добавления в оценку.

![](_images-01/image275.png)

Отметьте галочкой те вопросы, которые нужно добавить в данную оценку. Ограничения на количество вопросов нет.

Если подходящего вопроса нет в списке, нажмите «Добавить вопрос», чтобы создать новый.

![](_images-01/image74.png)

В открывшемся слайдере укажите формулировку вопроса, а также, при необходимости, развернутое описание вопроса. Описание поможет респондентам лучше понять суть вопроса и дать более точный ответ.

![](_images-01/image502.png)

Сохраните изменения. Новый вопрос появится в списке и будет доступен для добавления в текущую оценку и во все последующие.

![](_images-01/image350.png)

Для вопросов, добавленных в оценку, можно настроить два параметра:

- **Отвечают.** Какие роли оценивающих будут отвечать на этот вопрос. Только этим пользователям вопрос будет виден в карточке оценки. Доступен выбор следующих ролей:

- Отвечают все

- Руководитель

- Сотрудник + Руководитель

- Сотрудник + Руководитель + Подчиненные

- Сотрудник + Руководитель + Коллеги

![](_images-01/image89.png)

- **Обязательность.** Вопрос можно сделать обязательным для указанных ролей пользователей. Без ответа на вопрос пользователи не смогут сохранить свою оценку. Доступен выбор следующих ролей:

- Не обязательно

- Обязательно для Руководителя

- Обязательно для Сотрудника + Руководителя

- Обязательно для всех

![](_images-01/image169.png)

При необходимости на слайдере выбора вопроса можно отредактировать или удалить вопрос. При удалении вопроса он не исчезнет из уже запущенных оценок и будет доступен респондентам для ответа.

![](_images-01/image166.png)

## Запуск оценки для группы сотрудников

Если оценка запускается на группу сотрудников, выберите из списка нужный отдел компании или группу пользователей и нажмите «Далее».

Обратите внимание, что у массового запуска оценки есть ряд особенностей:

- Оценка всех выбранных вами сотрудников будет производиться по одному и тому же набору компетенций.

- К такой оценке нельзя привлечь клиентов и контактов из CRM.

![](_images-01/image293.png)

Если у сотрудников из выбранного отдела/группы уже есть карточки оценки, убедитесь, что все они имеют статус «Оценка завершена». Новую оценку нельзя запустить, пока не закончена предыдущая.

![](_images-01/image382.png)

### Настройка карточки оценки

На странице настройки вы увидите таблицу со списком всех сотрудников выбранного подразделения, которые будут проходить оценку. Напротив них будут указаны оценивающие роли:

1. **Руководитель.** В ячейку автоматически подставляется руководитель сотрудника, указанный в оргструктуре компании в Битрикс24. В оценке всегда должен участвовать пользователь в данной роли, поэтому поле обязательно для заполнения.

Кроме того, вы можете указать руководителя самостоятельно. Обратите внимание, что руководитель может быть только один.

![](_images-01/image425.png)

По умолчанию в оценке должен участвовать хотя бы один коллега, поэтому поле обязательно для заполнения. В настройках это условие можно отключить.

3. **Подчиненные.** Если сотрудник является руководителем, то в это поле будут автоматически добавлены два подчиненных из его отдела. Вы также можете выбрать подчиненных по своему усмотрению. Настроить количество привлекаемых подчиненных можно в административной части портала.

По умолчанию один пользователь может пригласить максимум двух коллег. Изменить этот лимит можно в настройках модуля.

## Удаление и добавление оцениваемых

Первый шаг при массовом запуске оценки — выбор отдела или группы пользователей, которые будут проходить оценку. Но это не означает, что запустить групповую оценку можно только на конкретный отдел или группу. В процессе настройки вы можете сформировать группу сотрудников по своему усмотрению: удалить одних и добавить других.

![](_images-01/image239.png)

Для удаления сотрудника из таблицы достаточно убрать галочку рядом с его именем. А для добавления нужно нажать соответствующую кнопку и выбрать любого пользователя из списка. Благодаря этому группа сотрудников для массового запуска может быть любой. Она может включать только руководителей, только подчиненных, только сотрудников на одинаковых должностях и так далее.

## Дублирование оценивающих в разных ролях

Перед запуском оценки убедитесь, что один и тот же пользователь не указан в карточке в разных ролях. Например, оценивающий не может быть одновременно коллегой и подчиненным — это не позволит запустить оценку.

## Проверка данных перед запуском

При запуске оценки на одного и нескольких пользователей на экране появляется слайдер с данными для проверки. В нем собрана информация о параметрах новой оценки и загруженности сотрудников. При необходимости эту проверку можно отключить в [настройках модуля](../06-administrirovanie-modulya/01-nastroyki-modulya-otsenka-360.md).

![](_images-01/image453.png)

На слайдере выводится следующая информация:

1. **Сводная информация об оценке.** Здесь отображаются сроки проведения, количество участников и оцениваемых компетенций.

В этом блоке можно настроить срок проведения оценки. Для этого нажмите на количество дней в скобках и выберите новое значение. Доступны значения от 7 до 30. Именно столько дней будет длиться сбор оценок.

![](_images-01/image376.png)

Также в блоке есть опция для добавления тегов к запускаемым оценкам. Подробнее о тегах написано в [отдельной статье].

![](_images-01/image376.png)

2. **Аналитика по компетенциям.** В этом блоке выводятся настройки компетенций, которые будут оцениваться: наличие/отсутствие индикаторов, сколько индикаторов с весами/баллами.

![](_images-01/image432.png)

3. **Наиболее загруженные сотрудники.** В таблице отображаются все сотрудники, которые сейчас выбраны в роли оценивающих. Здесь HR-специалист может проанализировать, сколько карточек оценки уже находится у этих сотрудников в работе и как сильно они загружены.

4. **Роли и компетенции.** В этом блоке показано, какое количество компетенций оценивают сотрудники с определенной ролью.

![](_images-01/image119.png)

## Согласование карточки оценки

Если в карточке оценки включена такая опция, то перед запуском карточка отправляется на согласование руководителю. Ему приходит уведомление, в котором находится ссылка на данную карточку.

![](_images-01/image441.png)

Перейдя по ссылке, руководитель может по своему усмотрению изменить настройки оценки для данного сотрудника, а затем запустить либо отменить оценку. Обратите внимание, что поле для выбора руководителя будет отсутствовать в карточке.

![](_images-01/image125.png)

## Приглашение коллег к оценке

Если в карточке оценки включена такая опция, то сотрудник может привлечь к своей оценке двух любых коллег. Лимит приглашенных коллег устанавливается в настройках модуля.

Приглашение происходит уже после того как оценка запущена. В карточке оценки у пользователя отобразится поле для выбора коллег. При этом коллегами могут быть любые сотрудники компании на усмотрение оцениваемого. Обратите внимание, что изменить выбор коллег после сохранения карточки — нельзя.

![](_images-01/image163.png)

Выбранным коллегам придет стандартное уведомление с приглашением принять участие в оценке.
