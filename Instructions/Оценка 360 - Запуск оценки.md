# Запуск оценки

**Модуль:** [[Module - Оценка 360]]

## Описание

Инструкция описывает процесс запуска оценки 360 градусов — как для одного сотрудника, так и для группы сотрудников. Включает настройку карточки оценки: выбор оцениваемого, руководителя, коллег, подчинённых, клиентов из CRM, компетенций и открытых вопросов. Также рассматриваются проверка данных перед запуском, согласование карточки с руководителем и приглашение коллег к оценке.

## Ключевые функции

- Запуск оценки для одного сотрудника с полной настройкой карточки
- Массовый запуск оценки для группы/отдела сотрудников
- Автоматическое заполнение ролей (руководитель, коллеги, подчинённые) из оргструктуры Битрикс24
- Привлечение клиентов и контактов из CRM к оценке
- Выбор компетенций из шаблона или вручную с настройкой, кто их оценивает
- Добавление открытых вопросов с настройкой ролей отвечающих и обязательности
- Согласование карточки оценки с руководителем перед запуском
- Возможность для сотрудника самостоятельно приглашать коллег к своей оценке
- Проверка данных и загруженности сотрудников перед запуском
- Удаление и добавление оцениваемых при массовом запуске
- Контроль дублирования оценивающих в разных ролях

## Предусловия

- Модуль «Оценка 360» установлен и настроен на портале Битрикс24
- В оргструктуре компании указаны руководители и коллеги сотрудников
- У сотрудников, для которых запускается новая оценка, все предыдущие карточки оценки имеют статус «Оценка завершена»
- Для привлечения клиентов из CRM — настроен SMS-шлюз (если требуется отправка по SMS)
- Для массового запуска: определён отдел или группа пользователей

## Шаги

### Запуск оценки для одного сотрудника

1. Выберите сотрудника из списка и нажмите «Далее».
2. Убедитесь, что все предыдущие карточки оценки сотрудника имеют статус «Оценка завершена».
3. На странице настройки карточки оценки заполните поля:
   - **Кого оцениваем** — автоматически подтягивается выбранный сотрудник.
   - **Руководители** — автоматически подставляется руководитель из оргструктуры. При необходимости укажите руководителя вручную (только один). При необходимости включите опцию отправки на согласование руководителю.
   - **Коллеги** — автоматически подставляются коллеги из отдела. Можно добавить других пользователей (максимум 25). При необходимости включите опцию самостоятельного приглашения коллег оцениваемым (лимит: 2 человека).
   - **Подчиненные** — автоматически подтягиваются подчинённые, если сотрудник является руководителем (максимум 25).
   - **Клиенты и контакты CRM** — выберите контакты и настройте способ отправки ссылки (SMS или E-mail), текст письма и СМС.
   - **Что оцениваем** — выберите компетенции из шаблона или вручную. Настройте, кто будет оценивать каждую компетенцию.
   - **Какие вопросы задать** — добавьте открытые вопросы из списка или создайте новые. Настройте роли отвечающих и обязательность вопросов.
4. Нажмите «Далее» для перехода к проверке данных.

### Запуск оценки для группы сотрудников

1. Выберите отдел компании или группу пользователей и нажмите «Далее».
2. Убедитесь, что все карточки оценки выбранных сотрудников имеют статус «Оценка завершена».
3. На странице настройки в таблице для каждого сотрудника проверьте и при необходимости измените:
   - **Руководитель** (автоматически из оргструктуры, только один)
   - **Коллеги** (автоматически 2 коллеги, макс. 25)
   - **Подчиненные** (автоматически 2 подчинённых, если сотрудник — руководитель)
4. В блоках **«Что оцениваем»** и **«Какие вопросы задать»** настройте компетенции и вопросы (аналогично оценке для одного сотрудника).
5. При необходимости удалите или добавьте сотрудников в таблицу с помощью галочек и кнопки добавления.
6. При необходимости включите опцию согласования карточки с руководителем и/или опцию самостоятельного приглашения коллег.
7. Нажмите «Далее» для перехода к проверке данных.

### Проверка данных перед запуском

На слайдере проверки данных:
1. Проверьте **сводную информацию об оценке**: сроки (от 7 до 30 дней), количество участников и компетенций. При необходимости настройте срок проведения.
2. При необходимости добавьте теги к запускаемым оценкам.
3. Проверьте **аналитику по компетенциям**: наличие индикаторов, весов и баллов.
4. Проверьте **наиболее загруженных сотрудников** — таблицу с количеством карточек оценки у каждого оценивающего.
5. Проверьте **роли и компетенции** — сколько компетенций оценивает каждая роль.
6. Убедитесь в отсутствии **дублирования оценивающих в разных ролях** (один пользователь не может быть одновременно коллегой и подчинённым).
7. Запустите оценку.

## Ожидаемый результат

- Созданы карточки оценки для выбранных сотрудников с настроенными ролями, компетенциями и вопросами
- Руководитель, коллеги, подчинённые и (при необходимости) клиенты из CRM получают уведомления с приглашением к участию в оценке
- Если включено согласование — карточка отправляется на утверждение руководителю
- Запущен сбор оценок на указанный период (от 7 до 30 дней)

## Граничные случаи

- **Оценка уже проводилась:** новую оценку нельзя запустить, пока все предыдущие карточки не имеют статус «Оценка завершена»
- **Более 25 коллег/подчинённых:** модуль автоматически выбирает 25 пользователей с самым ранним ID
- **Руководитель может быть только один** в карточке оценки
- **Дублирование ролей:** один и тот же пользователь не может быть указан в разных ролях (например, коллега и подчинённый одновременно) — оценка не запустится
- **Массовый запуск:** все сотрудники оцениваются по одному набору компетенций; нельзя привлечь клиентов из CRM
- **Приглашение коллег:** после сохранения карточки изменить выбор приглашённых коллег нельзя
- **Удаление вопроса:** при удалении вопроса он не исчезает из уже запущенных оценок
- **Контакты CRM** оценивают только soft-компетенции (согласно настройкам модуля)
- **Проверка данных:** при необходимости проверку можно отключить в настройках модуля
- **Количество дней сбора оценок:** доступны значения от 7 до 30 дней
