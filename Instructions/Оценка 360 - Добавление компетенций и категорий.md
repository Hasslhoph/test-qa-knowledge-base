# Оценка 360 - Добавление компетенций и категорий

**Модуль:** [[Module - Оценка 360]]

## Описание

Инструкция по созданию модели компетенций — добавлению категорий, компетенций, их редактированию и удалению. Добавлять, редактировать и удалять компетенции и разделы могут только администраторы и пользователи из группы HR-отдела.

## Ключевые функции

### Добавление категорий
- Перейти в «Компетенции» → «Настройки» (⚙️) → «Категории» → «Добавить»
- Поля: Название (обязательное), Родительский раздел (для вложенности), Активность, Цвет (HEX-код)
- Вложенность категорий не ограничена, но рекомендуется не более 4 уровней

### Редактирование категорий
- Через бургер-меню (≡) → «Изменить»
- Аналогичная форма создания

### Удаление категорий
- Через бургер-меню (≡) → «Удалить»
- Удаляются все вложенные подразделы и компетенции

### Добавление компетенций
- Перейти в «Компетенции» → «Настройки» (⚙️) → «Добавить»
- Поля:
  1. **Активность** — влияет на отображение в публичных интерфейсах
  2. **Наименование** — обязательно для заполнения
  3. **Раздел** — категория, куда будет прикреплена компетенция (обязательное)
  4. **Сортировка** — очередность компетенций (наименьшее значение = первая)
  5. **Описание** — текстовое поле с информацией о компетенции
  6. **Картинка** — иллюстрация в карточке компетенции
  7. **Индикаторы** — критерии оценки компетенции

### Редактирование компетенций
- Из общего списка: бургер-меню (≡) → «Редактировать»
- Из карточки компетенции: «Настройки» (⚙️) → «Редактировать»

## Предусловия

- Права администратора или группы HR-отдела
- Созданы категории для привязки компетенций

## Граничные случаи

- Если установлен «Корпоративный университет», раздел «Компетенции» в Оценке 360 не отображается
- Неактивная компетенция скрыта из публичных интерфейсов, но видна в существующих оценках
- Все пользователи могут просматривать неактивную компетенцию по прямой ссылке
- Рекомендуемый уровень вложенности — не более 4 уровней (3 раздела + компетенции)
