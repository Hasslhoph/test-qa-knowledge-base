# Оценка 360 - Добавление компетенций и категорий

**Модуль:** [[Module - Оценка 360]]

# Добавление компетенций и категорий

В процессе Оценки 360 рабочее окружение сотрудника оставляет обратную связь о его компетенциях. Соответственно, работа с модулем начинается с создания модели компетенций. Рекомендации о том, как проработать такую модель даны в отдельной статье. Ниже вы найдете инструкцию о том, как добавить компетенции в модуль.

Добавлять, редактировать и удалять компетенции и разделы могут только администраторы и пользователи из группы HR-отдела.

{% note warning %}

Если помимо Оценки 360 на портале установлен модуль «Корпоративный университет», то раздел «Компетенции» в Оценке 360 отображаться не будет. Этот раздел можно будет найти в Корпоративном университете — через него производится управление  компетенциями, категориями и индикаторами в рамках обоих модулей.

{% endnote %}

## Добавление категорий

Каждая компетенция должна быть привязана к определенной категории, то есть к разделу. Поэтому для начала в модуль нужно добавить соответствующие категории компетенций, например, «Hard skills» и «Soft skills».

Для добавления новой категории перейдите в раздел модуля «Компетенции» и нажмите «Настройки» (⚙️) > «Категории». В открывшемся слайдере кликните на кнопку «Добавить».

![](_images-01/image219.png)

![](_images-01/image316.png)

В форме создания категории заполните следующие поля:

1. **Название.** Обязательное для заполнения поле, где указывается наименование новой категории.

2. **Родительский раздел.** Данное поле используется создания вложенности. Если в поле указать уже существующий раздел, то новый станет его подразделом. При создании корневого раздела в поле нужно оставить значение «Верхний уровень».

3. **Активность.** Опция, которая влияет на отображение категории. Если галочку убрать, то данный раздел вместе со всеми его подкатегориями и компетенциями будет скрыт от обычных пользователей. При этом привилегированным пользователям он по-прежнему будет доступен через интерфейс управления категориями.

4. **Цвет.** Поле для указания цвета раздела в виде [HEX кода](https://colorhunt.co/), например `#5FC863`. По умолчанию подкатегории будут наследовать указанный цвет.

![](_images-01/image231.png)

## Редактирование категорий

Для редактирования категории перейдите на слайдер «Управление категориями». Здесь нажмите на бургер-меню (≡) рядом с нужной категорией и выберите «Изменить». Вы увидите форму, аналогичную форме для создания новой категории. Внесите необходимые правки и сохраните изменения.

![](_images-01/image330.png)

![](_images-01/image42.png)

## Удаление категорий

Для удаления категории или ее подкатегории нажмите на бургер-меню (≡) слева от названия и выберите «Удалить». В открывшемся слайдере подтвердите удаление. Обратите внимание, что при удалении категории также будут стерты все вложенные в нее подразделы и компетенции.

![](_images-01/image240.png)

## Просмотр подкатегорий

Кликните на название категории, чтобы перейти к списку ее подкатегорий. Отсюда вы можете аналогичным образом создавать новые подразделы для данной категории, удалять уже существующие или менять их очередность.

![](_images-01/image314.png)

Вложенность категорий не ограничена. Это означает, что в модуле можно создать любое количество категорий с компетенциями. Однако стоит учитывать нюансы отображения разделов. Использование более четырех вложенных категорий может сказаться на адаптивности раздела компетенций (см. ниже).

![](_images-01/image333.png)

Рекомендуемый уровень вложенности — не более четырех уровней, когда первые три уровня это разделы, а четвертый — компетенции.

## Добавление компетенций

Перейдите в раздел «Компетенции» и нажмите «Настройки» (⚙️) > «Добавить».

![](_images-01/image420.png)

1. **Активность.** Опция влияет на отображение компетенции в публичных интерфейсах Оценки 360. Если галочка включена, компетенция видна всем пользователям во всех интерфейсах.

![](_images-01/image33.png)

Если галочка отключена, то компетенция будет скрыта изо всех публичных интерфейсов за исключением уже существующих оценок, куда компетенция была добавлена.

Администраторы и группа HR-отдела по-прежнему будут видеть неактивную компетенцию в разделе «Компетенции».

Все пользователи могут просматривать детальную страницу неактивной компетенции по прямой ссылке. На странице деактивированной компетенции выводится соответствующее предупреждение.

![](_images-01/image168.png)

2. **Наименование.** Название компетенции. Данное поле обязательно для заполнения.

![](_images-01/image473.png)

3. **Раздел.** Поле для указания разделы (категории), куда будет прикреплена компетенция. Поле также является обязательным.

![](_images-01/image322.png)

4. **Сортировка.** Опция влияет на очередность компетенций в различных интерфейсах Оценки 360. Компетенция с наименьшим значением сортировки будет первой, а с наибольшим — последней. Если сортировка одинаковая, компетенции будут выстроены в алфавитном порядке.

![](_images-01/image190.png)

5. **Описание.** Тестовое поле с основной информацией о компетенции.

6. **Картинка.** Иллюстрация, которая будет выводиться в карточке компетенции.

![](_images-01/image144.png)

7. **Индикаторы.** Критерии, по которым может оцениваться данная компетенция. Например, для компетенции «Постановка задач» индикаторами могут быть «Понимание приоритетов» и «Учет интересов команды». Подробнее о работе с индикаторами читайте в [отдельной статье](./02-dobavlenie-indikatorov-i-privyazka-k-kompetentsiyam.md).

![](_images-01/image527.png)

## Редактирование компетенций

Отредактировать компетенцию можно из двух мест:

1. **Из общего списка компетенций.** Для этого раскройте нужную категорию, нажмите на бургер-меню рядом (≡) с названием компетенции и выберите «Редактировать».

![](_images-01/image410.png)

2. **Из карточки компетенции.** В этом случае кликните на название компетенции и перейдите на ее страницу. Здесь нажмите «Настройки» (⚙️) > «Редактировать».

![](_images-01/image296.png)

![](_images-01/image529.png)

В обоих случаях откроется слайдер для редактирования компетенции. Скорректируйте нужные данные и сохраните изменения.

![](_images-01/image84.png)
