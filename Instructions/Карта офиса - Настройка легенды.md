# Настройка легенды

**Модуль:** [[Module - Карта офиса + Бронирование]]

## Описание

Инструкция по настройке легенды — списка условных обозначений на карте офиса, который отображается при наведении на иконку знака вопроса.

## Ключевые функции

- Добавление условных обозначений в легенду
- Настройка иконок: цвета бронирования, типы рабочих мест, произвольные иконки
- Сортировка записей в легенде
- Редактирование, копирование, удаление записей

## Предусловия

- Доступ к настройкам есть только у администраторов портала

## Добавление условных обозначений

1. Перейти: `адрес_портала/bitrix/admin/index.php?lang=ru`
2. Контент → Highload-блоки → Карта офиса: Легенда
3. Нажать «Добавить запись»

### Поля
1. **Название** — описание условного обозначения (допустимы HTML-теги)
2. **Сортировка** — порядок отображения (меньше = выше)
3. **Код иконки** — иконка одним из способов:
   - Через Font Awesome (код иконки)
   - Стандартные значения: `grey`, `red`, `yellow`, `green`, `spell`, `comp`
   - Через HTML-теги: `<div width="17" height="17" style="background: #000000"></div>`
4. **Изображение** — загрузка PNG/JPG файла 17x17 пикселей

## Редактирование и удаление

Нажать на бургер-меню рядом с записью:
- **Изменить** — страница редактирования
- **Копировать** — создание дубликата
- **Удалить** — безвозвратное удаление

## Граничные случаи

- При добавлении иконок цветов бронирования цвета должны соответствовать значениям в настройках стилей модуля
- Если сортировка не указана — элемент отображается первым
- При нескольких элементах без сортировки — отображаются по порядку ID (создания)
