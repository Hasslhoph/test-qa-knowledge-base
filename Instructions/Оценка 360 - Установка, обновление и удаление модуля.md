# Оценка 360 - Установка, обновление и удаление модуля

**Модуль:** [[Module - Оценка 360]]

## Описание

Установка, обновление и удаление модуля «Оценка 360» через Marketplace Битрикс24. Доступно только администраторам портала.

## Ключевые функции

- Установка модуля через прямую ссылку
- Обновление через Marketplace
- Активация купона на покупку/продление
- Удаление модуля

## Предусловия

- Права администратора портала

## Шаги

### Установка модуля

1. Перейдите: `адрес_портала/bitrix/admin/update_system_partner.php?addmodule=ithive.assessment360`
2. Модуль «Оценка 360» будет автоматически отмечен галочкой
3. Нажмите «Загрузить»
4. Подтвердите согласие с Лицензионным соглашением и Условиями конфиденциальности, нажмите «Применить»
5. Дождитесь завершения установки
6. Появится уведомление об успешной загрузке

### Обновление модуля

1. Marketplace > «Обновление решений» > «Список обновлений»
2. Выберите «Улей: Оценка 360»
3. Нажмите «Загрузить»
4. ИЛИ перейдите: `адрес_портала/bitrix/admin/update_system_partner.php`

### Активация купона

1. Marketplace > «Обновление решений» > «Активация купона»
2. Введите номер купона
3. Нажмите «Активировать купон»
4. ИЛИ перейдите: `адрес_портала/bitrix/admin/update_system.php?lang=ru`

### Удаление модуля

1. Marketplace > «Установленные решения» > «Доступные решения»
2. Бургер-меню рядом с решением > «Удалить»
3. После обновления страницы подтвердите удаление
4. Страница обновится, появится сообщение об удалении
5. ИЛИ перейдите: `адрес_портала/bitrix/admin/update_system_partner.php`

## Ожидаемый результат

Модуль установлен / обновлён / удалён.

## Граничные случаи

- **Активация купона на неустановленный модуль:** Marketplace > «Установленные решения» > «Купленные решения» > бургер-меню > «Загрузить»
- **Активация купона для модуля с истёкшей демо-версией:** «Обновление решений» > «Установка обновлений» > «Зарегистрировать решение»
- Права на установку, обновление, активацию купона и удаление есть только у группы «Администраторы»
