# Оценка 360 - Установка, обновление и удаление модуля

**Модуль:** [[Module - Оценка 360]]

# Установка, обновление и удаление модуля

## Установка модуля

**Шаг 1.**

Перейдите по ссылке: адрес_портала`/bitrix/admin/update_system_partner.php?addmodule=ithive.assessment360`

![](_images-03/image463.png)

Данная ссылка ведет в административную часть портала на вкладку «Список обновлений». Здесь находится список всех решений, доступных к установке и обновлению.

**Шаг 2.**

Модуль «Оценка 360» будет автоматически отмечен галочкой, когда вы перейдете по ссылке. Для установки модуля нажмите на кнопку «Загрузить».

![](_images-03/image202.png)

В этот момент откроется окно с Лицензионным соглашением.

**Шаг 3.**

В открывшемся окне подтвердите согласие с Лицензионным соглашением и Условиями конфиденциальности, затем нажмите кнопку «Применить». Установка модуля начнется автоматически.

![](_images-03/image344.png)

**Шаг 4.**

Дождитесь завершения установки модуля «Оценка 360».

![](_images-03/image447.png)

**Шаг 5.**

Установка модуля завершится, когда на экране появится уведомление об успешной загрузке. Также будут загружены все обновления модуля.

![](_images-03/image470.png)

{% note info "" %}

Права на установку модуля есть только у сотрудников, которые входят в группу пользователей «Администраторы».

{% endnote %}

## Обновление модуля

1. В левом меню административной части портала выберите раздел «Marketplace».

2. Перейдите в пункт «Обновление решений».

3. Нажмите на вкладку «Список обновлений».

4. Выберите решение «Улей: Оценка 360».

5. Нажмите на кнопку «Загрузить».

![](_images-03/image261.png) Также вместо шагов 1-3 вы можете сразу перейти на вкладку «Список обновлений» по ссылке: адрес_портала`/bitrix/admin/update_system_partner.php`

6. При успешном обновлении появится соответствующее уведомление.

![](_images-03/image493.png)

{% note info "" %}

Права на обновление модуля есть только у сотрудников, которые входят в группу пользователей «Администраторы».

{% endnote %}

## Активация купона на покупку/продление

1. В левом меню административной части портала выберите раздел «Marketplace».

2. Перейдите в пункт «Обновление решений».

3. Выберите вкладку «Активация купона».

4. Введите номер купона и нажмите кнопку «Активировать купон».

![](_images-03/image99.png)

Также вместо шагов 1-3 можно осуществить переход к вкладке «Активация купона» по ссылке адрес_портала`/bitrix/admin/update_system.php?lang=ru`

### Активация купона на неустановленный модуль

В случае активации купона на неустановленный модуль нужно:

1. В разделе «Marketplace» выбрать пункт «Установленные решения».

2. В списке «Купленные решения» сначала нажать на бургер-меню рядом с модулем, а затем на кнопку «Загрузить».

![](_images-03/image436.png)

### Активация купона для модуля, у которого истек срок демо-версии

Если вы ввели купон для модуля, у которого истек срок демо-версии, выполните следующие шаги:

1. Перейдите в раздел «Обновление решений».

2. Во вкладке «Установка обновлений» нажмите «Зарегистрировать решение».

![](_images-03/image30.png)

{% note info "" %}

Права на активацию купона и установку модуля есть только у сотрудников, которые входят в группу пользователей «Администраторы».

{% endnote %}


## Удаление модуля

1. В левом меню административной части портала выберите раздел «Marketplace».

2. Перейдите в пункт «Установленные решения».

3. Затем перейдите в блок «Доступные решения».

4. Слева от выбранного решения откройте бургер-меню и нажмите на кнопку «Удалить».

![](_images-03/image531.png)

Вместо шагов 1-2 вы можете перейти к пункту «Установленные решения» по ссылке: адрес_портала`/bitrix/admin/update_system_partner.php`

5. Подтвердите удаление модуля после обновления страницы.

![](_images-03/image402.png)

6. Страница будет еще раз обновлена, и вы увидите сообщение, что модуль удален.

![](_images-03/image495.png)

{% note info "" %}

Права на удаление модуля есть только у сотрудников, которые входят в группу пользователей «Администраторы».

{% endnote %}
