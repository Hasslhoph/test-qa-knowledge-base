# Оценка 360 - Установка, обновление и удаление модуля

**Модуль:** [[Module - Оценка 360]]

## Описание

Процедуры установки, обновления, активации купона и удаления модуля «Оценка 360» (ithive.assessment360). Все операции требуют прав администратора.

## Ключевые функции

### Установка модуля
1. Перейти по ссылке: адрес_портала`/bitrix/admin/update_system_partner.php?addmodule=ithive.assessment360`
2. Модуль автоматически отмечен галочкой → «Загрузить»
3. Подтвердить согласие с Лицензионным соглашением → «Применить»
4. Дождаться завершения установки
5. Уведомление об успешной загрузке

### Обновление модуля
1. Marketplace → «Обновление решений» → «Список обновлений»
2. Выбрать «Улей: Оценка 360» → «Загрузить»
3. Или напрямую: адрес_портала`/bitrix/admin/update_system_partner.php`

### Активация купона
1. Marketplace → «Обновление решений» → «Активация купона»
2. Ввести номер купона → «Активировать купон»
3. Или напрямую: адрес_портала`/bitrix/admin/update_system.php?lang=ru`

**Активация на неустановленный модуль:**
- Marketplace → «Установленные решения» → «Купленные решения» → бургер-меню → «Загрузить»

**Активация после истечения демо-версии:**
- «Обновление решений» → «Установка обновлений» → «Зарегистрировать решение»

### Удаление модуля
1. Marketplace → «Установленные решения» → «Доступные решения»
2. Бургер-меню → «Удалить»
3. Подтвердить удаление
4. Уведомление об успешном удалении

## Предусловия

- Права администратора портала

## Граничные случаи

- Все операции (установка, обновление, активация купона, удаление) требуют прав администратора
- При удалении модуля все данные об оценках будут потеряны
