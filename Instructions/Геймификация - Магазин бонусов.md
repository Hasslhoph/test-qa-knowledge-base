# Геймификация - Магазин бонусов

**Модуль:** [[Module - Геймификация]]

## Описание

Накопленные баллы каждый сотрудник может потратить в магазине бонусов. Магазин поддерживает материальные и нематериальные бонусы, фильтрацию по категориям, многоэтапное согласование покупок.

## Ключевые функции

- **Покупка позиций** — виджет слева показывает баланс; кнопка «Получить» при достаточном балансе; заблокирована с указанием сколько не хватает
- **Фильтры по категориям** — быстрый поиск бонусов
- **Последние покупки** — до 10 последних покупок других пользователей (отклонённые не выводятся)
- **Количественный учёт** — при покупке количество уменьшается на 1, при отклонении — возвращается
- **Согласование покупки** — 3 роли ответственных: за бюджет/списание, за исполнение заказа, сопровождающий; каждый этап с уведомлением
- **Действия ответственного** — утвердить списание, отклонить (баллы возвращаются), делегировать (одноразово, повторное невозможно)

## Предусловия

- Модуль установлен
- В магазине добавлены бонусы и категории
- Назначены ответственные за согласование

## Шаги (покупка)

1. Перейдите в раздел «Магазин бонусов»
2. При необходимости используйте фильтры по категориям
3. Нажмите «Получить» на нужном товаре
4. На слайдере проверьте название, картинку, описание, стоимость
5. Подтвердите заказ — баллы спишутся сразу
6. Дождитесь согласования ответственным

## Ожидаемый результат

Покупка оформлена, баллы списаны, ответственный получил уведомление о согласовании.

## Граничные случаи

- Если баллов недостаточно — кнопка покупки заблокирована
- При отклонении покупки баллы сразу возвращаются на счёт
- Делегирование заявки возможно только один раз
- Если включён количественный учёт и отключена «покупка при отсутствии» — кнопка заблокирована при нулевом/отрицательном количестве
