# Настройка уведомлений для новых сотрудников

**Модуль:** [[Module - Настройка уведомлений]]

## Описание

В разделе «Уведомления для новых сотрудников» выполняются настройки, которые будут применены для новых сотрудников в момент добавления их на портал. Настройки уведомлений будут автоматически применяться ко всем новым пользователям при добавлении их на портал.

В отличие от настроек для текущих сотрудников, здесь не требуется указание конкретных пользователей — настройки применяются автоматически ко всем новым сотрудникам.

## Ключевые функции

- **Автоматическое применение настроек** — при добавлении нового сотрудника на портал для него автоматически применяется заданный пресет настроек уведомлений
- **Настройка 18 видов уведомлений** для трёх каналов:
  - Веб-версия и десктоп-приложение
  - Электронная почта
  - Push-уведомления (мобильное приложение)
- **Три уровня настройки**: все уведомления, группа уведомлений, конкретное уведомление
- **Три значения настроек**: Установить, Снять, Недоступно

## Предусловия

- Модуль установлен и включён
- Пользователь имеет права на настройку уведомлений

## Шаги настройки

### Шаг 1. Перейдите в раздел «Уведомления для новых сотрудников»

Раздел доступен в публичной части модуля (левое меню Битрикс24 или настройки мессенджера → «Настроить для всех»).

### Шаг 2. Настройте уведомления

На странице модуля показаны все виды уведомлений, предусмотренные в Битрикс24:

1. Чат и звонки
2. Сообщения в Ленте
3. Календарь событий
4. CRM
5. Диск
6. Форум
7. Рейтинги, Лайки
8. Открытые линии
9. Интранет
10. Почта
11. Фотогалерея 2.0
12. Группы и проекты
13. Задачи
14. Учет рабочего времени
15. Wiki
16. Email-маркетинг
17. Бизнес-процессы
18. Опросы, голосования

Каждый раздел можно развернуть, чтобы увидеть конкретные уведомления, относящиеся к нему.

### Шаг 3. Выберите значение настройки

Для каждого уведомления выберите одно из трёх значений:

| Значение | Описание |
|----------|----------|
| **Установить** | Уведомления будут приходить новым сотрудникам |
| **Снять** | Уведомления не будут приходить новым сотрудникам |
| **Недоступно** | В Битрикс24 нет возможности изменить настройки данных уведомлений |

> **Важно:** В настройках для новых пользователей отсутствует значение «Не менять». Данное значение используется только при настройке уведомлений для текущих пользователей.

### Шаг 4. Сохранение настроек

Нажмите кнопку **«Применить»** в нижней части страницы. Настройки уведомлений будут сохранены и станут автоматически применяться для новых пользователей при добавлении их на портал.

### Сброс настроек

Кнопка **«Сбросить»** возвращает значения абсолютно всех уведомлений к значениям, которые были применены в самый последний раз. При этом настройки не будут сохранены — сброс влияет только на заполнение значений, а не на их применение.

## Граничные случаи

- После сохранения настроек значения в поле «Все уведомления» очищаются и поля отображаются пустыми. При обновлении страницы никакие значения по умолчанию не будут выбраны
- То же самое происходит со значениями для группы настроек — они очищаются после применения
- Значение «Не менять» отсутствует в этом разделе (в отличие от настроек для текущих сотрудников)
- Настройки для новых сотрудников применяются в момент их первой авторизации на портале
