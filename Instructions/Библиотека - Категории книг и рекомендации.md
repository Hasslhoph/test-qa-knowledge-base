# Категории книг и рекомендации

**Модуль:** [[Module - Библиотека]]

## Описание

Инструкция по работе с тематическими категориями книг, созданием подборок и настройкой раздела «Рекомендуемые книги». Категории используются как фильтры для поиска и для создания тематических подборок литературы.

## Ключевые функции

- **12 категорий по умолчанию:** Работа в команде, Про здоровье, Продажи, Управление персоналом, Системное мышление, Лидерство, Маркетинг, Нацеленность на результат, Бизнес литература, Корпоративное развитие, Художественная литература, Развитие подчинённых
- Создание, редактирование и удаление категорий
- Настройка цвета блоков с книжными подборками
- Раздел «Рекомендуемые книги» с настраиваемым количеством книг
- Доступ к подборкам из трёх мест: раздел «Категории», раздел «Книги» (поиск), карточка книги

## Предусловия

- Пользователь входит в группу «Модераторы предложенных книг» или «Администраторы» (для управления категориями)
- Для настройки рекомендаций — права администратора портала

## Шаги

### Создание новой категории

1. Перейдите в раздел «Категории» в Библиотеке
2. Нажмите «Настройки» (⚙️) → «Управление категориями»
3. Нажмите «Добавить»
4. Заполните поля:
   - **Название** — наименование категории (обязательное)
   - **Активность** — отображение категории на портале (по умолчанию активна)
   - **Цвет** — цвет блока с категорией
5. Сохраните изменения

### Редактирование и удаление категорий

1. На слайдере «Управление категориями» нажмите бургер-меню (≡) у нужной категории
2. Выберите «Редактировать» или «Удалить»
3. Сохраните изменения

### Добавление книги в рекомендации

1. Перейдите в административную часть: `адрес_портала/bitrix/admin/index.php`
2. «Контент» → «Библиотека» → «Библиотека»
3. Откройте раздел с нужной книгой, кликните на название
4. Во вкладке «Параметры» найдите поле «Выводить в рекомендации в категориях»
5. Установите значение «Y» и сохраните

### Настройка количества книг в рекомендациях

1. Перейдите: `адрес_портала/bitrix/admin/index.php`
2. «Настройки» → «Настройки модулей» → «Улей: Библиотека»
3. В поле «Количество книг в рекомендациях» введите нужное значение
4. Сохраните изменения

## Граничные случаи

- Если категория неактивна, она не выводится в Библиотеке, но книги остаются доступны в общем каталоге
- В рекомендациях отображается не более книг, чем задано в настройках количества
- Если рекомендованных книг больше, чем лимит — система выбирает книги с самой поздней датой добавления
- Права на вывод книг в рекомендации есть только у администраторов портала
