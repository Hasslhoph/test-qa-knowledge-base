# Legacy. Макет в Figma

**Модуль:** [[Module - Брендирование Битрикс24]]

## Описание

Инструкция по работе с интерактивным макетом Figma для дизайна «Legacy». Макет позволяет визуально создать оформление портала и затем импортировать стили через плагин.

## Ключевые функции

### Копирование макета

- Ссылка на макет: [\[S\] Брендирование Б24 / Конструктор "Classic"](https://www.figma.com/design/8RbvYj4ik9joBkzNeKMJSW/Hive-Branding-UI-Library-v.1.0.0?node-id=2001-5909&p=f&t=Khi5HXkroQI8NvDl-0)
- Пароль для доступа: **wehive**
- Копирование через «Duplicate to your drafts»
- Редактирование в браузере или десктопном приложении, скачивание только через десктопное приложение

### Обзор макета

Три шаблона:
1. **Упрощенный** — 5 цветовых настроек для быстрого дизайна
2. **Детальный** — максимальная проработка всех элементов
3. **Шаблон (Orange)** — для копирования цветов из брендбука (с примером)

Каждый макет состоит из трёх страниц Битрикс24 с учётом всех изменяемых элементов.

### Редактирование через Variables

- Variables — набор переменных, изменение одной применяется ко всем аналогичным элементам
- Меню открывается кликом на пустое пространство → Variables
- Структура меню идентична настройкам модуля «Брендирование»

### Элементы, недоступные в Figma

- Все настройки раздела «Авторизация»
- Шрифт
- Стили текста
- Скрыть/показать виджеты
- Загружаемые элементы: фавикон, наборы иконок

## Предусловия

- Аккаунт в Figma (бесплатная регистрация)
- Десктопное приложение Figma для скачивания стилей

## Граничные случаи

- **Приоритет настроек виджетов отличается**: в Figma общие настройки перекрывают индивидуальные (в модуле — наоборот). Для отображения индивидуальных настроек в Figma нужно установить прозрачность общих настроек в 0%
- Не рекомендуется удалять/добавлять Variables и элементы шаблона — это может повлиять на корректность экспорта
- Отмена изменений: Ctrl+Z / Command+Z (Mac)
