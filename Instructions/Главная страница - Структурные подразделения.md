# Структурные подразделения

**Модуль:** [[Module - Главная страница]]

## Описание

Блок на странице «О компании» для знакомства сотрудников с отделами компании и их руководителями.

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- **Разделы (департаменты)** — кнопки для переключения между направлениями деятельности
- **Список отделов** слева в каждом разделе
- **Детальное описание** отдела и профиль руководителя справа
- **Бегунок прокрутки** при большом объёме контента
- **Создание/редактирование разделов и отделов**
- **Переименование и изменение очередности** через [[Главная страница - Управление блоками]]

## Предусловия

- Права администратора или пользователя с правами на редактирование инфоблока
- Включённая административная панель (режим правки)

## Шаги

### Создание раздела

1. `/bitrix/admin/` → «Контент» > «О компании» > «Структурные подразделения» > «Моя компания»
2. Нажмите «Добавить элемент»
3. Введите название раздела (например, «Департамент по развитию»)
4. Сохраните

### Создание отдела

1. Откройте нужный раздел
2. Нажмите «Добавить элемент»
3. Вкладка «Параметры»:
   - Активность, Название, Сортировка
   - Сотрудник (руководитель отдела, привязка к профилю Битрикс24)
4. Вкладка «Анонс»:
   - Картинка для анонса (150x150)
   - Описание для анонса (информация об отделе)
5. Сохраните

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- Фото руководителя подтягивается из профиля Битрикс24 (можно загрузить своё)
- При большом объёме текста появляется бегунок прокрутки
