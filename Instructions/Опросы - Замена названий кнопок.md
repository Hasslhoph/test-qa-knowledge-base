# Замена названий кнопок «Голосовать» и «Переголосовать»

**Модуль:** [[Module - Опросы]]

## Описание

Инструкция по кастомизации названий кнопок «Голосовать» и «Переголосовать» на странице со списком опросов. Для каждой группы опросов могут быть разные названия.

## Ключевые функции

- Замена названий кнопок только на главной странице модуля (на странице голосования и в других интерфейсах названия остаются стандартными)
- Каждая группа опросов может иметь свои названия кнопок
- Выполняется через создание собственного языкового файла

## Предусловия

- Пользователь — администратор портала
- Созданы группы опросов с известными символьными идентификаторами

## Шаги

### Создание языкового файла

1. Перейти в административную часть: `адрес_портала/bitrix/admin/`
2. «Контент» → «Структура сайта» → «Файлы и папки» → «Добавить» → «Добавить папку»
3. Создать цепочку папок: `local` → `php_interface` → `user_lang` → `ru`
4. Итоговый путь: `/local/php_interface/user_lang/ru`

### Создание файла lang.php

1. Открыть текстовый редактор кода (например, Notepad++)
2. Создать новый файл с содержимым:

```php
<?php
$MESS["/bitrix/components/ithive/voting.list/templates/ithive.default/lang/ru/template.php"]["LABEL_VOTE_BUTTON_CORPORATE_LIFE"] = 'Название для кнопки Голосовать';

$MESS["/bitrix/components/ithive/voting.list/templates/ithive.default/lang/ru/template.php"]["LABEL_REVOTE_BUTTON_CORPORATE_LIFE"] = 'Название для кнопки Переголосовать';
?>
```

3. Вместо `CORPORATE_LIFE` указать символьный код нужной группы опросов (можно узнать: «Сервисы» → «Опросы» → «Дополнительно» → «Группы опросов» → колонка «Символьный идентификатор»)
4. Для нескольких групп — продублировать строки с разными идентификаторами
5. Сохранить файл в кодировке UTF-8 с именем `lang.php`
6. Загрузить файл в папку `/local/php_interface/user_lang/ru` через административную часть портала

### Редактирование фраз

1. Перейти в папку `/local/php_interface/user_lang/ru`
2. Бургер-меню (≡) рядом с `lang.php` → «Редактировать как php»
3. Изменить фразы → «Сохранить»

## Ожидаемый результат

Названия кнопок «Голосовать» и «Переголосовать» изменены для указанных групп опросов на странице со списком опросов.

## Граничные случаи

- Замена работает только на странице со списком опросов (главная страница модуля)
- На странице голосования и в других интерфейсах названия кнопок остаются стандартными
- Файл должен быть сохранён в кодировке UTF-8 и назван именно `lang.php`
