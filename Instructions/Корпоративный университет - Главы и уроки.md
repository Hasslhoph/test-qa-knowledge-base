# Главы и уроки

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Структура онлайн-курса: главы, уроки, тесты и вопросы.

## Ключевые функции

### Основные понятия

- **Глава** — тематический раздел с дочерними элементами
- **Урок** — отдельная страница с контентом (урок с вложенными уроками становится главой)
- **Тест** — инструмент оценки знаний
- **Вопрос** — элемент теста

### Добавление урока/главы

1. **Из описания курса** — шестерёнка (⚙️) → «Добавить главу/урок»
2. **Из текущего урока** — шестерёнка → «Добавить урок/главу»
   - На один уровень с текущим уроком
   - Вложенный в текущий урок

### Поля слайдера

- **Название** (обязательное)
- **Символьный код** — уникальная ссылка
- **Предпросмотр страницы** — обложка урока/главы

### Наполнение контентом

- Кнопка «Добавить блок» → слайдер с блоками (текст, изображения, видео, интерактивные элементы)

### Редактирование урока/главы

- «Редактировать» → режим редактирования контента
- Шестерёнка → изменение названия, символьного кода, обложки
- Опция «Снять скрин» — скриншот верхней части для превью

### HTML-блок

- Раздел «Другое» → «Онлайн-курс: HTML-код»
- Позволяет добавлять CSS-стили, HTML-теги, скрипты
- Загрузка видео через HTML-блок (код с тегом `<video>`)

## Предусловия

- Права: администраторы, HR-отдел, авторы курсов, пользователи с правами

## Граничные случаи

- Фото с другого домена не отображаются на скриншоте превью
- Представленный HTML-код для видео не адаптирован для мобильного приложения
