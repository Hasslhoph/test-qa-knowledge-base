# Настройки модуля Настройка уведомлений

**Модуль:** [[Module - Настройка уведомлений]]

## Описание

Основные настройки модуля «Настройка уведомлений для всех» выполняются в административной части портала. Доступ к данным настройкам есть только у администраторов.

Перейти в административную часть: адрес_портала`/bitrix/admin/settings.php?lang=ru&mid=ithive.imsettingsforall`

## Ключевые функции

### Основные настройки

| № | Настройка | Описание |
|---|-----------|----------|
| 1 | **Вкладки сайтов** | Если на портале создано несколько сайтов (департаментов), то в настройках будет выводиться отдельная вкладка для каждого из них. Перечень настроек полностью дублируется в каждой вкладке |
| 2 | **Модуль включён** | Опция влияет на доступ к модулю в публичной части портала. Если модуль отключён, то у пользователей с правами на настройку уведомлений не будет доступа к разделам с настройками |
| 3 | **Экспертный режим** | Опция влияет на доступ к управлению экспертным режимом в публичной части модуля. Если галочка стоит — доступ есть, когда галочка не стоит — доступ отсутствует |
| 4 | **Логирование** | Если галочка активна, то при каждом сохранении настроек уведомлений в административной части будет формироваться текстовый log-файл со списком пользователей, для которых были применены настройки |
| 5 | **ID пользователя для настроек по умолчанию** | Из профиля указанного пользователя подтягиваются настройки «по умолчанию» для уведомлений. Если ID не указан — настройки подтягиваются из профиля текущего пользователя |
| 6 | **Группы пользователей** | В данном поле нужно выбрать одну или несколько групп пользователей, которым будут доступны настройки уведомлений для всех в публичной части портала. По умолчанию выбрана группа «Администраторы» |

### Дополнительные настройки уведомлений

Данные настройки доступны только администраторам и применяются сразу для всех текущих сотрудников:

1. Отключить уведомления о днях рождения сотрудников
2. Отключить уведомления о вступлении в группу
3. Включить уведомления о посте участника группы
4. Включить автоматическое сообщение о новом сотруднике в общий чат
5. Включить автоматическое сообщение об увольнении сотрудника в общий чат
6. Отключить уведомление «Я с вами в команде!»
7. Отключить появление пользователя в списке (чат как список контактов)
8. Отключить уведомление в общий чат о том, что пользователь скопировал ссылку на приглашение новых сотрудников
9. Отключить уведомление о том, что пользователь А пригласил пользователя Б в общий чат
10. Отключить уведомление о том, что пользователь А зарегистрировал пользователя Б в общий чат
11. Отключить уведомление о том, что сотрудник покинул чат
12. Отключить уведомление о том, что сотрудник присоединился к чату

## Предусловия

- Права администратора портала Битрикс24
- Модуль установлен

## Шаги доступа к настройкам

### Шаг 1. Перейдите в административную часть

Откройте ссылку: адрес_портала`/bitrix/admin/settings.php?lang=ru&mid=ithive.imsettingsforall`

### Шаг 2. Настройте основные параметры

На вкладке «Основные настройки» configure:
- Включение/отключение модуля
- Доступ к экспертному режиму
- Логирование
- ID пользователя для настроек по умолчанию
- Группы пользователей с доступом к модулю

### Шаг 3. Настройте дополнительные уведомления

В нижней части настроек модуля расположены дополнительные настройки уведомлений, которые применяются сразу для всех текущих сотрудников.

## Граничные случаи

- Если никакая группа пользователей не выбрана в поле «Группы пользователей», то публичная часть модуля недоступна даже администраторам
- Множественный выбор групп доступен через зажатую клавишу «Ctrl» (⌘ Command на Mac)
- Log-файлы формируются при сохранении настроек для текущих сотрудников, для новых сотрудников (в момент их первой авторизации) и для настроек экспертного режима
- Файл логов доступен для открытия через текстовый редактор, например Блокнот или Notepad++
