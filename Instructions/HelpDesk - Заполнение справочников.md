# HelpDesk - Заполнение справочников

**Модуль:** [[Module - HelpDesk]]

## Описание

Справочники — наборы свойств, используемых в обращениях HelpDesk. Справочники являются частью стандартного модуля Битрикс24 «Техподдержка». В HelpDesk используются 5 справочников: Категории, Критичности, Статусы, Оценки ответов, Сложность. Доступ — только у администраторов портала и администраторов ТП.

## Ключевые функции

### 1. Категории
- Техническая функция: автоматизация назначения ответственных
- Привязываются к разделам мастера обращений
- Создание: Сервисы → Техподдержка → Справочники → Категории
- Поля: Тип, Сайты (обязательно), Сортировка, Заголовок, Описание, Ответственный
- При изменении заголовка — меняется во всех интерфейсах; при смене ответственного — только для новых обращений

### 2. Критичности
- Отражает срочность и важность обращения (Низкая, Средняя, Высокая)
- Указывается автором при создании, может быть изменена сотрудником ТП
- Опция «Выбирать по умолчанию» — автовыбор при создании
- Информационный характер, помогает выстраивать приоритеты

### 3. Статусы
- Этапы работы над обращением (Новое, На рассмотрении, В работе, Завершено)
- Новому обращению автоматически присваивается первый статус
- Сортировка влияет на порядок в строке статусов и на работу опции редактирования в первом статусе
- Менять статусы могут только сотрудники ТП

### 4. Оценки ответов
- Оценки при закрытии обращения (числа, фразы, эмодзи)
- Появляется только у автора при закрытии заявки
- Используется для контроля качества работы ТП

### 5. Сложность
- Уровни сложности обращений
- Оценивается сотрудником ТП при рассмотрении
- Используется в аналитике для контроля времени решения

## Предусловия

- Права администратора портала или администратора ТП
- Путь: `/bitrix/admin/` → Сервисы → Техподдержка → Справочники

## Граничные случаи

- Удаление категории убирает привязку к виду обращения — автоматическое назначение ответственных перестаёт работать
- Удаление статуса, указанного в настройке редактирования в первом статусе, убирает возможность редактирования
- При удалении элемента справочника он удаляется из всех созданных обращений
