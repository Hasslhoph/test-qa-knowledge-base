# Создание нового блока

**Модуль:** [[Module - Главная страница]]

## Описание

Инструкция по созданию новых блоков на странице «О компании» (например, блок типа «Медиабанк компании» или «Команда профессионалов»).

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- **Создание нового инфоблока** для хранения контента
- **Настройка инфоблока:** создание раздела сайта, привязка к странице
- **9 типов блоков** на выбор (определяют внешний вид и способ редактирования)
- **Навигация:** вывод блока в меню навигации страницы «О компании»
- **Текст сноски** с переходом в другой раздел

## Предусловия

- Права администратора портала
- Доступ к административной части (`/bitrix/admin/`)

## Шаги

### Создание нового инфоблока

1. `/bitrix/admin/` → «Контент» > «Инфоблоки» > «Типы инфоблоков» > «О компании»
2. Нажмите «Добавить инфоблок»
3. Вкладка «Параметры»:
   - Символьный код (например, `products_and_services`)
   - Символьный код API (без спецсимволов и пробелов)
   - Сайты (галочка для нужного сайта)
   - Название (тематика, например «Продукты и услуги»)
   - Уберите галочку «Индексировать разделы для модуля поиска»
4. Вкладка «Доступ»: поставьте галочку «Расширенное управление правами»
5. Сохраните

### Настройка нового инфоблока

1. Кликните на название нового инфоблока
2. Нажмите «Добавить раздел» → создайте раздел с названием текущего сайта
3. Перейдите: «Контент» > «Управление блоками» > «Моя компания» → «Добавить элемент»
4. Заполните:
   - Название (заголовок блока на странице)
   - Вывести в навигации (да/нет)
   - Название навигации (для меню)
   - Откуда выводить данные (выберите созданный инфоблок и сайт)
   - Текст сноски (опционально)
   - Ссылка сноски (опционально)
   - Тип блока (один из 9 типов)
5. Сохраните

### Добавление контента

1. «Контент» > «О компании» → кликните на созданный инфоблок
2. Откройте раздел «Моя компания»
3. Нажмите «Добавить элемент» и заполните поля (зависят от типа блока)
4. Сохраните

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- Без создания папки с названием сайта контент не будет отображаться на странице
- Количество и тип полей зависит от выбранного типа инфоблока
- Контент отображается на странице сразу после добавления
