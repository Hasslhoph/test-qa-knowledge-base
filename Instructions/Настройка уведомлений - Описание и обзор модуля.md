# Описание и обзор модуля Настройка уведомлений

**Модуль:** [[Module - Настройка уведомлений]]

## Описание

Модуль «Настройка уведомлений для всех» (`ithive.imsettingsforall`) позволяет индивидуально и массово управлять уведомлениями в Битрикс24. Решение даёт возможность отключить лишние уведомления для отдельных пользователей или целых отделов, чтобы снизить информационный шум.

Концепция модуля основана на потребности крупных компаний упорядочить потоки оповещений сотрудников. Пользователи корпоративного портала часто получают массу нерелевантных уведомлений через десктоп, мобильное приложение и почту. В таком потоке важные уведомления теряются, а рабочие процессы выглядят перегружено.

**Технические данные:**
- Модуль совместим с PHP версией 8.1 и выше
- Модуль совместим с СУБД: MySQL, PostgreSQL
- Модуль поддерживает английский язык
- Модуль поддерживает многодепартаментность
- Функционал модуля доступен через современный десктопный браузер и мобильное приложение Б24
- Инструкция актуальна для версии модуля: **4.3.0**

## Ключевые функции

- **Гибкая настройка уведомлений** — доступна как точечная, так и массовая настройка уведомлений. Можно включить отправку сразу всех видов push-уведомлений или выбрать конкретные разделы: Чаты, Календарь, Группы и проекты и т.д.
- **Настройка уведомлений для текущих пользователей** — уведомления можно в пару кликов настроить любым выбранным сотрудникам, отделам и рабочим группам
- **Настройка уведомлений для новых пользователей** — можно заранее включить/отключить те или иные уведомления для новых пользователей. При добавлении нового сотрудника на портал для него будет автоматически применён ранее созданный пресет настроек
- **Управление экспертным режимом** — модуль позволяет управлять экспертным режимом (интерфейс, при котором уведомления о задачах не дублируются в ленте) и включать его для нужных сотрудников, отделов и рабочих групп
- **Логирование** — модуль может автоматически формировать текстовые файлы со списком пользователей, для которых были изменены настройки уведомлений

## Предусловия

- Портал Битрикс24 с установленным модулем `ithive.imsettingsforall`
- Права администратора или пользователя из группы, которой доступен функционал модуля в публичной части
- PHP 8.1+

## Где доступен модуль

Модуль виден только пользователям с правами на настройку уведомлений. Доступен в двух местах:

1. **В левом меню Битрикс24**
2. **В настройках мессенджера** (адрес_портала`/services/`) по клику на кнопку «Настроить для всех»

В публичной части модуль включает три раздела:
1. **Настройка уведомлений для текущих сотрудников** — массовая и индивидуальная настройка для сотрудников, которые уже присутствуют на портале
2. **Настройка уведомлений для новых сотрудников** — единые настройки, которые будут применяться для всех новых сотрудников при добавлении на портал
3. **Экспертный режим** — массовая и индивидуальная настройка экспертного режима в ленте новостей

В административной части доступны дополнительные настройки по пути: адрес_портала`/bitrix/admin/settings.php?lang=ru&mid=ithive.imsettingsforall`

## Граничные случаи

- Экспертный режим доступен в Битрикс24 только до версии intranet 25.900.0 (обновление «Зефир»). Начиная с этого обновления данный режим не используется и уведомления о задачах по умолчанию не выводятся в ленте
- Если никакая группа пользователей не выбрана в настройках модуля, то публичная часть модуля недоступна даже администраторам
- Настройки по умолчанию берутся из профиля пользователя с ID 1. Если пользователь не указан — из профиля текущего пользователя
