# Описание и возможности модуля

**Модуль:** [[Module - Главная страница]]

## Описание

Главная страница — это модуль, который добавляет на корпоративный портал две основные страницы: «Главная» и «О компании». «Главная» становится стартовой страницей корпоративного портала для всех сотрудников, объединяет в себе корпоративные сервисы и полезные инструменты и обеспечивает доступ к ним в режиме единого окна.

Страница «О компании» выполняет роль визитной карточки организации, где представлены ее ценности, структура, контактная информация и другие детали.

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- **Упрощение навигации** — создание единой стартовой страницы с востребованными инструментами (виджеты, баннеры, новости, сервисы, фото/видео галерея)
- **Адаптация новых сотрудников** — страница «О компании» знакомит с ценностями и историей организации, блок кадровых изменений показывает обновления в команде
- **Создание единого канала для новостей** — разделы «Новости компании» и «Новости СМИ» с возможностью таргетирования, лайков, комментариев
- **Вовлечение сотрудников** — фото/видео альбомы, доска почета (Лидеры продаж)
- **Создание информационных баннеров** — карусель баннеров с картинками и видео, таргетирование по сотрудникам
- **Создание раздела «О компании»** — визитная карточка организации с настраиваемыми блоками
- **Публикация новостей компании** — два новостных раздела с возможностью дублирования в ленту
- **Таргетирование новостей** — видимость публикаций для конкретных отделов/групп
- **Вывод всех сервисов в одном месте** — централизация бизнес-процессов Битрикс24
- **Отслеживание кадровых изменений** — прием, перевод, увольнение сотрудников (интеграция с 1С:ЗУП)
- **Ведение фото и видеоальбома** — галерея корпоративных событий
- **Создание списка отличившихся сотрудников** — доска почета с возможностью переименования
- **Управление виджетами** — Опросы, Дни рождения, Ссылки, Ближайшие события, Пульс компании, Мои задачи, Важные сообщения, Бизнес-процессы

## Предусловия

- Установленный модуль «Главная страница» (ithive.homepage)
- Права администратора для настройки блоков
- Режим правки (административная панель) для редактирования контента

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- Интеграция с модулями Улья: Корпоративный университет, Оценка 360, Управление целями, Геймификация, HelpDesk, Карта офиса + Бронирование, Опросы
- При установке модуля «Брендирование Битрикс24» Главная страница автоматически стилизуется под фирменный стиль
