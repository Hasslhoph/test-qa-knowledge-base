# Оценка 360 - Шаблоны компетенций

**Модуль:** [[Module - Оценка 360]]

# Шаблоны компетенций

Модуль позволяет заранее создавать наборы компетенций, по которым будет производиться оценка. Делать это могут привилегированные пользователи в разделе «Шаблоны».

![](_images-06/4.png)

При запуске оценки из выбранного шаблона автоматически подтянутся те компетенции, которые респонденты должны будут оценить.

## Добавление шаблона

Для создания нового шаблона нажмите на кнопку «Добавить».

![](_images-06/image439.png)

В открывшемся слайдере заполните следующие поля:

1. **Название.** Наименование шаблона. К примеру, шаблон может называться «Оценка руководителей проектов». В этот шаблон будут входить те компетенции, которыми должны владеть сотрудники в данной должности.

![](_images-06/image397.png)

2. **Компетенции**. Отметьте галочками компетенции, которые будут оцениваться в рамках данного шаблона. Например, если шаблон составляется для оценки руководителей проектов, в него можно включить такие компетенции как «Архитектура Битрикс24», «Сметный процесс» и «Проектирование».

![](_images-06/image40.png)

Сохраните изменения. Новый шаблон сразу же отобразится в списке.

![](_images-06/image409.png)

## Редактирование и удаление шаблона

Чтобы отредактировать шаблон наймите на бургер-меню (≡) рядом с нужным названием и выберите «Изменить». Скорректируйте информацию на слайдере и сохраните изменения.

![](_images-06/image140.png)

![](_images-06/image194.png)

Для удаления шаблона при нажатии на бургер-меню выберите «Удалить». Кроме того, вы можете выделить галочкой несколько нужных шаблонов и нажать на кнопку «Удалить».

![](_images-06/image203.png)

![](_images-06/image468.png)

Изменение и удаление шаблонов не отразится на уже запущенных карточках оценки.

## Использование шаблонов

Выбор шаблона происходит на этапе запуска оценки. Для этого в выпадающем меню кликните на нужный шаблон компетенций. Все компетенции, добавленные в шаблон автоматически подтянутся в карточку оценки.

![](_images-06/image118.png)

![](_images-06/image412.png)

Между шаблонами можно свободно переключаться — чтобы сменить набор компетенций достаточно выбрать другой шаблон.

![](_images-06/image185.png)

![](_images-06/image415.png)

Если набор компетенций из шаблона требуется изменить, сделать это можно прямо в карточке оценки. Для этого уберите и/или добавьте нужные галочки.
