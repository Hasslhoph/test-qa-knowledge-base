# Оценка 360 - Шаблоны компетенций

**Модуль:** [[Module - Оценка 360]]

## Описание

Шаблоны — заранее созданные наборы компетенций для оценки. При запуске оценки из шаблона автоматически подтягиваются компетенции. Доступно привилегированным пользователям в разделе «Шаблоны».

## Ключевые функции

### Добавление шаблона
- Раздел «Шаблоны» → «Добавить»
- Поля: Название (обязательное), Компетенции (галочками отметить нужные)
- Пример: шаблон «Оценка руководителей проектов» с компетенциями «Архитектура Битрикс24», «Сметный процесс», «Проектирование»

### Редактирование шаблона
- Бургер-меню (≡) → «Изменить»
- Корректировка данных на слайдере → сохранение

### Удаление шаблона
- Бургер-меню (≡) → «Удалить»
- Массовое удаление: выделить галочками → «Удалить»
- Изменение/удаление не влияет на уже запущенные карточки оценки

### Использование шаблонов
- Выбор шаблона при запуске оценки из выпадающего меню
- Компетенции автоматически подтягиваются в карточку
- Свободное переключение между шаблонами
- Набор компетенций можно изменить прямо в карточке оценки

## Предусловия

- Права на запуск и настройку оценки (администраторы, HR-отдел)
- Созданы компетенции для включения в шаблон

## Граничные случаи

- Изменение/удаление шаблона не отражается на уже запущенных оценках
- Шаблоны видны только привилегированным пользователям
