# Подведение итогов

**Модуль:** [[Module - Управление целями. Каскад]]

## Описание

Подведение итогов по достижению целей после завершения процесса и заполнения всех карточек оценок. Выполняется в разделе «Сводная».

## Ключевые функции

> Анализ результатов достижения целей.

- Сводный отчет с итоговыми значениями прогресса
- Средний процент достижения по подразделению
- Процент достижения конкретной цели конкретным подразделением
- Общий процент достижения каждой цели
- Процент достижения индивидуальных целей
- Оценка индивидуального прогресса сотрудника

## Сводная по целям

Раздел «Сводная» — сводный отчет с итоговыми значениями:

- Средний процент достижения целей по подразделению
- Процент достижения конкретной цели конкретным подразделением
- Общий процент достижения каждой цели
- Процент достижения индивидуальных целей в подразделениях

### Особенности

- Процент достижения выводится по целям, а не по критериям
- Критерии задают направление движения и помогают оценивать прогресс
- Модуль воспринимает цель как одну единицу
- Разный прогресс по одной цели в разных карточках сводится к среднему значению

## Оценка индивидуального прогресса

- Сводный отчет — массовый контроль, общая картина
- Для оценки вклада конкретного сотрудника — просмотр его карточки оценки
- На основании индивидуального прогресса принимается решение о финансовой мотивации

## Предусловия

- Завершенный процесс
- Заполненные и утвержденные карточки оценок

## Граничные случаи

- Цель воспринимается как одна единица даже если добавлена в разные карточки
- Критерии не учитываются в сводном отчете — только цели
