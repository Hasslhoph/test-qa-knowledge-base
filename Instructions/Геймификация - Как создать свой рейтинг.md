# Геймификация - Как создать свой рейтинг

**Модуль:** [[Module - Геймификация]]

## Описание

Инструкция по созданию новых рейтингов в модуле «Геймификация». Поддерживаются два вида рейтингов: автоматические (с папкой логики) и ручные (без папки логики).

## Ключевые функции

- **Автоматические рейтинги** — имеют папку с логикой, пересчитываются ежесуточно, ежемесячно начисляют баллы призерам, участвуют в Общем рейтинге
- **Ручные рейтинги** — без папки логики, баллы начисляются вручную, не пересчитываются ежесуточно, не участвуют в Общем рейтинге
- Параметры рейтинга: опубликовано, сортировка, название, описание, иконка (PNG/JPEG/SVG), видят рейтинг (подразделения), участвуют в рейтинге (подразделения)
- Для автоматических рейтингов: папка с логикой расчета, коэффициент веса в Общем рейтинге, активность, автоначисление

## Предусловия

- Права администратора портала
- Доступ к административной части портала (`/bitrix/admin/`)

## Шаги (ручной рейтинг)

1. Перейдите в административную часть портала → Настройки → Улей: Геймификация
2. Вкладка «Общие настройки» → таблица с рейтингами → «Добавить параметр»
3. Настройте параметры: Опубликовано, Сортировка, Название, Описание, Иконка, Видят рейтинг
4. Сохраните изменения

## Шаги (автоматический рейтинг)

1. Скопируйте папку существующего рейтинга из `/bitrix/modules/ithive.gamification/classes/ratings`
2. Переименуйте папку (латиница, без спецсимволов)
3. Переименуйте PHP-файл внутри папки в соответствие с названием папки
4. Отредактируйте файл: измените название класса (строка 13) и фильтр логики
5. Создайте рейтинг в настройках модуля, указав папку с логикой
6. Включите опции: Акт. (активность), Автоначисл. (автоначисление)
7. Настройте: название, сортировку, коэффициент веса, описание, иконку, подразделения

## Ожидаемый результат

Новый рейтинг создан и отображается в публичной части портала. Автоматический рейтинг пересчитывается ежесуточно и ежемесячно начисляет баллы.

## Граничные случаи

- Создание автоматического рейтинга требует знания кода — рекомендуется делать с помощью специалистов
- Удалить можно только рейтинг, по которому ещё не было призеров
- Ручные рейтинги не участвуют в Общем рейтинге
- Иконка для автоматического рейтинга поддерживает PNG, JPEG и SVG
