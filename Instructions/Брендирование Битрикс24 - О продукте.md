# О продукте

**Модуль:** [[Module - Брендирование Битрикс24]]

## Описание

Модуль «Брендирование Битрикс24» (пакет `ithive.changeportaldefaultheme`) — это решение для стилизации корпоративного портала под фирменный стиль компании. Модуль позволяет изменить дизайн более 200 элементов Битрикс24 без навыков программирования и знания CSS. Все настройки выполняются в публичной части портала через понятный пользовательский интерфейс.

**Технические данные:**
- Совместим с PHP версии 8.1 и выше
- Совместим с СУБД: MySQL, PostgreSQL
- Поддерживает английский язык
- Поддерживает многодепартаментность
- Функционал доступен через современный десктопный браузер

## Ключевые функции

- **Настройка страницы авторизации** — установка фона, логотипа, цветов кнопок и шрифтов на странице входа в Битрикс24
- **Настройка виджетов** — кастомизация внешнего вида виджетов в Ленте (общая и индивидуальная настройка каждого виджета)
- **Настройка кнопок и ссылок** — изменение оформления всех кнопок и ссылок, включая цвета для разных состояний (наведение, клик)
- **Настройка шапки портала** — кастомизация верхней панели, левого навигационного меню и правого сайдбара
- **Загрузка макетов из Figma** — импорт готового дизайна из графического редактора Figma через специальный шаблон и плагин
- **Экспорт/импорт настроек** — создание резервных копий дизайна в виде ZIP-файла
- **CSS-стили** — возможность добавления произвольных CSS-стилей для опытных пользователей

## Предусловия

- Модуль должен быть установлен на корпоративный портал Битрикс24
- Доступ к модулю по умолчанию есть только у администраторов портала
- Портал должен использовать классическую тему (для Legacy) или светлую тему (для Зефир)

## Граничные случаи

- Модуль работает только с классическими темами портала (Legacy: «Битрикс 24», «Битрикс 24 с узором»; Зефир: светлые темы)
- Если пользователь установит другую тему, настройки брендирования не будут отображаться
- Исключение: настройки раздела «Авторизация» и загружаемые элементы (фавикон, логотип, наборы иконок) отображаются у всех пользователей независимо от темы
- Битрикс24 поддерживает только 4 шрифта: `family-system`, `system-mono`, `helvetica`, `open-sans`
- CSS-стили, добавленные в раздел «Дополнительно», не сбрасываются при сбросе настроек — их нужно удалять вручную
