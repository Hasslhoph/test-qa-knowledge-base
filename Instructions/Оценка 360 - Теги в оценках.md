# Оценка 360 - Теги в оценках

**Модуль:** [[Module - Оценка 360]]

## Описание

Теги позволяют объединять оценки в группы и отбирать нужные в списке. Аналогичный интерфейс тегов есть в «Задачах и Проектах» Битрикс24 (не связаны друг с другом).

## Ключевые функции

### Добавление тегов
Два места для добавления:
1. **На слайдере проверки данных** перед запуском — теги добавляются ко всем запускаемым оценкам
2. **В разделе «Оценки», колонка «Теги»** — к любой оценке из списка (нужно предварительно отобразить колонку)

Процесс: «Добавить» → ввести название → «Создать новый тег»

### Редактирование и удаление тегов
- «Добавить» → «Посмотреть теги» → слайдер со списком всех тегов
- Поиск по названию при большом количестве
- Переименование: бургер-меню (≡) → «Переименовать»
- Удаление: бургер-меню (≡) → «Удалить» (удаляется из всех оценок)
- Массовое удаление: выделить несколько → удалить

### Поиск по тегам
- Клик на нужный тег — показываются все оценки с этим тегом
- Рядом с тегом выводится имя автора
- Привилегированные пользователи могут управлять любыми тегами независимо от автора

## Предусловия

- Права администратора или пользователя с правами на запуск оценки

## Граничные случаи

- Теги Оценки 360 не пересекаются с тегами «Задач и Проектов»
- Слайдер проверки данных может быть отключён в настройках модуля
