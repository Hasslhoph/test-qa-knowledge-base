# Оценка 360 - Теги в оценках

**Модуль:** [[Module - Оценка 360]]

# Теги в оценках

Модуль позволяет добавлять теги к каждой запущенной оценке. С помощью тегов оценки можно объединять в группы и отбирать нужные в оценки списке.

Аналогичный интерфейс тегов есть в разделе Битрикс24 «Задачи и Проекты». Обратите внимание, то эти два интерфейса никак не связаны друг с другом, т.е. их теги не пересекаются.

![](_images-02/7.png)

Создавать и видеть теги могут только администраторы и пользователи с правами на запуск оценки.

## Добавление тегов

Добавлять теги к оценкам можно в двух местах:

1. **На слайдере проверки данных перед запуском оценки.** В этом случае теги будут добавлены ко всем запускаемым в данный момент оценкам. Обратите внимание, что слайдер для проверки данных может быть отключен в [настройках модуля](../06-administrirovanie-modulya/01-nastroyki-modulya-otsenka-360.md).

![](_images-02/image503.png)

2. **В разделе «Оценки», в колонке «Теги».** В данном случае добавить теги можно к любой оценке из списка.

![](_images-02/image267.png)

Для добавления тегов нужно предварительно отобразить колонку «Теги».

![](_images-02/image291.png)

В обоих случаях добавление тегов выполняется одинаково. Нажмите на кнопку «Добавить», чтобы открыть меню создания тегов.

![](_images-02/image381.png)

![](_images-02/image466.png)

В строку поиска введите название тега и ниже нажмите кнопку «Создать новый тег». Тег будет сразу же создан и прикреплен к выбранной оценке.

![](_images-02/image170.png)

Рядом с тегом всегда выводится имя его автора. Привилегированные пользователи могут прикреплять, откреплять, редактировать и удалять любые теги независимо от того, кто их автор.

![](_images-02/image277.png)

## Редактирование и удаление тегов

Нажмите на кнопку «Добавить», чтобы открыть меню создания тегов. Затем нажмите «Посмотреть теги».

![](_images-02/image149.png)

В результате откроется слайдер со списком всех созданных тегов. На слайдере можно увидеть количество оценок, к которым добавлен тот или иной тег.

![](_images-02/image319.png)

При большом количестве тегов можно воспользоваться поиском по названию.

![](_images-02/image183.png)

Для редактирования тега нажмите на бургер-меню (≡) слева от название и выберите «Переименовать».

![](_images-02/image431.png)

Измените название тега и нажмите «Enter». Тег будет изменен.

![](_images-02/image465.png)

![](_images-02/image197.png)

Для удаление тега выберите пункт «Удалить». Тег будет удален из всех оценок, куда он добавлен.

![](_images-02/image290.png)

При необходимости вы можете выбрать сразу несколько тегов из списка и удалить их через массовое действие.

![](_images-02/image28.png)

## Поиск по тегам

Когда теги указаны, становится возможным использовать их для отбора оценок в списке. Для этого нажмите на нужный тег — все оценки с этим тегом будут сразу же показаны в разделе.

![](_images-02/image406.png)
