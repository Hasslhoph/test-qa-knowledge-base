# Геймификация - Уровень сотрудника

**Модуль:** [[Module - Геймификация]]

## Описание

Баллы, заработанные в Геймификации, повышают уровень (ранг) сотрудника. Уровень зависит от суммы баллов за всё время. Понизить уровень невозможно.

## Ключевые функции

- Уровень отображается в профиле сотрудника и в виджете магазина бонусов
- В профиле виден: текущий уровень, сумма баллов за всё время, сумма до следующего уровня
- Настройка уровней: названия, количество баллов для достижения, иконки (PNG/JPEG/SVG)
- Готовые наборы иконок от Улей в Figma: Стандарт (2 набора × 7 иконок), Путь к величию (15), Гарри Поттер (15), Корпоративная эволюция (15), Корпоративная орнитология (15)

## Предусловия

- Права администратора (для настройки уровней)
- Доступ к административной части портала

## Шаги (настройка уровней)

1. Перейдите в административную часть → Настройки → Улей: Геймификация
2. Вкладка «Настройка уровней»
3. Измените названия уровней, количество баллов, иконки
4. Для добавления нового уровня — кнопка «Добавить параметр»
5. Сохраните изменения

## Шаги (выгрузка иконок из Figma)

1. Откройте [Figma-файл с иконками](https://www.figma.com/design/8AHCqud4rFWV4nW9oQqvIv/-S--%D0%93%D0%B5%D0%B9%D0%BC%D0%B8%D1%84%D0%B8%D0%BA%D0%B0%D1%86%D0%B8%D1%8F---%D0%98%D0%BA%D0%BE%D0%BD%D0%BA%D0%B8-%D0%B4%D0%BB%D1%8F-%D1%83%D1%80%D0%BE%D0%B2%D0%BD%D0%B5%D0%B9?node-id=1-1746)
2. Кликните на иконку (Shift+клик для нескольких)
3. В панели «Export» нажмите «+»
4. Выберите формат PNG или JPEG
5. Нажмите «Export»

## Ожидаемый результат

Уровни настроены, иконки загружены. Сотрудники видят свой уровень в профиле и магазине.

## Граничные случаи

- Уровень повышается при накоплении баллов, но понизить невозможно
- Для иконок уровней поддерживаются только PNG и JPEG (не SVG)
- Для просмотра Figma требуется регистрация в сервисе
- Количество баллов должно быть разным для каждого уровня
