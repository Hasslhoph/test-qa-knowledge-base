# HelpDesk - Настройка SLA для критических обращений

**Модуль:** [[Module - HelpDesk]]

## Описание

Настройка уровней поддержки SLA для критических обращений в модуле HelpDesk.

## Шаги

1. Перейти в Настройки модуля HelpDesk
2. Выбрать раздел SLA
3. Создать новый уровень SLA
4. Настроить время реакции
5. Настроить время решения
6. Назначить на типы обращений
7. Сохранить

## Ожидаемый результат

Для критических обращений установлены временные рамки SLA.
