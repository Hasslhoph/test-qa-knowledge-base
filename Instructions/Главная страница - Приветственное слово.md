# Приветственное слово

**Модуль:** [[Module - Главная страница]]

## Описание

Первый блок на странице «О компании» — формулирует миссию и задачи раздела, помогает сориентировать пользователей в содержимом страницы.

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- **Изображение** (240x360px) — фото или иллюстрация
- **Текст приветствия** — миссия и описание раздела «О компании»
- **Визуальный редактор Битрикс24** для форматирования текста
- **Переименование и изменение очередности** через [[Главная страница - Управление блоками]]

## Предусловия

- Права администратора или пользователя с правами на редактирование инфоблока
- Включённая административная панель (режим правки)

## Шаги

### Редактирование контента

1. В режиме правки наведите курсор на блок → «Изменить элемент»
2. Вкладка «Анонс»:
   - **Картинка для анонса** (240x360px)
   - **Описание для анонса** — текст приветствия (визуальный редактор)
3. Сохраните

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- Название блока можно поменять через управление блоками
- Очередность на странице и в меню навигации настраивается через управление блоками
