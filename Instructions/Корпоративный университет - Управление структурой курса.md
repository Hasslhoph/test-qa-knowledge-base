# Управление структурой курса

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Интерфейс для изменения расположения уроков, глав и тестов в курсе.

## Ключевые функции

### Права на управление

- Группа «HR-отдел»
- Администраторы
- Пользователи с правами на редактирование курса
- Группа «Автор курсов»

### Редактирование структуры

1. Курс → «Настройки» (⚙️) → «Управление структурой»
2. Перетаскивание элементов мышкой
3. Вкладывание элементов друг в друга

### Ограничения

- Раздел «Итоговые тесты» нельзя перемещать
- Тесты нельзя разместить на корневом уровне
- Тесты нельзя вложить друг в друга
- Вопросы к тестам перемещать нельзя (только на слайдере вопроса)

### Информация в шапке

- Общее количество глав, уроков, вопросов и тестов (включая деактивированные)
- Клик на количество вопросов/тестов → слайдер со списком

### Формат N/M

- N — количество вопросов/тестов, привязанных к данной главе
- M — общее количество включая вложенные уроки и подглавы

## Предусловия

- Права: HR-отдел, администраторы, авторы курсов
