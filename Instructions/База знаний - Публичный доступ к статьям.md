# База знаний - Публичный доступ к статьям

**Модуль:** [[Module - База знаний]]

## Описание

Создание публичных ссылок на статьи для внешних пользователей, не зарегистрированных на корпоративном портале. Ссылки позволяют открыть документ в браузере без авторизации.

## Ключевые функции

- Создание публичных ссылок на статьи
- Удаление публичных ссылок
- Запрет публичных ссылок на уровне статьи
- Глобальный запрет публичных ссылок на уровне модуля
- Список всех публичных статей с числовым индикатором
- Массовая деактивация/удаление публичных ссылок
- Иконка публичного доступа у статей

## Предусловия

- Права на редактирование статьи
- Функционал публичных ссылок не отключён в настройках модуля

## Шаги

### Создание публичной ссылки

1. Откройте статью
2. В правом верхнем углу нажмите «Поделиться» > «Ссылка на публичную страницу»
3. Ссылка скопируется в буфер обмена

### Удаление публичной ссылки

1. Откройте статью
2. В правом верхнем углу нажмите «Поделиться» > «Удалить публичную ссылку»
3. Ссылка станет недействительной

### Запрет публичной ссылки на уровне статьи

1. При создании/редактировании статьи поставьте галочку «Запретить создание публичной ссылки»
2. Сохраните статью

### Глобальный запрет публичных ссылок

1. Перейдите в административную часть: `адрес_портала/bitrix/admin`
2. «Настройки» > «Настройки продукта» > «Настройки модулей» > «Улей: База знаний»
3. Поставьте галочку «Отключить функционал публичных ссылок для статей»
4. Сохраните изменения

### Управление публичными ссылками

1. В настройках модуля найдите индикатор количества публичных статей
2. Кликните на индикатор для перехода к списку
3. Деактивируйте или удалите нужные ссылки

## Ожидаемый результат

Публичные ссылки созданы/удалены. Внешние пользователи могут/не могут просматривать статьи.

## Граничные случаи

- При глобальном запрете существующие публичные ссылки остаются активными (нужно отключить вручную)
- После запрета публичных ссылок поле «Запретить создание публичной ссылки» исчезает из слайдера статьи
- Статьи с публичным доступом помечены специальной иконкой
- При деактивации/удалении публичной ссылки сама статья не удаляется
