# Добавление модуля на второй сайт

**Модуль:** [[Module - Главная страница]]

## Описание

Инструкция по добавлению модуля «Главная страница» и всех его разделов на второй, третий и другие департаменты (сайты).

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- **Копирование файлов модуля** в папку второго сайта
- **Вывод пунктов меню** «Главная» и «О компании» на втором сайте
- **Создание папок** для хранения контента каждого департамента
- **Настройка инфоблоков** для каждого сайта
- **Разделение контента** между департаментами

## Предусловия

- Права администратора портала
- Созданные департаменты (сайты) в Битрикс24
- Настройки модуля: `/bitrix/admin/settings.php?lang=ru&mid=ithive.homepage`

## Шаги

### Добавление модуля на второй сайт

1. Перейдите в настройки модуля → вкладка «Источники данных»
2. Откройте вкладку второго сайта
3. Поставьте галочки:
   - Вывести в левом меню пункт «Главная»
   - Вывести в левом меню пункт «О компании»
4. Сохраните

### Создание папок для контента

1. Перейдите в нужный инфоблок (например, «Контент» > «Главная страница» > «Баннеры»)
2. Нажмите «Добавить раздел» → укажите название (например, «Баннеры деп.1»)
3. Перенесите существующие файлы в созданную папку
4. Создайте аналогичную папку для второго сайта
5. В настройках модуля → «Источники данных» → укажите папку для каждого сайта

### Инфоблоки для создания папок

- Баннеры
- Лидеры продаж
- Новости компании
- Новости СМИ
- Сервисы и заявки
- Социальные сервисы
- Ссылки
- Фотоальбом: «Контент» > «Фотогалерея» > «Фотогалерея пользователей»
- Видеоальбом: «Контент» > «Сервисы» > «Видео архив» (новые) или «Фотогалерея» > «Видео архив» (старые)

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- Если папка не указана, контент будет общим для всех сайтов
- Некоторые разделы могут быть общими (Сервисы, Фото/Видеоальбом, Ссылки) — оставьте поля пустыми
- После настройки папок Главная на втором сайте будет визуально пустой до добавления контента
