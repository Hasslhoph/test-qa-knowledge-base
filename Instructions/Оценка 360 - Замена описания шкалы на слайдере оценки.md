# Оценка 360 - Замена описания шкалы на слайдере оценки

**Модуль:** [[Module - Оценка 360]]

## Описание

Инструкция по замене стандартных фраз описания шкалы на слайдере оценки компетенций через создание собственного языкового файла.

## Ключевые функции

### Замена фраз через языковой файл
1. Создать папку `/bitrix/php_interface/user_lang/ru`
2. Создать файл `lang.php` с кодировкой UTF-8
3. Определить фразы через массив `$MESS`

### Стандартные фразы для замены
- `ASSESSMENT_RADAR_CHART_COLOR_LABELS_BAD` — низкая оценка
- `ASSESSMENT_RADAR_CHART_COLOR_LABELS_NORMAL` — средняя/допустимая оценка
- `ASSESSMENT_RADAR_CHART_COLOR_LABELS_GOOD` — высокая оценка

### Добавление своей фразы после описания шкалы

**Десктопная версия:**
```php
$MESS["..."]["INFO_BEFORE_COMPETENCE_ASSESSMENT"] = '
<div class="rating-heading assessment-heading">Заголовок</div>
<div class="rating-wrapper" style="grid-template-columns: none;">Текст описания</div>';
```

**Мобильное приложение:**
```php
$MESS["/bitrix/mobileapp/ithive.assessment360/extensions/assessment/my-task-detail/lang/ru/extension.php"]["INFO_BEFORE_COMPETENCE_ASSESSMENT"] = 'Текст описания';
```

### Редактирование фраз
- Папка `/bitrix/php_interface/user_lang/ru` → бургер-меню → «Редактировать как php»
- Изменить фразы → «Сохранить»

## Предусловия

- Доступ к административной части портала
- Текстовый редактор кода (например, Notepad++)

## Граничные случаи

- Название файла должно быть именно `lang.php`
- Для мобильного приложения нельзя использовать HTML-верстку (только текст)
- Файл должен быть в кодировке UTF-8
