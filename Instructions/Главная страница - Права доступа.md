# Права доступа

**Модуль:** [[Module - Главная страница]]

## Описание

Настройка прав доступа к редактированию инфоблоков Главной страницы и страницы «О компании».

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- **Администраторы портала:** полные права (добавление, редактирование, удаление, настройки компонентов, создание блоков)
- **Пользователи с правами на изменение:** только CRUD-операции через режим правки (без настроек компонентов и создания блоков)
- **2 уровня доступа к инфоблокам:** Изменение, Полный доступ
- **Настройка доступа** к разделам: Фотогалерея, Главная страница, О компании

## Предусловия

- Права администратора для настройки прав доступа
- Доступ к административной части (`/bitrix/admin/`)

## Шаги

### Настройка доступа к инфоблоку

1. `/bitrix/admin/` → «Контент» → нужный раздел (Фотогалерея / Главная страница / О компании)
2. Нажмите на нужный инфоблок
3. Внизу страницы: «Настройки информационного блока»
4. Вкладка «Доступ» → «Добавить» → выберите группу пользователей
5. Укажите уровень доступа:
   - **Изменение:** CRUD через режим правки + массовые действия в админке
   - **Полный доступ:** всё выше + редактирование инфоблока и настройка прав
6. Сохраните

### Доступ к разделу «Главная страница»

Инфоблоки: Баннеры, Новости компании, Новости СМИ, Сервисы и заявки, Лидеры продаж, Ссылки, Социальные сервисы

> Для Баннеров обязательно укажите «Полный доступ» — только так пользователи смогут управлять баннерами через публичную часть

### Доступ к странице управления блоками

1. «О компании» > «Управление блоками» → «Настройки информационного блока»
2. Вкладка «Доступ» → добавьте группу с нужным уровнем
3. Сохраните

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- Права на создание новых блоков для «О компании» есть только у администраторов
- Администраторы портала не обладают правами модераторов (кроме скрытия/удаления комментариев при включённой постмодерации)
- Модераторы не получают уведомления о собственных комментариях
