# Карточка компетенции

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Детальная карточка (профиль) компетенции с информацией о позиции, развивающих действиях и сотрудниках.

## Ключевые функции

### Структура карточки

1. **Информация о компетенции**
   - Описание
   - Индикатор уровня владения (серая галочка, синяя галочка, академическая шапочка)
   - Кто обладает данной компетенцией
   - Кто выступает экспертом
   - Кто развивает компетенцию

2. **Позиции**
   - Позиции, для достижения которых нужно освоить компетенцию
   - Клик на стрелочку → профиль позиции

3. **Развивающие действия**
   - Фильтры по обязательности и типам
   - Все шесть типов развивающих действий

### Интеграция с Оценкой 360

- Рядом с иконкой уровня владения — оценка по компетенции
- Блок «Индикаторы компетенции» с весами и оценками
- Серая точка / синяя галочка в зависимости от оценки

## Предусловия

- Доступно всем пользователям
