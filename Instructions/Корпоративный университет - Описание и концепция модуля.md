# Описание и концепция модуля

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Корпоративный университет (КУ) — это инструмент для обучения сотрудников компании. С помощью модуля сотрудники могут осваивать и развивать профессиональные компетенции без отрыва от производственной деятельности.

## Ключевые функции

### Для чего нужен Корпоративный университет

- Освоение и развитие профессиональных компетенций (hard и soft skills)
- Программы развития — план развивающих действий для «прокачки» компетенций
- Карьерный рост (горизонтальный и вертикальный) на основе освоенных компетенций
- Адаптация новых сотрудников
- Повышение квалификации кадров

### Концепция модуля

- **Адаптация** — эффективное прохождение испытательного срока через программы адаптации
- **Развитие** — повышение квалификации через программы развития
- **Адаптация + Развитие** — одновременная адаптация и оттачивание знаний

### Шесть типов развивающих действий

1. **Курсы и тесты** — собственные обучающие курсы с тестами для проверки знаний
2. **Книги** — электронный каталог книг через модуль «Библиотека» (интеграция с МИФ, Альпина)
3. **Мероприятия** — тренинги, лекции, воркшопы
4. **Наставник** — эксперты помогают коллегам в освоении компетенций
5. **Задачи и проекты** — применение знаний на практике
6. **Развитие на рабочем месте** — любые развивающие действия (внешние сайты, онлайн-лекции и т.п.)

### Уровни владения компетенциями

- **Серая галочка** — сотрудник не владеет компетенцией
- **Синяя галочка** — сотрудник владеет компетенцией на достаточном уровне
- **Академическая шапочка** — сотрудник владеет компетенцией на экспертном уровне (может быть наставником)

### Варианты запуска развития

1. **Запуск по карьерной траектории** — развитие в позицию, автоматическое определение компетенций
2. **Запуск по шаблону** — заранее подготовленный набор компетенций (например, «Адаптация нового сотрудника»)
3. **Свободный выбор компетенций** — ручной выбор компетенций для развития

### Интеграция с Оценкой 360

При установленном модуле «Оценка 360» уровень владения компетенцией определяется в числовом формате (по шкале), что позволяет глубже анализировать пробелы в навыках.

## Предусловия

- Установленный модуль «Корпоративный университет»
- Настроенные карьерные траектории (опционально)
- Заполненная система компетенций
