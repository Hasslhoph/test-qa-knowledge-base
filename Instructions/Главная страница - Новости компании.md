# Новости компании

**Модуль:** [[Module - Главная страница]]

## Описание

Основной новостной раздел Главной страницы для публикации корпоративных статей с возможностью таргетирования, лайков, комментариев и дублирования в ленту как важных сообщений.

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- **4 шаблона отображения:** news_sparkle, news_card (по умолчанию), news_content, news_smi
- **Создание новостей** с вкладками: Параметры, Анонс, Подробно, Доступ
- **Важные новости** — выделение на фоне других с красной плашкой
- **Дублирование в ленту** как важное сообщение (только для Новостей компании)
- **Таргетирование** — доступ к новостям для конкретных отделов/групп
- **Рубрики** — категоризация новостей
- **Избранное** — добавление новостей в избранное
- **Время на просмотр/чтение** — отображается при наведении на обложку
- **5 шаблонов детальной страницы:** Картинка сверху, Картинка слева, Картинка сверху + 3 изображения, Картинка сверху + слайдер, Картинка слева + слайдер, Файл
- **Постмодерация комментариев** — модераторы могут скрывать/удалять комментарии
- **Статистика:** просмотры, лайки, комментарии

## Предусловия

- Права привилегированного пользователя с доступом к инфоблоку «Новости компании»
- Включённая административная панель (режим правки)

## Шаги

### Создание новости

1. На Главной в режиме правки: наведите на блок новостей → «Добавить новость»
   Или на странице «Новости компании»: нажмите «Добавить новость»
2. Вкладка «Параметры»:
   - Активность, Начало/Окончание активности
   - Название (обязательное), Сортировка
   - Важная новость (да/нет)
   - Видимость комментариев/лайков
   - Время на просмотр/чтение (мин.)
   - Дополнительные изображения
   - Дублировать как важное сообщение в ленте
   - Избранная новость, Отображать на главной
   - Показывать кол-во просмотров
   - Рубрики, Сообщение будет важным (срок)
   - Файлы, Шаблон (5 вариантов)
3. Вкладка «Анонс»: Картинка для анонса, Описание для анонса
4. Вкладка «Подробно»: Детальная картинка, Детальное описание
5. Вкладка «Доступ»: настройка доступа для групп пользователей
6. Сохраните

### Настройка блока новостей

1. В режиме правки наведите на блок новостей → шестерёнка (⚙️)
2. Выберите шаблон компонента, количество новостей, сортировку, ограничения символов

### Постмодерация комментариев

1. В настройках модуля вкладка «Новости» → раздел «Модерация комментариев»
2. Режим модерации: «Постмодерация»
3. Выберите модераторов
4. Модераторы видят счётчики непросмотренных комментариев, получают уведомления, могут скрывать/удалять комментарии

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- Опция «Дублировать как важное сообщение» доступна только для Новостей компании (не для Новостей СМИ)
- Публикация в ленте появляется только при создании новой статьи (не при редактировании существующей)
- При удалении статьи публикация в ленте не исчезает
- Модераторы не получают уведомления о собственных комментариях
- При установке модуля «Геймификация» сотрудники получают очки рейтинга за просмотры, лайки и комментарии
