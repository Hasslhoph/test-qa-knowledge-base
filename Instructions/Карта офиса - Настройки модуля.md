# Настройки модуля

**Модуль:** [[Module - Карта офиса + Бронирование]]

## Описание

Инструкция по основным настройкам модуля «Карта офиса» в административной части портала. Настройки включают четыре вкладки: Общие настройки, Настройки подтверждений, По типам рабочих мест, Настройки стилей.

## Ключевые функции

- Управление доступом к публичной части модуля
- Настройка названий разделов (город/офис/кабинет)
- Настройка уведомлений
- Лимиты бронирования
- Настройки подтверждений
- Настройка стилей (цвета статусов, закреплённых мест)

## Предусловия

- Доступ к настройкам есть только у администраторов портала
- Перейти: `адрес_портала/bitrix/admin/settings.php?lang=ru&mid=ithive.workplaces`

## Вкладка «Общие настройки»

1. **Доступ к публичной части** — выбор отделов/сотрудников, которые видят модуль
2. **Настройки инфоблоков** — не рекомендуется менять
3. **Названия разделов 1-3 уровней** — кастомные названия для городов/офисов/кабинетов
4. **Уведомления** — текст стандартных уведомлений (бронирование за другого, изменение, отмена, лимит, приближение начала, окончание)
5. **Минимальное/максимальное время бронирования** — размер шкалы времени (по умолчанию 8:00–19:00)
6. **Лимит прошедших бронирований** — количество в «Мои бронирования» (по умолчанию 50, мин. 50, макс. 200)
7. **Лимит бронирований в неделю (предупреждение)** — информационное предупреждение (макс. 7, 0 = отключено)
8. **Лимит месяцев для бронирования вперёд** — от 1 до 12 (по умолчанию 3)
9. **Часы для поиска свободного места** — предупреждение о нежелательности частичного бронирования (1–13 часов, по умолчанию 3)
10. **Время уведомления о приближении начала** — от 0 до 24 часов (по умолчанию 24, 0 = отключено)
11. **Время уведомления об окончании** — от 0 до 24 минут (по умолчанию 15, 0 = отключено)
12. **Деактивировать бронирования увольняющихся** — автоматическая отмена
13. **Аватарки сотрудников на карте** — отображение при масштабе 1X, 2X, 3X
14. **При отсутствии нужного дня при повторении** — пропускать/двигать назад/двигать вперёд
15. **Блок «Где я сижу»** — показывать/скрыть в профиле сотрудника
16. **Закреплённые места в «Кто в офисе»** — сверху или снизу
17. **Масштаб карты по умолчанию** — Вписать/1X/2X/3X

## Вкладка «Настройки подтверждений»

1. **Время ДО начала для подтверждения** — от 10 до 60 минут (по умолчанию 15)
2. **Время ПОСЛЕ начала для подтверждения** — от 10 до 60 минут (по умолчанию 15)
3. **Текст уведомлений** — текст о подтверждении/отсутствии подтверждения

## Вкладка «По типам рабочих мест»

Настройка типов рабочих мест (грация, лимиты, подтверждение, повторения).

## Вкладка «Настройки стилей»

### Цвета статусов
- Свободное место — по умолчанию зелёный `#A4D124`
- Место занятое на несколько часов — по умолчанию жёлтый `#FFB23E`
- Место занятое на весь день — по умолчанию красный `#FF4343`

### Цвета закреплённого места
- Фон — по умолчанию серый `#F3F3F3`
- Бордюр — по умолчанию серый `#D7D7D7`
- Номер — по умолчанию серый `#A5ADB3`

### Прочие цвета
- Выбранный блок — по умолчанию тёмно-серый `#535C69`

## Граничные случаи

- Настройки инфоблоков не рекомендуется менять для корректной работы модуля
- Размер шкалы времени регулирует только визуальный размер, не влияет на доступные часы бронирования
- Суммарное время подтверждения = время ДО начала + время ПОСЛЕ начала
- Цвета указываются в формате HEX-кода
