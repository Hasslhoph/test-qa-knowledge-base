# Описание и возможности модуля

**Модуль:** [[Module - Опросы]]

## Описание

Модуль «Опросы» — инструмент для HR-специалистов и руководителей, автоматизирующий сбор обратной связи от сотрудников. Результаты предоставляются в виде графиков и диаграмм. Модуль совместим с PHP 8.1+, СУБД MySQL/PostgreSQL, поддерживает английский язык и многодепартаментность.

## Ключевые функции

- Создание и редактирование опросов с любыми типами и количеством вопросов
- Восемь типов вопросов: одиночный выбор, множественный выбор, один из списка, несколько из списка, целое число, дробное число, шкала, ранжирование
- Группировка опросов по тематикам (опросы дня, для HR-отдела, для руководителей и т.д.)
- Управление доступом к каждому опросу отдельно (группы пользователей, отдельные сотрудники, отделы, группы соцсети)
- Пульс-опросы — всплывающие окна на каждой странице портала + виджет на Главной странице (требуется модуль [[Module - Главная страница]])
- Управление сроками опросов (дата начала/окончания)
- Графики и диаграммы для анализа результатов (круговые диаграммы, гистограммы, тепловая диаграмма для ранжирования)
- Выгрузка результатов в Excel

## Предусловия

- Модуль установлен и активирован
- Пользователь имеет права редактора опросов или администратора для создания/редактирования
