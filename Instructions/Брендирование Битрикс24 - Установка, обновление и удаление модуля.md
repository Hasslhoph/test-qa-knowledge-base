# Установка, обновление и удаление модуля

**Модуль:** [[Module - Брендирование Битрикс24]]

## Описание

Инструкция по установке, обновлению, активации купона и удалению модуля «Брендирование Битрикс24» (пакет `ithive.changeportaldefaultheme`). Все операции выполняются из административной части портала.

## Ключевые функции

### Установка модуля

1. Перейти: `адрес_портала/bitrix/admin/update_system_partner.php?addmodule=ithive.changeportaldefaultheme`
2. Модуль автоматически отмечен галочкой → нажать «Загрузить»
3. Подтвердить Лицензионное соглашение → «Применить»
4. Дождаться завершения установки
5. Убедиться в уведомлении об успешной загрузке

### Обновление модуля

1. Административная часть → Marketplace → Обновление решений → Список обновлений
2. Выбрать «Улей: Брендирование Битрикс24» → «Загрузить»
3. Или сразу: `адрес_портала/bitrix/admin/update_system_partner.php`

### Активация купона

1. Marketplace → Обновление решений → Активация купона
2. Ввести номер купона → «Активировать купон»
3. Или: `адрес_портала/bitrix/admin/update_system.php?lang=ru`

**Для неустановленного модуля:**
- Marketplace → Установленные решения → Купленные решения → бургер-меню → «Загрузить»

**Для модуля с истёкшей демоверсией:**
- Обновление решений → Установка обновлений → «Зарегистрировать решение»

### Удаление модуля

1. Marketplace → Установленные решения → Доступные решения
2. Бургер-меню рядом с решением → «Удалить»
3. Подтвердить удаление после обновления страницы

## Предусловия

- Права группы «Администраторы» для всех операций
- Доступ к административной части портала

## Граничные случаи

- Все операции (установка, обновление, активация купона, удаление) доступны **только** администраторам
- При удалении модуля требуется двойное подтверждение (страница обновляется дважды)
- Купон можно активировать как для установленного, так и для неустановленного модуля
