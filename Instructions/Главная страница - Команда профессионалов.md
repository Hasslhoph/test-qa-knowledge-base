# Команда профессионалов

**Модуль:** [[Module - Главная страница]]

## Описание

Блок на странице «О компании» для представления сотрудников компании, выделения ведущих специалистов и публикации контактов.

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- **Карточки сотрудников** в виде слайдера с кнопками перелистывания
- **Привязка к профилю** Битрикс24 (ФИО, фото, должность, контакты подтягиваются автоматически)
- **Ссылка на профиль** сотрудника по клику на имя
- **Ссылка «Оргструктура»** на страницу со структурой компании
- **Создание новых карточек** сотрудников
- **Переименование и изменение очередности** через [[Главная страница - Управление блоками]]

## Предусловия

- Права администратора или пользователя с правами на редактирование инфоблока
- Включённая административная панель (режим правки)

## Шаги

### Создание карточки сотрудника

1. `/bitrix/admin/` → «Контент» > «О компании» > «Команда профессионалов» > «Моя компания»
2. Нажмите «Добавить элемент»
3. Вкладка «Параметры»:
   - Название (для административной части, рекомендуется ФИО)
   - Сортировка
   - Сотрудник (привязка к профилю Битрикс24)
4. Вкладка «Анонс»:
   - Картинка для анонса (280px высота, если не загружена — фото из профиля)
   - Описание для анонса (детальная информация)
5. Сохраните

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- Если фото не загружено, отображается фото из профиля сотрудника в Битрикс24
- Ширина картинки меняется пропорционально высоте (280px)
