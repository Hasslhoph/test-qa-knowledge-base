# Деактивация рудиментных полей в карточке компетенции

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Скрытие неактуальных полей из слайдера редактирования компетенции, оставшихся от ранних версий модуля.

## Ключевые функции

### Рудиментные поля

1. Уровень
2. Компетенции предыдущего уровня
3. Компетенции следующего уровня

### Деактивация полей

1. Административная часть → адрес_портала`/bitrix/admin/`
2. «Контент» → «Корпоративный университет» → «Компетенции»
3. Настройки информационного блока (под списком компетенций)
4. Вкладка «Свойства»
5. Убрать галочки активности напротив:
   - «Уровень»
   - «Компетенции предыдущего уровня»
   - «Компетенции следующего уровня»
6. Сохранить изменения

### Результат

- Лишние свойства больше не отображаются на слайдере редактирования компетенции

## Предусловия

- Права: администраторы, группа HR-отдела
