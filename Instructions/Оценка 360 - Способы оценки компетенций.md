# Оценка 360 - Способы оценки компетенций

**Модуль:** [[Module - Оценка 360]]

## Описание

Три способа оценки компетенций: по шкале, по баллам (одиночный выбор), по баллам (множественный выбор). Менять способы оценки, указывать баллы и вес индикаторов могут только администраторы и группа HR-отдела.

## Ключевые функции

### 1. Оценка по шкале
- Компетенцию или каждый индикатор оценивают по шкале (по умолчанию 1-10)
- Два элемента: Шкала и Вес индикатора
- Вес — условная величина значимости индикатора (положительное или отрицательное число)
- Сумма отрицательных весов не может быть больше суммы положительных
- Итоговая оценка = средневзвешенное значение от всех респондентов

**Расчёт:**
- Шаг 1: Респонденты оценивают индикаторы по шкале
- Шаг 2: Оценка от каждого респондента = (балл × вес) sum / (вес sum)
- Шаг 3: Итоговая оценка = среднее арифметическое оценок всех респондентов

### 2. Оценка по баллам (одиночный выбор)
- Каждому индикатору заранее присваивается балл (целое число от 1 до макс. шкалы)
- Индикаторы становятся вариантами ответа — респондент выбирает только один
- Оценивающие не видят баллы (скрытая балльная оценка)
- Для этого способа компетенции формулируются как вопросы, индикаторы — как взаимоисключающие варианты
- Итоговая оценка = среднее арифметическое баллов выбранных индикаторов

### 3. Оценка по баллам (множественный выбор)
- Индикаторы — варианты ответа, респондент может отметить несколько
- Сумма баллов индикаторов должна быть > 0 и ≤ макс. значения шкалы
- Можно указывать дробные и отрицательные числа (0.25, -3)
- Итоговая оценка = сумма баллов индикаторов, отмеченных > 50% респондентов

## Предусловия

- Права администратора или группы HR-отдела
- Для балльной оценки (способы 2 и 3) обязательно наличие индикаторов

## Граничные случаи

- Балльная оценка возможна только с индикаторами
- Числа для баллов зависят от настроек шкалы (10-балльная = 1-10, 5-балльная = 1-5)
- Отрицательные баллы задавать нельзя (для одиночного выбора)
- Респондент может выбрать «Нет оценки/Не могу оценить» = 0 баллов
- Можно запретить воздерживаться от оценки в настройках модуля
