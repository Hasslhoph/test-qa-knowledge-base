# Установка, обновление и удаление модуля

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Процедуры установки, обновления, активации купона и удаления модуля «Корпоративный университет».

## Ключевые функции

### Установка модуля

1. Перейти по ссылке: адрес_портала`/bitrix/admin/update_system_partner.php?addmodule=ithive.ipr`
2. Модуль автоматически отмечен → «Загрузить»
3. Подтвердить согласие с Лицензионным соглашением → «Применить»
4. Дождаться завершения установки
5. Уведомление об успешной загрузке

### Обновление модуля

1. Marketplace → «Обновление решений» → «Список обновлений»
2. Выбрать «Улей: Корпоративный университет» → «Загрузить»
3. Или сразу: адрес_портала`/bitrix/admin/update_system_partner.php`

### Активация купона

1. Marketplace → «Обновление решений» → «Активация купона»
2. Ввести номер купона → «Активировать купон»
3. Или: адрес_портала`/bitrix/admin/update_system.php?lang=ru`

**Для неустановленного модуля:**
- Marketplace → «Установленные решения» → «Купленные решения» → «Загрузить»

**Для модуля с истёкшей демоверсией:**
- «Обновление решений» → «Установка обновлений» → «Зарегистрировать решение»

### Удаление модуля

1. Marketplace → «Установленные решения» → «Доступные решения»
2. Бургер-меню → «Удалить»
3. Подтвердить удаление

## Предусловия

- Права на все операции: администраторы
