# HelpDesk - Аналитика по обращениям

**Модуль:** [[Module - HelpDesk]]

## Описание

Аналитика по обращениям доступна администраторам ТП и администраторам модуля. Два раздела: «Рабочий стол техподдержки» и «Графики».

## Ключевые функции

### Рабочий стол техподдержки
- Путь: Сервисы → Техподдержка → Рабочий стол ТП
- Сводные отчёты:
  - Ответственный
  - Критичности
  - Статус
  - Уровень сложности
  - Категория
  - Источник
  - Оценка ответов
  - Уровни поддержки
- Фильтры: промежуток времени, сайт/ответственный

### Графики
- Путь: Сервисы → Техподдержка → Графики
- **Нагрузка на техподдержку** — график количества обращений по дням
- **Длительность решения проблем** — круговая диаграмма времени закрытия (в днях)
- **Количество сообщений для решения** — процентное соотношение комментариев в закрытых обращениях
- Фильтры: множество критериев

## Предусловия

- Права администратора ТП или администратора портала
