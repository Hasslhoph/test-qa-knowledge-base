# Настройки модуля

**Модуль:** [[Module - Опросы]]

## Описание

Основные настройки модуля «Опросы» через административную часть портала. Доступ только у администраторов.

## Ключевые функции

### Расположение настроек

`адрес_портала/bitrix/admin/settings.php?lang=ru&mid=ithive.polls`

### Параметры настройки

| № | Параметр | Описание |
|---|---|---|
| 1 | Вкладки сайтов | Настройка индивидуально для каждого сайта/департамента |
| 2 | Публичная часть модуля доступна | Доступ к публичной части модуля (если не заполнено — сотрудники не видят модуль) |
| 3 | Исключить типы вопросов | Убрать любые типы вопросов из меню добавления (не удаляет из существующих опросов) |
| 4 | Отображаемые группы опросов | Группы, отображаемые на главной странице модуля (скрытые группы доступны только администраторам и редакторам) |
| 5 | Группа Пульс-опросов | Выбор групп для вывода как пульс-опросов (множественный выбор через Ctrl/⌘) |
| 6 | Не показывать всплывающие пульс-опросы на страницах | URL-адреса страниц, где окно пульс-опросов не должно выводиться |
| 7 | Ограничение символов для многострочного ответа | Макс. кол-во символов в поле «Свой вариант» (многострочное). 0 = без ограничений |
| 8 | Ограничение символов для однострочного ответа | Макс. кол-во символов в поле «Свой вариант» (однострочное). 0 = без ограничений |
| 9 | Ограничение символов для комментария | Макс. кол-во символов в комментарии к ответу. 0 = без ограничений |

## Предусловия

- Пользователь — администратор портала

## Шаги

1. Перейти по адресу: `адрес_портала/bitrix/admin/settings.php?lang=ru&mid=ithive.polls`
2. При наличии нескольких сайтов — выбрать нужную вкладку
3. Настроить нужные параметры
4. Сохранить изменения

## Ожидаемый результат

Настройки модуля применены для выбранного сайта/департамента.

## Граничные случаи

- Исключённые типы вопросов не пропадают из уже созданных опросов
- Скрытые группы опросов и их опросы остаются доступны через административную часть портала
