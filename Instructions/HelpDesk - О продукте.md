# HelpDesk - О продукте

**Модуль:** [[Module - HelpDesk]]

## Описание

Модуль «Улей: Helpdesk» (ithive.helpdesk) — модуль для автоматизации работы любых внутренних сервисных служб корпоративного портала Битрикс24. Предоставляет сотрудникам сервис в концепции единого окна: централизованный сбор обращений, маршрутизация по веткам, назначение ответственных, контроль SLA и аналитика. Версия документации: 6.0.0 (Апрель 2026).

**Технические данные:**
- Совместим с PHP 8.1+
- СУБД: MySQL, PostgreSQL
- Поддерживает английский язык
- Поддерживает многодепартаментность
- Доступен через десктопный браузер и мобильное приложение Б24

## Ключевые функции

- **Универсальный мастер создания обращений** — многошаговые формы с различными типами полей (текст, флажок, выпадающий список, переключатель, дата/время, число)
- **Гибкая настройка ответственных** — автоматическое назначение по категории обращения и/или подразделению автора
- **Уровни поддержки (SLA)** — приоритизация групп пользователей, расписание работы, время реакции
- **Анонимность обратной связи** — опция анонимной отправки через спецпользователя helpdesk.anonymous
- **Скрытые комментарии** — общение сотрудников ТП без видимости для автора обращения
- **Ссылка на создание обращения** — прямая ссылка на конкретный шаг мастера для размещения в любом разделе портала
- **Оценка удовлетворённости** — система оценок работы ТП при закрытии обращения
- **Аналитика по обращениям** — графики, сводные таблицы, отчёты по нагрузке и длительности
- **Привязка задач к обращениям** — создание и привязка задач Битрикс24 к тикетам (с v6.0.0)
- **Управление доступом через настройки** — настройка доступов к модулю через административную часть (с v6.0.0)

## Предусловия

- Модуль ithive.helpdesk установлен и активирован
- Пользователи имеют соответствующие роли (клиент ТП, сотрудник ТП, администратор ТП, администратор портала)
- Настроены ветки обращений и справочники
