# Настройка типов рабочих мест

**Модуль:** [[Module - Карта офиса + Бронирование]]

## Описание

Инструкция по созданию и настройке типов рабочих мест. Каждый тип имеет свои условия бронирования: длительность, лимиты, подтверждение, повторения.

## Ключевые функции

- Настройка параметров каждого типа рабочего места
- Градация фиксации времени (интервалы бронирования)
- Лимит длительности бронирования (часы)
- Лимит количества бронирований одним сотрудником в неделю
- Подтверждение бронирований
- Повторения бронирований и лимит повторений
- Создание новых типов рабочих мест

## Предусловия

- Доступ к настройкам есть только у администраторов портала
- Модуль установлен и активирован

## Настройка типов рабочих мест

Перейти: `адрес_портала/bitrix/admin/settings.php?lang=ru&mid=ithive.workplaces` → вкладка «По типам рабочих мест»

### Параметры
1. **Тип места** — название типа. Строка «По умолчанию» — общие настройки для типов с отключённой активностью
2. **Иконка** — загрузка иконки (JPG, PNG, SVG)
3. **Активность** — индивидуальные или общие настройки
4. **Градация фиксации времени, мин** — минимальное время и интервалы бронирования (5–90 мин)
5. **Лимит длительности, часы** — максимальное количество часов (0–7, где 0 = без лимита)
6. **Кол-во в неделю одним сотрудником** — лимит бронирований одного места (0–4, где 0 = без лимита)
7. **Подтверждение** — обязательность подтверждения брони за 15 минут до начала
8. **Повторения** — включение/отключение опции повторения бронирования
9. **Лимит кол-ва повторений** — максимальное количество повторяющихся бронирований

## Создание новых типов

1. Перейти: `адрес_портала/bitrix/admin/index.php?lang=ru`
2. Контент → Карта офиса → Карта офиса
3. Нажать «Настройки информационного блока»
4. Вкладка «Свойства» → три точки в поле «Тип»
5. В таблице «Значения списка» ввести название и символьный код
6. Сохранить изменения

## Граничные случаи

- Лимиты не действуют на администраторов, офис-менеджеров и модераторов
- Лимиты не действуют на сотрудников из группы исключений (настраивается на уровне города/офиса/кабинета)
- При превышении лимита кнопка «Забронировать» блокируется с уведомлением
- При отключении повторений уже созданные повторяющиеся бронирования не отменяются
- Тип «По умолчанию» нельзя деактивировать
