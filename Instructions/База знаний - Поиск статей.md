# База знаний - Поиск статей

**Модуль:** [[Module - База знаний]]

## Описание

Поиск и фильтрация статей в Базе знаний. Поиск работает с привязкой к текущему разделу. Для поиска по всему модулю нужно находиться на главной странице.

## Ключевые функции

- Строка поиска (по названию и содержимому)
- Фильтры: Название, Описание статьи, Дата создания, Дата изменения, Кто изменил, Кто создал, Теги
- Сохранение комбинаций фильтров
- Поиск по всему модулю с главной страницы

## Предусловия

- Права на чтение статей

## Шаги

### Поиск через строку поиска

1. Находясь в нужном разделе (или на главной странице для поиска по всему модулю)
2. Введите запрос в строку «Фильтр + поиск»
3. Система ищет совпадения в названии и содержании статей

### Поиск через фильтры

1. Откройте панель фильтров
2. Заполните нужные поля:
   - **Название**
   - **Описание статьи**
   - **Дата создания**
   - **Дата изменения**
   - **Кто изменил**
   - **Кто создал**
   - **Теги**
3. Примените фильтры

### Сохранение фильтров

1. Настройте нужные фильтры
2. Сохраните комбинацию фильтров
3. Используйте сохранённый фильтр для быстрого поиска

## Ожидаемый результат

Найдены статьи, соответствующие запросу и/или фильтрам.

## Граничные случаи

- Поиск всегда привязан к текущему разделу
- Для поиска по всему модулю нужно находиться на главной странице Базы знаний
- Любые комбинации фильтров можно сохранять
