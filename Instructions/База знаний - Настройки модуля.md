# База знаний - Настройки модуля

**Модуль:** [[Module - База знаний]]

## Описание

Административные настройки модуля «База знаний». Доступны только администраторам портала. Включают настройку для каждого сайта (департамента), визуального редактора, версионности, публичных ссылок, комментариев и других параметров.

## Ключевые функции

- Настройка отдельно для каждого сайта (департамента)
- Управление видимостью модуля для групп пользователей
- Выбор режима визуального редактора (3 режима)
- Настройка версионности (4 режима + лимит версий)
- Размер превью изображений
- Группа пользователей с правами на HTML-редактор
- Отключение публичных ссылок
- Отключение комментариев
- Вывод статей из подразделов
- Иконки для разделов (включение + уровень вложенности)
- Время запуска агента для пересчёта просмотров

## Предусловия

- Права администратора портала

## Шаги

1. Перейдите: `адрес_портала/bitrix/admin/settings.php?lang=ru&mid=ithive.knowledgebase`
2. Настройте параметры:

### Параметры

1. **Вкладки для переключения между сайтами** — настройка для каждого департамента отдельно
2. **Публичная часть модуля доступна** — какие группы видят модуль в публичной части
3. **Режим использования расширенного визуального редактора**:
   - Для каждой новой статьи включать расширенный редактор
   - Включать упрощённый, но можно конвертировать (по умолчанию)
   - Не использовать расширенный визуальный редактор
4. **Функционал версионности статей** — 4 режима (см. [[База знаний - Версионность статей]])
5. **Количество хранимых версий на статью** — от 1 до 50
6. **Максимальный размер превью картинки** — по умолчанию 300x300 px (только упрощённый редактор)
7. **Группа пользователей с правами на html редактор** — поиск файлов в структуре сайта
8. **Отключить функционал публичных ссылок для статей**
9. **Отключить функционал комментирования статей**
10. **Отключить вывод в разделах статей вложенных подразделов**
11. **Включить иконки для разделов**
12. **Иконки в дереве разделов выводятся до уровня вложенности**
13. **Время запуска агента для пересчета просмотров** — по умолчанию 00:00

## Ожидаемый результат

Настройки модуля применены. Изменения влияют на все статьи и разделы (если не указано иное).

## Граничные случаи

- Настройки доступны только администраторам
- При отключении публичных ссылок существующие ссылки остаются активными
- При отключении комментариев существующие комментарии скрываются и появляются снова при включении
- Настройка превью изображений действует только для статей в упрощённом редакторе
