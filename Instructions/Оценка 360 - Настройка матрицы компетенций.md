# Оценка 360 - Настройка матрицы компетенций

**Модуль:** [[Module - Оценка 360]]

## Описание

Настройка матрицы компетенций для оценки 360. Матрица компетенций позволяет структурировать набор компетенций, индикаторов и ролей для проведения оценки сотрудников.

## Ключевые функции

- Создание матрицы компетенций
- Добавление компетенций в матрицу
- Назначение ролей для оценки
- Активация матрицы

## Предусловия

- Установлен и активирован модуль «Оценка 360»
- Наличие настроенных компетенций и категорий

## Шаги

1. Создать матрицу
2. Добавить компетенции
3. Назначить роли
4. Активировать

## Ожидаемый результат

Матрица компетенций настроена и готова к использованию в оценке 360.

## Граничные случаи

- Матрица без компетенций не может быть активирована
- Роли должны быть назначены до активации
