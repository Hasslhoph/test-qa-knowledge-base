# Назначение курсов / тестов

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Функция назначения курсов и тестов сотрудникам для включения в карточку развития.

## Ключевые функции

### Как назначить курс/тест

1. Открыть курс → кнопка «Назначить»
2. Блок «Что назначить»:
   - **Дата начала тестов**
   - **Шаг** — промежуток между тестами (в днях)
   - **Сроки прохождения тестов**
3. Блок «Кому назначить» — выбор сотрудника/сотрудников
4. «Назначить» → подтверждение

### Повторное назначение

- Если курс/тест уже в активных карточках — повторное назначение невозможно

### Назначение курса самому себе

- Раздел «Онлайн-курсы» → кнопка «Пройду»
- Указать дату планируемого прохождения
- Если нет активных карточек — создаётся новая «Свободный выбор компетенций»
- Если есть карточки — добавляется в самую позднюю

## Предусловия

- Права на назначение: администраторы, группа «Назначить курс»
- Каждый сотрудник может назначить курс себе самостоятельно

## Граничные случаи

- При нескольких активных карточках действие добавляется в последнюю созданную
- Курс добавляется в компетенцию «Саморазвитие»
