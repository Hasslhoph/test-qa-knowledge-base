# Видео о компании

**Модуль:** [[Module - Главная страница]]

## Описание

Блок на странице «О компании» для загрузки видеоролика о корпоративной жизни, продуктах компании, интервью с руководством.

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- **Загрузка видео** в формате MP4
- **Одновременно отображается только одно видео** (самое позднее по ID)
- **Переименование и изменение очередности** через [[Главная страница - Управление блоками]]

## Предусловия

- Права администратора или пользователя с правами на редактирование инфоблока
- Включённая административная панель (режим правки)

## Шаги

### Редактирование видео

1. В режиме правки наведите на блок → «Изменить элемент»
2. В поле «Видео» загрузите новый файл (MP4)
3. Сохраните

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- При загрузке нескольких видео отображается только одно — с самым поздним ID
- Название блока можно поменять через управление блоками
