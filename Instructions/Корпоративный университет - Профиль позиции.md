# Профиль позиции

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Профиль (карточка) позиции содержит описание позиции, перечень компетенций и развивающие действия для их освоения.

## Ключевые функции

### Где открыть профиль

1. Из блока с позиции — три точки (...) → «Открыть профиль позиции»
2. Из слайдера с информацией — три точки (...) → «Открыть профиль позиции»

### Структура профиля

1. **Описание позиции и данные о развитии**
   - Задачи специалиста и необходимые знания
   - Аналитика: Работают, Развиваются, В резерве

2. **Компетенции**
   - Фильтры по разделам компетенций
   - Фильтры по приоритетности (Обязательные / Рекомендуемые)
   - Индикаторы уровня владения (серая галочка, синяя галочка, академическая шапочка)

3. **Развивающие действия**
   - Фильтры по обязательности и типам действий
   - Все шесть типов развивающих действий

## Предусловия

- Доступно всем пользователям для просмотра
- Редактирование: администраторы и группа HR-отдела
