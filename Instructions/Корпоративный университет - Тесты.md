# Тесты

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Создание и настройка тестов для контроля усвоения материала онлайн-курсов.

## Ключевые функции

### Добавление теста

- Курс → «Настройки» (⚙️) → «Тесты курса» → «Добавить тест»

### Настройки теста

1. **Название** (обязательное)
2. **Описание**
3. **Выводится в** — привязка к корню курса / главе / уроку
4. **Запретить доступ к материалам после теста, если не сдан**
5. **В тесте участвуют** — выбор вопросов (все, из главы, заданное количество)
6. **Тип прохождения теста** — ограничения при переключении между вопросами
7. **Добавить оценки** — процент правильных ответов → оценка → примечание
8. **Активность**
9. **Количество попыток** (0 = не ограничено)
10. **Сортировка**
11. **Дней/часов/минут между попытками**
12. **Время на прохождение** — таймер
13. **Случайный порядок вопросов / ответов**
14. **Автоматическая проверка результатов**
15. **% для прохождения**
16. **Завершать попытку, если % недостижим**
17. **Настройка результатов** — что видит пользователь
18. **Настройка доступа к тесту** — prerequisite-тест

### Отображение в разделах курса

- К главе — после списка уроков
- К уроку — иконка урока меняется на главу
- К корню курса — раздел «Итоговые тесты»

### Цветовая индикация

- **Зелёный** — успешно пройден
- **Красный** — неудовлетворительный результат / в процессе
- **Серый** — не приступал

### Список тестов

- «Настройки» (⚙️) → «Тесты курса» → «Список тестов»
- Действия: Редактировать, Активировать/Деактивировать, Удалить

## Предусловия

- Права: администраторы, HR-отдел, пользователи с правами на курс
- Созданные вопросы для тестов

## Граничные случаи

- При удалении теста удаляются все попытки прохождения
- Текстовые вопросы требуют ручной проверки
