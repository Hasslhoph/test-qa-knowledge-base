# HelpDesk - Интеграция с Jira

**Модуль:** [[Module - HelpDesk]]

## Описание

Настройка интеграции HelpDesk с Jira для синхронизации обращений.

## Шаги

1. Перейти в раздел **Интеграции** в настройках модуля HelpDesk.
2. Выбрать **Jira** из списка доступных интеграций.
3. Настроить **токен** доступа к Jira:
   - Указать URL экземпляра Jira.
   - Ввести API-токен или логин/пароль.
4. Сохранить настройки.

## Проверка

После сохранения настроек убедитесь, что статус интеграции отображается как **Активно**.
