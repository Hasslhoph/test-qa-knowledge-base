# История развития

**Модуль:** [[Module - Главная страница]]

## Описание

Блок на странице «О компании» для отображения ключевых этапов становления и развития компании: расширение продуктовой линейки, выход на новые рынки, внедрение новых инструментов.

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- **Разделы (этапы)** — кнопки для переключения между годами/этапами развития
- **Детальное описание** каждого этапа с иллюстрацией
- **Хронологическая сортировка** разделов
- **Создание новых этапов** развития
- **Переименование и изменение очередности** через [[Главная страница - Управление блоками]]

## Предусловия

- Права администратора или пользователя с правами на редактирование инфоблока
- Включённая административная панель (режим правки)

## Шаги

### Создание этапа развития

1. `/bitrix/admin/` → «Контент» > «О компании» > «История развития» > «Моя компания»
2. Нажмите «Добавить элемент»
3. Вкладка «Параметры»:
   - Активность, Название (кнопка), Сортировка (хронология)
   - Заголовок (для блока с детальной информацией)
4. Вкладка «Анонс»:
   - Картинка для анонса (пропорции сохраняются)
   - Описание для анонса (детальное описание этапа)
5. Сохраните

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- По умолчанию разделами являются года, но можно использовать любые маркеры
- Ширина и высота картинки зависят от разрешения экрана
