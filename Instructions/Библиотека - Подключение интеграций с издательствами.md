# Подключение интеграций с издательствами

**Модуль:** [[Module - Библиотека]]

## Описание

Инструкция по подключению интеграции Библиотеки с издательствами «МИФ» или «Альпина Паблишер» для автоматического обновления каталога. Подключить можно только одно издательство.

## Ключевые функции

### Интеграция с МИФ (7 полей)

- **Время запуска** — время синхронизации данных
- **ID библиотеки** — предоставляется МИФ
- **Email адрес администратора библиотеки** — предоставляется МИФ
- **Ссылка на api miflib** — предоставляется МИФ
- **Secret key для пользования API** — предоставляется МИФ
- **Company Name** — предоставляется МИФ (соответствует полю Audience в ЛК)
- **Учитывать период активности книги** — ссылки на скачивание активны пока действует подписка

### Интеграция с Альпина Паблишер (5 полей)

- **Время запуска** — время синхронизации данных
- **API Key** — предоставляется Альпина
- **API Secret** — предоставляется Альпина
- **Адрес API** — предоставляется Альпина
- **Адрес хранилища** — предоставляется Альпина

### Периодичность обновления

- По умолчанию: каждый день в 00:00
- Настраивается через Агентов Битрикс24
- Агент МИФ: `ITHive\Library\Service\Miflib::initImport(1);`
- Агент Альпина: `ITHive\Library\Service\Alpina::executeImport();`
- Интервал по умолчанию: 86400 секунд (1 сутки)

## Предусловия

- Права администратора портала
- Активная подписка на издательство (МИФ или Альпина)
- Данные для подключения из личного кабинета издательства

## Шаги

### Подключение интеграции

1. Перейдите: `адрес_портала/bitrix/admin`
2. «Настройки» → «Настройки модулей» → «Улей: Библиотека»
3. Перейдите на вкладку «Интеграция с МИФ/Альпина»
4. Выберите одно издательство
5. Активируйте галочку «Активность»
6. Заполните поля (значения из ЛК издательства)
7. Нажмите «Сохранить»

### Настройка периодичности обновления

1. Перейдите: `адрес_портала/bitrix/admin`
2. «Настройки» → «Настройки продукта» → «Агенты»
3. В поле «Модуль» введите «ithive.library»
4. Выберите нужный Агент (МИФ или Альпина)
5. Нажмите бургер-меню (≡) → «Изменить»
6. Настройте:
   - **Дата и время следующего запуска** — первая синхронизация по умолчанию на следующий день в 00:00
   - **Интервал (сек.)** — периодичность обновления (по умолчанию 86400 сек = 1 сутки)
7. Сохраните

## Граничные случаи

- Подключить можно только одно издательство — механизмы МИФ и Альпина не должны пересекаться
- Значения для полей 2–6 (МИФ) и 2–5 (Альпина) берутся из личного кабинета организации на сайте издательства
- При опции «Учитывать период активности книги» (МИФ) ссылки на скачивание активны только пока действует подписка
- Первая синхронизация происходит на следующий день после подключения интеграции в 00:00
