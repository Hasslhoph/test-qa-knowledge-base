# Установка, обновление и удаление модуля

**Модуль:** [[Module - Карта офиса + Бронирование]]

## Описание

Инструкция по установке, обновлению, активации купона и удалению модуля «Карта офиса + Бронирование рабочих мест» (`ithive.workplaces`).

## Ключевые функции

- Установка модуля через административную часть портала
- Обновление модуля через Marketplace
- Активация купона на покупку/продление
- Удаление модуля

## Предусловия

- Права на установку, обновление, активацию купона и удаление есть только у группы «Администраторы»

## Установка модуля

1. Перейти: `адрес_портала/bitrix/admin/update_system_partner.php?addmodule=ithive.workplaces`
2. Модуль автоматически отмечен галочкой — нажать «Загрузить»
3. Подтвердить согласие с Лицензионным соглашением и Условиями конфиденциальности
4. Нажать «Применить» — установка начнётся автоматически
5. Дождаться завершения установки
6. Уведомление об успешной загрузке — установка завершена

## Обновление модуля

1. Marketplace → Обновление решений → Список обновлений
2. Выбрать решение «Улей: Карта офиса + бронирование рабочих мест»
3. Нажать «Загрузить»

Или сразу перейти: `адрес_портала/bitrix/admin/update_system_partner.php`

## Активация купона

1. Marketplace → Обновление решений → Активация купона
2. Ввести номер купона и нажать «Активировать купон»

Или сразу перейти: `адрес_портала/bitrix/admin/update_system.php?lang=ru`

### На неустановленный модуль
1. Marketplace → Установленные решения → Купленные решения
2. Бургер-меню рядом с модулем → «Загрузить»

### Для модуля с истёкшей демоверсией
1. Marketplace → Обновление решений → Установка обновлений
2. Нажать «Зарегистрировать решение»

## Удаление модуля

1. Marketplace → Установленные решения → Доступные решения
2. Бургер-меню слева от решения → «Удалить»
3. Подтвердить удаление после обновления страницы
4. Сообщение об удалении модуля

## Граничные случаи

- Все операции (установка, обновление, купон, удаление) требуют прав администратора
- При переустановке модуля PHP-файл блока «Возникла проблема?» будет перезаписан
- Обновления загружаются автоматически вместе с установкой модуля
