# Интерфейс Legacy

**Модуль:** [[Module - Брендирование Битрикс24]]

## Описание

Статья описывает интерфейс модуля «Брендирование» для необновлённой версии Битрикс24 — «Legacy» (до версии модуля intranet 25.900.0). Работа с модулем выполняется в публичной части портала. Открыть настройки можно через пункт в левом меню или через меню профиля (блок «Тема оформления»).

## Ключевые функции

Модуль состоит из десяти разделов:

- **Настройки** — общие настройки модуля (включение/выключение, режим только для администратора, сброс), экспорт и импорт настроек в ZIP
- **Общие** — шрифт на всех страницах (4 варианта), цвет фона портала (HEX-код), скрытие/показ виджетов, загрузка фавикона
- **Авторизация** — настройка страницы входа: логотип, фон, цвета кнопок и ссылок для всех состояний, GIF-анимация, чекбокс «Запомнить меня»
- **Виджеты** — стилизация 8 виджетов Ленты (Пульс компании, Мои задачи, Дни рождения, Ближайшие события, Важные сообщения, Опросы, Бизнес-процессы, Ссылки). Общие и индивидуальные настройки
- **Шапка портала** — верхняя панель, левое меню, ссылки в верхнем меню, правый сайдбар
- **Кнопки** — цвета ссылок и кнопок (зелёные, белые, прозрачные, синие) для всех состояний
- **Контент** — фон важных сообщений, иконка лайка, плашки дней рождения, набор иконок файлов (PNG спрайт)
- **Другое** — цвет выделенной папки диска, цвет шапки виджета в задачах
- **Дополнительно** — поле для произвольных CSS-стилей

## Предусловия

- Пользователь должен иметь доступ к модулю (администратор или группа с правами)
- У пользователя должна быть включена классическая тема портала («Битрикс 24» или «Битрикс 24 с узором»)

## Граничные случаи

- При отключении модуля настройки не сбрасываются — только скрываются
- При экспорте сохраняются все настройки, включая состояние «Выключить работу модуля» и «Модуль работает только для администратора»
- CSS-стили из раздела «Дополнительно» экспортируются, но не сбрасываются кнопкой «Сбросить все настройки»
- Набор иконок «Плитка» и иконки бизнес-процессов являются рудиментарными опциями (начиная с ui 17.5.5)
- Индивидуальные настройки виджетов имеют приоритет над общими
