# О продукте

**Модуль:** [[Module - Главная страница]]

## Описание

Инструкция к модулю «Улей: Главная страница». Модуль позволяет создать единую точку входа на корпоративный портал с быстрым доступом ко всем ключевым сервисам, новостям и любым другим корпоративным материалам.

## Ключевые функции

> Техническая информация о модуле.

- Модуль совместим с PHP версией 8.1 и выше
- Модуль совместим с СУБД: MySQL, PostgreSQL
- Модуль поддерживает английский язык
- Модуль поддерживает многодепартаментность
- Функционал модуля доступен через современный десктопный браузер
- Инструкция актуальна для версии модуля: 8.0.0

## Предусловия

- Установленный Битрикс24
- PHP 8.1+
- СУБД MySQL или PostgreSQL
