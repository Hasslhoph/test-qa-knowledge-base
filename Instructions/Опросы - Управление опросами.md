# Управление опросами

**Модуль:** [[Module - Опросы]]

## Описание

Слайдер «Управление опросами» для массовой работы с опросами. Доступен только администраторам и редакторам опросов.

## Ключевые функции

- Список всех существующих опросов: активных, деактивированных, прошедших, запланированных
- Настраиваемые колонки: ID опроса, Название, Описание, Дата начала, Дата окончания, Активность, Группа опроса, Количество голосов, Автор
- Поиск с учётом статусов, дат, групп и других параметров
- Действия с отдельным опросом (бургер-меню ≡): Предпросмотр, Редактировать, Активировать/Деактивировать, Копировать, Удалить
- Массовые действия: выделение нескольких опросов галочками → выбор действия из списка

## Предусловия

- Пользователь — администратор или редактор опросов
- Модуль установлен и активен

## Шаги

1. Перейти на главную страницу модуля «Опросы»
2. Нажать шестерёнку (⚙️) в правом верхнем углу
3. Выбрать «Управление опросами»
4. Использовать поиск и фильтры для нахождения нужных опросов
5. Выполнить нужное действие через бургер-меню или массовые действия

## Ожидаемый результат

Операции (редактирование, активация, копирование, удаление) выполнены над одним или несколькими опросами.
