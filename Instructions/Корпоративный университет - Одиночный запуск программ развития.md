# Одиночный запуск программ развития

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Индивидуальный запуск программ развития через профиль конкретного пользователя.

## Ключевые функции

### Права на одиночный запуск

- **Администратор** — для любых пользователей
- **Группа HR-отдела** — для любых пользователей
- **Руководитель** — только для прямых подчинённых
- **Обычный пользователь** — только для себя (при включённых опциях в настройках)

### Три варианта запуска

1. **По шаблону программы развития/адаптации** — заранее подготовленный набор компетенций
2. **На основе карьерной траектории/текущей позиции** — компетенции из профиля позиции
3. **Свободный выбор компетенций** — ручной выбор компетенций

### Запуск по шаблону

- Выбор шаблона → компетенции автоматически выбраны
- Поля: Руководитель, Отправить на согласование, Наблюдатели, Срок, Целевая должность/позиция, Следующая программа

### Запуск на основе карьерной траектории

- На основе карьерной траектории — развитие в вышестоящую/смежную позицию
- На основе текущей позиции — развитие в рамках текущей позиции
- Поле «Целевая позиция» заблокировано

### Свободный выбор компетенций

- Компетенции не зависят от шаблонов и позиций
- Без выбора компетенций запуск невозможен

### Сроки и развивающие действия

- Для каждой компетенции — выбор даты дедлайна
- Кнопка «Действий N/M» — выбор развивающих действий
- Кнопка «Добавить» — добавить любые другие виды развивающих действий

### Лимит на количество программ

- По умолчанию: не более 2 программ в работе
- В настройках модуля можно увеличить до 5

## Предусловия

- Права на запуск программ развития
- Созданные шаблоны (для запуска по шаблону)
- Привязка сотрудника к позиции (для запуска по карьерной траектории)

## Граничные случаи

- Если лимит превышен — кнопка запуска заблокирована
- Компетенции с серой галочкой нельзя убрать
- Компетенции с академической шапочкой не выбраны, но можно добавить
