# Геймификация - Описание и концепция модуля

**Модуль:** [[Module - Геймификация]]

## Описание

Модуль «Геймификация» добавляет на корпоративный портал игровые механики: рейтинги, уровни (ранги), игровую валюту, магазин бонусов. Модуль делает повседневную работу более интересной и позволяет получать выгоду даже выполняя рутинные действия.

## Ключевые функции

- **Участие в рейтингах** — шесть автоматических рейтингов (Лидер продаж, Эффективный исполнитель задач и др.), призеры получают баллы
- **Накопление баллов** — внутренняя валюта за рейтинги, благодарности, круглые даты работы, выполнение задач
- **Сказать «Спасибо»** — ежемесячное начисление баллов «Спасибо» для поощрения коллег
- **Покупки за баллы** — магазин бонусов с мерчем, гаджетами, сертификатами, оплачиваемыми выходными
- **Повышение уровня сотрудника** — система рангов, зависящая от суммы баллов за всё время
- **Поддержка корпоративных ценностей** — категории «Спасибо» соответствуют ценностям компании

## Предусловия

- Модуль «Геймификация» установлен и активирован
- Пользователь имеет доступ к публичной части модуля

## Шаги

1. Изучите концепцию модуля: облегчение восприятия работы + поддержка корпоративных ценностей
2. Ознакомьтесь с пятью основными возможностями модуля
3. Перейдите к детальным инструкциям по каждому разделу

## Ожидаемый результат

Пользователь понимает концепцию модуля, его основные возможности и связь с корпоративными ценностями.

## Граничные случаи

- Геймификация учитывает психологию поведения: награды, признание коллег, соревновательный азарт
- Модуль эффективен для решения насущных задач компании (конкурсы, сезонные активности)
