# Создание разделов позиций

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Создание и редактирование разделов для позиций на схеме карьерных траекторий. Каждая позиция должна быть привязана к разделу.

## Ключевые функции

### Добавление разделов

1. Схема карьерных траекторий → бургер-меню (≡) → «Управление разделами»
2. Нажать «Добавить»
3. Заполнить поля:
   - **Название** — наименование раздела (например, «Отдел разработки»)
   - **Родительский раздел** — для создания вложенности
   - **Активность** — скрытие раздела от обычных пользователей
   - **Сортировка** — порядок разделов (меньший индекс = выше)

### Редактирование разделов

- Бургер-меню (≡) → «Изменить»

### Копирование разделов

- Бургер-меню (≡) → «Копировать»

### Удаление разделов

- Бургер-меню (≡) → «Удалить»
- ⚠️ Удаляются все вложенные подразделы и позиции

### Просмотр подразделов

- Клик на название раздела → список подразделов
- Вложенность разделов не ограничена

## Предусловия

- Права: администраторы и группа HR-отдела
