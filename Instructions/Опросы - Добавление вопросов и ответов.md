# Добавление вопросов и ответов

**Модуль:** [[Module - Опросы]]

## Описание

Инструкция по созданию, редактированию и управлению вопросами и ответами в опросах. Доступно только администраторам и редакторам опросов.

## Ключевые функции

### 8 типов вопросов

**Стандартные вопросы:**
- **Одиночный выбор** — выбор одного варианта
- **Множественный выбор** — выбор нескольких вариантов
- **Один из списка** — выбор одного из выпадающего списка
- **Несколько из списка** — выбор нескольких из списка

**Специальные вопросы:**
- **Целое число** — ответ в виде целого числа (настраиваемый диапазон)
- **Дробное число** — ответ в виде дробного числа (настраиваемый диапазон и точность 1-3 знака)
- **Шкала** — оценка по шкале с настраиваемыми крайними подписями и количеством точек
- **Ранжирование** — расстановка вариантов в порядке приоритета (всегда тепловая диаграмма)

### Параметры стандартного вопроса

| Параметр | Описание |
|---|---|
| Тип вопроса | Переключение между 4 стандартными типами |
| Вопрос | Текст вопроса (обязательно) |
| Картинка | Иллюстрация к вопросу |
| Активность | Видимость вопроса для респондентов |
| Обязательный вопрос | Без ответа нельзя перейти дальше |
| Показывать результаты на диаграмме | Вывод итогов на странице результатов |
| Вид диаграммы | Круговая диаграмма или гистограмма |
| Добавить связь с вопросом | Ветвление — последующие вопросы зависят от ответов |
| Ответы | Список ответов с управлением |

### Параметры ответа

| Параметр | Описание |
|---|---|
| Активность | Видимость ответа |
| Сделать своим ответом | Текстовое поле (строка или текстовое поле) для свободного ответа |
| Ответ | Текст ответа (обязательно) |
| Комментарий к ответу | Поле для комментария (может быть обязательным) |
| Цвет ответа на диаграмме | Цвет варианта на диаграмме |
| Порядок сортировки | Очередность ответов; одинаковые значения = случайный порядок |
| Добавить изображение | Иллюстрация к ответу (JPG, PNG) |

### Параметры специальных вопросов

- **Целое число / Дробное число:** минимальное/максимальное значение, кол-во столбцов на диаграмме
- **Дробное число:** символы после запятой (1-3)
- **Шкала:** левая/правая подпись, мин/макс значение, количество точек на шкале
- **Ранжирование:** всегда тепловая диаграмма, нельзя добавить комментарий к ответу

### Управление вопросами

- Предпросмотр, Редактировать, Копировать, Деактивировать, Сделать обязательным, Скрыть результаты на диаграмме, Удалить
- Массовые действия с несколькими вопросами (выделить галочками)

## Предусловия

- Опрос сохранён (вопросы можно создать только для существующего опроса)
- Пользователь — администратор или редактор опросов

## Шаги

1. Открыть слайдер редактирования опроса
2. Нажать «Добавить вопрос» → выбрать тип вопроса
3. Заполнить параметры вопроса
4. Добавить варианты ответов через «Добавить ответ»
5. Сохранить вопрос
6. При необходимости настроить ветвление (связь с другим вопросом)

## Ожидаемый результат

Вопросы и ответы добавлены в опрос и доступны для респондентов.

## Граничные случаи

- Комментарий нельзя добавить к текстовому полю для свободного ответа
- Для типа «Ранжирование» нельзя добавить комментарий к ответу и нельзя выбрать вид диаграммы (всегда тепловая)
- Исключённые типы вопросов в настройках модуля не пропадают из уже созданных опросов
- При одинаковом порядке сортировки ответы перемешиваются случайно при каждом открытии
