# HelpDesk - Ссылка на шаг создания обращения

**Модуль:** [[Module - HelpDesk]]

## Описание

Копирование ссылки на конкретный шаг создания обращения для размещения в любом разделе корпоративного портала. Позволяет сотрудникам отправлять тикеты без перехода в модуль HelpDesk.

## Ключевые функции

- Доступно только администраторам портала
- Ссылка ведёт на любой шаг кроме финального (предпросмотр)
- Можно предзаполнить поля перед копированием (например, готовый текст обращения)
- По клику открывается слайдер заполнения обращения
- Интеграция с [[Module - Карта офиса + Бронирование]] — ссылка на слайдере бронирования в блоке «Возникла проблема»

## Предусловия

- Права администратора портала
- Создана ветка обращений с нужными шагами

## Граничные случаи

- Ссылка не может вести на финальный шаг (предпросмотр)
