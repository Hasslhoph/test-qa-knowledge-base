# Оценка 360 - Анализ результатов оценки в Excel

**Модуль:** [[Module - Оценка 360]]

## Описание

Раздел «Оценки» модуля «Оценка 360» позволяет выгружать данные об оценках сотрудников в CSV-формате. С помощью Excel можно преобразовать выгруженные данные в наглядную аналитику: построить сводные таблицы, графики и отследить динамику развития компетенций по подразделениям и периодам.

## Ключевые функции

- Выгрузка данных об оценках в CSV-формате
- Импорт CSV в Excel с корректной кодировкой (UTF-8)
- Построение сводных таблиц со средними оценками по компетенциям
- Фильтрация и группировка по подразделениям и периодам
- Построение гистограмм с линиями тренда, метками данных и таблицами данных
- Настройка фильтров для выборочного отображения подразделений на графике

## Предусловия

- Наличие завершённых оценок в модуле «Оценка 360»
- Доступ к разделу «Оценки» модуля
- Установленный Microsoft Excel (или аналог, поддерживающий сводные таблицы и CSV)
- CSV-файл, скачанный из модуля через «Экспорт списка» > «CSV»

## Шаги

### 1. Выгрузка и импорт данных

1. Перейдите в раздел «Оценки» модуля «Оценка 360».
2. Нажмите на шестеренку (⚙️).
3. Выберите «Экспорт списка» > «CSV».
4. Откройте Excel, на панели инструментов нажмите «Данные» > «Из текстового/CSV-файла».
5. Выберите скачанный CSV-файл.
6. В поле «Источник файла» выберите «65001: Юникод (UTF-8)».
7. В поле «Разделитель» укажите «Точка с запятой».
8. Нажмите «Загрузить».

### 2. Построение сводной таблицы

1. Выделите таблицу (Ctrl+A).
2. Перейдите во вкладку «Вставка» > «Сводная таблица», нажмите «ОК».
3. В окне «Поля сводной таблицы» отметьте компетенции для анализа (выберите тип оценки: «Среднее» или «Среднее без самооценки»).
4. В блоке «Значения» для каждой компетенции через «Параметры полей значений» выберите «Среднее».
5. Для улучшения вида:
   - Уменьшите количество знаков после запятой до одного.
   - Сократите названия столбцов, оставив только названия компетенций.
   - Перенесите «Значения» из блока «Столбцы» в блок «Строки».
6. Перетащите поле «Дата завершения» в блок «Столбцы», оставьте только «Годы».
7. Перетащите поле «Подразделения» в блок «Строки».

### 3. Построение графика

1. Выделите сводную таблицу (Ctrl+A).
2. Нажмите «Вставка» и выберите стандартную гистограмму.
3. Переместите гистограмму на отдельный лист для удобства.
4. При необходимости измените цвет столбцов.
5. Настройте отображение:
   - **Фильтры по подразделениям** — для вывода данных по конкретным отделам.
   - **Линии тренда** — для визуализации роста или падения.
   - **Метки данных** — для отображения числовых значений на графике.
   - **Таблица данных** — для детализации оценок по годам и отделам.

## Ожидаемый результат

Сформирована наглядная аналитика в Excel: сводная таблица со средними оценками по компетенциям в разрезе подразделений и периодов, а также гистограмма с линиями тренда, метками данных и возможностью фильтрации. Пользователь может отследить динамику развития компетенций в компании за выбранные периоды.

## Граничные случаи

- **Некорректная кодировка CSV**: при импорте необходимо вручную выбрать UTF-8 (65001) и разделитель «Точка с запятой», иначе данные отобразятся некорректно.
- **Пустые значения оценок**: если по компетенции нет оценок, в сводной таблице могут отображаться пустые ячейки — следует проверить исходные данные.
- **Большое количество компетенций**: при выборе всех компетенций сразу сводная таблица может стать перегруженной — рекомендуется фильтровать только нужные.
- **Отсутствие поля «Дата завершения»**: если оценки не завершены, данные за период могут отсутствовать — убедитесь, что оценки имеют статус «Завершена».
- **Разные версии Excel**: для версий Excel 2021 и старше могут отличаться названия пунктов меню — используйте универсальные формулы при необходимости.
