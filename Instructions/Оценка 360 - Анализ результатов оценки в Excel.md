# Оценка 360 - Анализ результатов оценки в Excel

**Модуль:** [[Module - Оценка 360]]

# Анализ результатов оценки в Excel

Раздел «Оценки» позволяет выгружать данные об оценках сотрудников в CSV-формате для анализа в Excel. В этой статье мы рассмотрим, как с помощью Excel преобразовать выгруженные данные в наглядную аналитику. 

## Выгрузка данных об оценках

Чтобы выгрузить данные об оценках сотрудников перейдите в раздел модуля «Оценки» и нажмите на шестеренку (⚙️). Выберите «Экспорт списка» > «CSV». В этот момент будет скачан csv-файл.

![](_images-08/image104.png)

Как правило, данные в csv-файлах отображаются в виде кодировки. Чтобы преобразовать данные в понятный вид, на панели инструментов нажмите «Данные» > «Из текстового/CSV-файла». Затем выберите csv-файл, скачанный из модуля «Оценка 360».

![](_images-08/image520.png)

![](_images-08/image373.png)

В новом окне в поле «Источник файла» выберите значение «65001: Юникод (UTF-8)». В поле «Разделитель» укажите значение «Точка с запятой». Нажмите на кнопку «Загрузить».

![](_images-08/image148.png)

В результате данные об оценке отобразятся в корректном для анализа виде.

![](_images-08/image77.png)

## Построение сводной таблицы

В качестве примера для этой статьи взят файл с данными о завершенных оценках за 2023, 2024 и 2025 годы. В каждой оценке участвовали сотрудники шести отделов компании:

1. Отдел информационно-технический поддержки (ИТП)

2. Отдел маркетинга и рекламы

3. Производственный отдел

4. Отдел продаж

5. Отдел партнерских продаж

6. Отдел прямых продаж

Оценка производилась по следующим компетенциям:

1. Клиентоориентированность

2. Проактивность

3. Постановка задач

4. Работа на результат

5. Контроль. Требовательность

6. Управление конфликтами

На основе этих данных покажем, как проанализировать динамику развития компетенций в подразделениях.

![](_images-08/image198.png)

Выделите свою таблицу с помощью команды Ctrl+A. Перейдите во вкладку «Вставка» > «Сводная таблица». В открывшемся окне нажмите «ОК».

![](_images-08/image243.png)

![](_images-08/image153.png)

В результате в файле будет создан новый лист со сводной таблицей. Теперь поля этой таблицы нужно настроить, то есть вывести колонки и строки таким образом, чтобы получить желаемую аналитику.

![](_images-08/image320.png)

Для начала в окне «Поля сводной таблицы» отметьте галочками все компетенции, по которым вы хотите отследить динамику. Здесь вы увидите, что у каждой компетенции есть два вида оценки: «Среднее» и «Среднее без самооценки». Для построения аналитики выберите ту оценку, которая по вашему мнению является более показательной. В данном примере мы будем строить динамику на основе значения «Среднее».

![](_images-08/image516.png)

Отмеченные галочками поля автоматически попали в блок «Значения» и отобразились в таблице как колонки. По умолчанию Excel поместил в колонки сумму средних оценок по компетенциям, но суммарные оценки не интересуют нас с точки зрения аналитики — нам важно знать средние значения.

![](_images-08/image253.png)

Чтобы изменить значения в колонках, в блоке «Значения» кликните левой кнопкой мыши на любую компетенцию и выберите «Параметры полей значений».

![](_images-08/image379.png)

В списке значений выберите «Среднее» и нажмите «ОК». Аналогичным способом настройте значения во всех компетенциях. Теперь в таблице отражены средние оценки по каждой компетенции без разбивки на сотрудников и отделы.

![](_images-08/image208.png)

![](_images-08/image21.png)

На этом этапе нужно привести данные в более удобный для восприятия вид. Для этого выполните следующие действия:

1. Выделите строку с числовыми значениями и уменьшите количество чисел после запятой до одного.

![](_images-08/image217.png)

![](_images-08/image178.png)

2. Дважды кликните на названия столбцов и удалите лишние слова из названий, чтобы их сократить. Например, можно оставить только названия компетенций.

![](_images-08/image278.png)

3. В блоке «Столбцы» зажмите пункт «Значения» и перенесите его в блок «Строки». В результате компетенции и оценки по ним выстроятся в две колонки — так таблица станет более компактной и наглядной.

![](_images-08/image428.png)

![](_images-08/image83.png)

Теперь можно добавить в таблицу даты завершения оценки, чтобы увидеть изменение средних оценок за разные периоды. Для этого мышкой перетащите значение «Дата завершения» в блок «Столбцы». Удалите лишние значения и оставьте только поле «Годы».

![](_images-08/image35.png)

![](_images-08/image221.png)

Таблица обновится и вы увидите, какими были средние оценки по компетенциям за все периоды проведения оценки, в данном случае за три года.

![](_images-08/image307.png)

На этом этапе нужно добавить еще один разрез аналитики — подразделения. Для этого мышкой переместите поле «Подразделения» в блок «Строки».

![](_images-08/image209.png)

В результате будет сформирована таблица с наглядной динамикой оценок по компетенциям в разрезе подразделений. Чтобы визуализировать динамику, на основе полученной таблицы можно построить график.

![](_images-08/image110.png)

## Построение графика

Выделите всю таблицу командой Ctrl+A и нажмите «Вставка». Выберите стандартную гистограмму.

![](_images-08/image48.png)

Для удобства создайте в таблице новый лист, скопируйте туда полученную гистограмму и растяните ее до удобного размера. При необходимости измените цвет столбцов.

![](_images-08/image279.png)

![](_images-08/image421.png)

На графике видно, как в течение трех лет менялся уровень владения компетенциями в отделах компании. Для более подробного анализа можно воспользоваться настройками диаграммы:

- **Фильтры по подразделениям.** Фильтры помогут вывести на графике данные только по конкретным подразделениям, по одному или нескольким.

![](_images-08/image354.png)

![](_images-08/image214.png)

- **Линии тренда.** Элементы, которые визуально покажут рост или падение в развитии компетенций.

![](_images-08/image232.png)

- **Метки данных.** Числовые данные на графике, в нашем случае это средние оценки по компетенциям в каждом отделе.

![](_images-08/image401.png)

- **Таблица данных.** Детализация оценок по компетенциям в каждом отделе за каждый год.

![](_images-08/image497.png)

Таким образом с помощью Excel можно проанализировать результаты оценки сотрудников, полученные в модуле «Оценка 360».
