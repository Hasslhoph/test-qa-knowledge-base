# Оценка 360 - Анализ результатов оценки в Excel

**Модуль:** [[Module - Оценка 360]]

## Описание

Выгрузка данных об оценках в CSV-формате и построение аналитики в Excel с помощью сводных таблиц и графиков.

## Ключевые функции

### Выгрузка данных
- Раздел «Оценки» → шестерёнка (⚙️) → «Экспорт списка» → «CSV»
- Преобразование кодировки: «Данные» → «Из текстового/CSV-файла» → кодировка UTF-8, разделитель «Точка с запятой»

### Построение сводной таблицы
1. Выделить таблицу (Ctrl+A) → «Вставка» → «Сводная таблица»
2. Отметить компетенции, выбрать «Среднее» или «Среднее без самооценки»
3. Изменить агрегатную функцию с «Сумма» на «Среднее»
4. Уменьшить количество знаков после запятой до одного
5. Перенести «Значения» из «Столбцы» в «Строки»
6. Добавить «Дата завершения» в «Столбцы» → оставить только «Годы»
7. Добавить «Подразделения» в «Строки»

### Построение графика
1. Выделить таблицу → «Вставка» → гистограмма
2. Настройки диаграммы:
   - **Фильтры по подразделениям** — вывод данных по конкретным отделам
   - **Линии тренда** — визуальный рост/падение компетенций
   - **Метки данных** — числовые данные на графике
   - **Таблица данных** — детализация оценок

## Предусловия

- Права на выгрузку оценок (настраивается в модуле)
- Завершённые оценки с данными для анализа

## Граничные случаи

- CSV-файл может содержать большое количество колонок при множестве компетенций и индикаторов
- Для BI-аналитики требуется дополнительная подготовка файла
