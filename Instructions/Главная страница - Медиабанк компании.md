# Медиабанк компании

**Модуль:** [[Module - Главная страница]]

## Описание

Блок на странице «О компании» для публикации тематических материалов: бенефиты, система развития, достижения компании, направления работы и продукты.

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- **Тематические блоки** с заголовками слева и детальным описанием справа
- **Кликабельное меню** — по клику на заголовок открывается блок с описанием
- **Бегунок прокрутки** при большом объёме контента
- **Создание новых тематических блоков**
- **Переименование и изменение очередности** через [[Главная страница - Управление блоками]]

## Предусловия

- Права администратора или пользователя с правами на редактирование инфоблока
- Включённая административная панель (режим правки)

## Шаги

### Редактирование блока

1. `/bitrix/admin/` → «Контент» > «О компании» > «Медиабанк компании» > «Моя компания»
2. Кликните на название нужного блока
3. Вкладка «Параметры»:
   - Активность, Название (тематический заголовок), Сортировка
4. Вкладка «Анонс»:
   - Картинка для анонса (пропорции сохраняются)
   - Описание для анонса (детальная информация)
5. Сохраните

### Создание нового блока

1. Перейдите в инфоблок медиабанка
2. Нажмите «Добавить элемент»
3. Заполните поля аналогично редактированию
4. Сохраните

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- Ширина и высота картинки зависят от разрешения экрана
- Изображение выводится с сохранением пропорций (не обрезается)
