# Описание и концепция модуля

**Модуль:** [[Module - Управление целями. Каскад]]

## Описание

**Управление целями. Каскад** (УЦ. Каскад) — инструмент для автоматизации процесса постановки целей в компании. Модуль позволяет вести список корпоративных целей, ставить эти цели нужным подразделениям и контролировать их достижение по принципу каскадирования.

## Ключевые функции

> Основные концепции и возможности модуля.

- **Каскадирование целей** — цели передаются от руководства к подчиненным через оргструктуру
- **Интеграция с Битрикс24** — использует существующую оргструктуру и задачи портала
- **Методология OKR** — поддержка амбициозных целей и ключевых результатов (критериев)
- **Регулярные отчеты о прогрессе** — сотрудники заполняют отчеты, которые согласуются в несколько этапов
- **Сводные отчеты** — средний процент достижения цели по подразделениям
- **Индивидуальные цели** — сотрудники могут создавать личные цели с собственными критериями
- **Функциональная подчиненность** — возможность указать функционального руководителя помимо прямого
- **Многоэтапное согласование** — руководитель → функциональный руководитель → директор → администратор

## Интеграция с оргструктурой и задачами Битрикс24

- Процесс целеполагания запускается на основе существующей оргструктуры
- Связь с оргструктурой позволяет запускать процесс на нужный отдел
- Руководители каскадируют задачи подчиненным, которые определяются автоматически
- Сотрудники могут привязывать задачи к своим карточкам целей

## Гибкий механизм управления целями

- Управление измеримыми и абстрактными целями
- Планирование на квартал, полгода, год и дольше
- Перезапуск процесса с теми же параметрами
- Индивидуальные цели с собственными критериями и процентом значимости

## Многоэтапное согласование

Цели согласуются поэтапно:
1. Руководитель
2. Функциональный руководитель (если есть)
3. Директор
4. Администратор процесса

Отчеты о прогрессе проходят те же этапы согласования.

## Кому подойдет модуль

Компаниям, которым нужен инструмент для:
- **Постановки целей** — пользователи ставят цели и каскадируют их по оргструктуре
- **Согласования целей** — контроль объективности и релевантности целей руководством
- **Корректировки целей** — изменение критериев из-за внешних факторов
- **Сбора отчетности о прогрессе** — регулярные отчеты с настраиваемой периодичностью
- **Согласования прогресса** — внесение корректировок и отслеживание динамики
- **Анализа результатов** — сводная таблица с средним процентом достижения по подразделениям

## Предусловия

- Установленный и настроенный Битрикс24
- Настроенная организационная структура
- Определенные корпоративные цели (рекомендуется провести стратсессию)

## Граничные случаи

- Не всем компаниям нужно автоматизировать процесс постановки целей — если цели уже сформулированы и распределены, рассмотрите модуль [[Module - Управление целями. Дерево]]
- Модуль подходит когда процесс постановки целей происходит во время работы с модулем, а не заранее
