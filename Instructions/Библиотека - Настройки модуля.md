# Настройки модуля

**Модуль:** [[Module - Библиотека]]

## Описание

Инструкция по основным настройкам модуля «Библиотека» в административной части портала. Настройки разделены на 4 вкладки: Общие настройки, Интеграция с МИФ/Альпина, Подписка, Доступ.

**URL настроек:** `адрес_портала/bitrix/admin/settings.php?lang=ru&mid=ithive.library`

## Ключевые функции

### Вкладка «Общие настройки»

- **Вкладки для переключения между сайтами** — настройка Библиотеки для каждого департамента отдельно
- **Публичная часть модуля доступна** — выбор групп/отделов/сотрудников, которые видят модуль
- **Группа модераторов предложенных книг** — группа, ответственная за модерацию (добавление, удаление, публикация)
- **ID Шаблона БП «Публикация»** — бизнес-процесс для автоматизации публикации (не рекомендуется менять)
- **ID шаблона БП «Модерация предложенной книги»** — бизнес-процесс модерации (не рекомендуется менять)
- **Скрыть возможность предложить книгу** — скрывает кнопку «Предложить» у всех пользователей
- **Скрыть выбор количества книг на странице** — отключает поле выбора количества книг
- **Количество книг в рекомендациях** — максимум книг в разделе «Рекомендуемые книги»
- **ИБ Библиотеки** — инфоблок для хранения контента (не рекомендуется менять)

### Вкладка «Интеграция с МИФ/Альпина»

- Выбор одного издательства (МИФ или Альпина) для интеграции
- Подробная настройка в отдельной инструкции

### Вкладка «Подписка»

- **Скрыть возможность подписаться** — скрывает кнопку «Подписка» у всех пользователей
- **Время запуска (по понедельникам)** — время отправки уведомлений о новинках
- **Последняя успешная рассылка** — дата и время последней рассылки
- **Количество подписчиков** — число пользователей с активной подпиской

### Вкладка «Доступ»

- Настройка доступа к административной части модуля для других групп
- По умолчанию доступ закрыт для всех, кроме администраторов

## Предусловия

- Права администратора портала

## Граничные случаи

- При наличии нескольких сайтов (департаментов) настройки можно конфигурировать индивидуально для каждого
- Можно подключить интеграцию только с одним издательством (МИФ или Альпина), но не с обоими одновременно
- Не рекомендуется самостоятельно менять ID бизнес-процессов и настройки инфоблоков
