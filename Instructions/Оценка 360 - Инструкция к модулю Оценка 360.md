# Инструкция к модулю «Улей: Оценка 360»

В этом разделе вы найдете инструкцию к модулю «Оценка 360». Модуль автоматизирует оценку компетенций по методу «360°» и помогает HR получать аналитику о сильных и слабых сторонах сотрудников.

**Технические данные**

- Модуль совместим с PHP версией 8.1 и выше

- Модуль совместим с СУБД: MySQL, PostgreSQL

- Модуль поддерживает английский язык

- Модуль поддерживает многодепартаментность

- Функционал модуля доступен через современный десктопный браузер и мобильное приложение Битрикс24.

- Документация по API модуля доступна [здесь](https://market.dev.wehive.digital/doxygen/assessment360/html/annotated.html)
