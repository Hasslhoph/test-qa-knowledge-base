# Кадровые изменения

**Модуль:** [[Module - Главная страница]]

## Описание

Стандартный раздел Битрикс24, отображающий кадровые изменения в организации: прием, перевод и увольнение сотрудников. Данные могут поступать из 1С:ЗУП или создаваться вручную.

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- **3 типа изменений:** прием, перевод, увольнение
- **2 шаблона:** sparkle (по умолчанию), .default
- **Выбор типов для отображения** на Главной (одиночный или множественный)
- **Настройка количества записей** на странице
- **Автоматическая загрузка** из 1С:ЗУП или других учетных систем

## Предусловия

- Права администратора для настройки блока
- Включённая административная панель (режим правки)
- Стандартный функционал Битрикс24 для кадровых операций

## Шаги

### Настройка блока на Главной

1. В режиме правки наведите на блок → шестерёнка (⚙️)
2. Выберите шаблон компонента
3. В «Основные параметры»:
   - Выводимые свойства: выберите типы (прием/перевод/увольнение)
   - Кол-во новостей на странице: максимальное количество записей
4. Сохраните

### Добавление записей

Записи появляются автоматически при кадровых операциях в Битрикс24:
- Прием сотрудника
- Перевод в другой отдел
- Увольнение

Подробнее о добавлении: [Кадровые изменения Битрикс24](https://helpdesk.bitrix24.ru/open/8509587/)

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- Это стандартный раздел Битрикс24 — данные поступают из HR-операций портала
- Можно отключить отдельные вкладки (например, «Увольнение») чтобы не демотивировать сотрудников
- Данные могут автоматически поступать из 1С:ЗУП
