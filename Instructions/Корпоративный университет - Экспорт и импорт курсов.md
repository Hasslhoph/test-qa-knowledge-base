# Экспорт и импорт курсов

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Импорт и экспорт курсов для переноса между порталами и создания резервных копий.

## Ключевые функции

### Процесс экспорта

1. Страница курса → «Настройки» (⚙️) → «Экспортировать курс»
2. Информация о курсе: Название, Символьный код, Контент Сайты24, Главы, Уроки, Тесты, Вопросы
3. «Экспортировать» → файл с архивом

### Процесс импорта

1. Главная страница «Онлайн-курсы» → «Настройки» (⚙️) → «Импортировать курс»
2. Выбор файла → информация о курсе
3. Редактирование названия и символьного кода
4. «Импортировать» → главная страница курса

### Импорт SCORM-курсов

- Поддержка SCORM-формата из стороннего ПО
- Галочка «SCORM» активируется автоматически
- Прогресс в SCORM-курсах не учитывается в статистике
- Кнопка для открытия курса на весь экран

### Нюансы

- Экспорт/импорт через один интерфейс (административная или публичная часть)
- Курсы Сайты24 — только через публичную часть
- При больших файлах могут потребоваться настройки сервера:
  - upload_max_filesize, post_max_size, memory_limit, max_input_time, client_max_body_size

## Предусловия

- Права по умолчанию: администраторы, HR-отдел

## Граничные случаи

- SCORM-курсы могут скачиваться вместо воспроизведения (решение: новая папка или настройка Nginx)
- Битрикс24 не полностью поддерживает формат SCORM
