# Видеоальбом

**Модуль:** [[Module - Главная страница]]

## Описание

Блок для загрузки и просмотра видеозаписей корпоративных событий. Поддерживает загрузку файлов и ссылки с YouTube и Rutube.

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- **Загрузка видеофайлов:** mp4, mpg, avi, wmv, mpeg, mpe, flv
- **Ссылки на видео:** YouTube, Rutube
- **2 шаблона на Главной:** sparkle_slider (по умолчанию), .default
- **Создание альбомов** — тематические группы видеозаписей
- **Сортировка** видеозаписей
- **Расширение форматов загрузки** через настройки инфоблока
- **Управление альбомами и видео** через административную часть

## Предусловия

- Права администратора или пользователя с правами на редактирование видеоархива
- Включённая административная панель (режим правки)

## Шаги

### Создание альбома

1. Перейдите на страницу «Видеогалерея» → кнопка «Элементы»
2. Нажмите «Добавить раздел»
3. Заполните: Раздел активен, Название
4. Сохраните

### Загрузка видео

1. Перейдите на страницу «Видеогалерея» → «Добавить элемент»
2. Вкладка «Параметры»:
   - Активность, Название (обязательное), Сортировка
   - URL (ссылка YouTube/Rutube) или Видео (файл)
   - Отображать на главной странице (да/нет)
3. Вкладка «Разделы»: выберите альбом
4. Сохраните

### Расширение форматов загрузки

1. Страница «Видеогалерея» → «Элементы» → «Настройки информационного блока»
2. Вкладка «Свойства» → поле «Видео» → кнопка (...)
3. В «Типы загружаемых файлов» добавьте форматы через запятую
4. Сохраните

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- Для раздела не предусмотрен пункт меню в настройках модуля
- Видео отображаются только из альбома, выбранного в настройках модуля
- При удалении альбома удаляются все вложенные видеозаписи
- Для новых установок модуля инфоблок видео: «Контент» > «Сервисы» > «Видео архив»
- Для старых установок: «Контент» > «Фотогалерея» > «Видео архив»
