# Добавление характеристик рабочих мест

**Модуль:** [[Module - Карта офиса + Бронирование]]

## Описание

Инструкция по созданию и управлению справочником характеристик (оснащения) рабочих мест: принтеры, телефоны, ПК, интерактивные доски и другое оборудование.

## Ключевые функции

- Создание новых характеристик рабочего места
- Положительный и отрицательный тип характеристик (цвет иконки)
- Иконки из библиотеки Font Awesome или загрузка файла
- Возможность блокировки бронирования при наличии определённой характеристики
- Сортировка характеристик
- Редактирование, копирование, удаление характеристик

## Предусловия

- Права на добавление и редактирование есть только у администраторов портала

## Создание новой характеристики

1. Перейти: `адрес_портала/bitrix/admin/index.php?lang=ru`
2. Контент → Highload-блоки → Библиотека иконок бронирования рабочих мест
3. Нажать «Добавить запись»

### Поля
1. **Тип** — положительный (тёмно-серая иконка) или отрицательный (красная иконка, для неисправностей)
2. **Название** — название характеристики (Монитор, Веб-камера и т.д.)
3. **Сортировка** — порядок отображения (меньше = выше, нет значения = первый)
4. **Ссылка** — код иконки из Font Awesome (версия 5.15.4, только бесплатные)
5. **Описание** — внутреннее описание (инвентарные номера, не выводится в публичной части)
6. **Возможность брони** — если снято, место с такой характеристикой нельзя забронировать
7. **Изображение** — загрузка файла иконки (JPG, PNG, SVG) вместо поля «Ссылка»

## Редактирование, копирование, удаление

Нажать на бургер-меню рядом с характеристикой:
- **Изменить** — страница редактирования
- **Копировать** — создание дубликата
- **Удалить** — безвозвратное удаление

## Граничные случаи

- При снятии чекбокса «Возможность брони» рабочее место нельзя забронировать, но уже созданные бронирования не отменяются
- Отрицательные характеристики (красные) используются для указания неисправностей
- Иконки с пометкой PRO в Font Awesome не поддерживаются
- При удалении характеристика пропадает из описания всех рабочих мест
