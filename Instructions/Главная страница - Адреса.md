# Адреса

**Модуль:** [[Module - Главная страница]]

## Описание

Последний блок на странице «О компании» для размещения контактной информации: адреса офисов, телефоны, ссылки на сайт и соцсети.

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- **Несколько блоков с адресами** (офисы в разных городах)
- **Контактная информация:** адрес, телефон, ссылка на сайт
- **Кликабельный заголовок** и название сайта
- **Сортировка** блоков
- **Создание новых блоков** с адресами
- **Переименование и изменение очередности** через [[Главная страница - Управление блоками]]

## Предусловия

- Права администратора или пользователя с правами на редактирование инфоблока
- Включённая административная панель (режим правки)

## Шаги

### Создание блока с адресом

1. `/bitrix/admin/` → «Контент» > «О компании» > «Адреса» > «Моя компания»
2. Нажмите «Добавить элемент»
3. Вкладка «Параметры»:
   - Название (заголовок, например «Офис в Санкт-Петербурге»)
   - Сортировка
   - Название ссылки на сайт (текст)
   - Ссылка для названия (# = кликабельно, пусто = не кликабельно)
   - Ссылка на сайт (URL)
   - Телефон
4. Вкладка «Анонс»:
   - Картинка для анонса (240x160px)
   - Описание для анонса (адрес офиса)
5. Сохраните

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- Блок можно убрать полностью через управление блоками
- Если стереть «#» в поле «Ссылка для названия», заголовок и название сайта перестанут быть кликабельными
