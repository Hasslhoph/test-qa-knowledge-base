# Создание и редактирование опроса

**Модуль:** [[Module - Опросы]]

## Описание

Инструкция по созданию, редактированию и удалению опросов. Доступно только администраторам и редакторам опросов.

## Ключевые функции

### Добавление опроса

1. Перейти на главную страницу модуля → кнопка «Настройки» (⚙️) → «Добавить опрос»
2. Заполнить слайдер создания опроса:

| Параметр | Описание |
|---|---|
| Название | Обязательно для заполнения |
| Описание | Цели опроса, информация для респондентов |
| Картинка | Обложка для опроса |
| Активность | Влияет на видимость в общем списке |
| Обязательный опрос | Для пульс-опросов — всплывающее окно на каждой странице |
| Дата начала / окончания | Срок сбора голосов |
| Группа | Категория опросов (обязательно) |
| Оповещать автора | Уведомления о новых голосах (мессенджер / e-mail) |
| Кол-во вопросов на странице | Постраничная навигация по вопросам |
| Разрешить менять мнение | Кнопка «Переголосовать» для респондентов |
| Сделать опрос анонимным | Скрыть результаты от обычных пользователей |
| Права доступа | Настройка по категориям: группы пользователей, пользователи, сотрудники и отделы, группы соцсети, другие роли |

### Права доступа к опросу

- **Закрыт** — опрос не отображается в общем списке
- **Смотреть результаты** — только просмотр результатов без голосования
- **Принять участие** — доступ к голосованию

### Дополнительные функции (при наличии хотя бы одного вопроса)

- **Предпросмотр** — как будет выглядеть опрос после публикации
- **Результаты** — смотреть диаграмму, смотреть таблицу, обнулить, Excel с группировкой / без группировки
- **Копировать** — копия опроса с идентичными настройками и вопросами
- **Удалить** — удаление опроса (все вопросы и ответы будут стёрты)

### Описание для респондентов и резюме

- **Описание для респондентов** — дополнительная информация на странице голосования (текст, изображения, файлы через визуальный редактор Битрикс24)
- **Описание для резюме** — вывод после прохождения опроса на странице результатов (текст + файлы, настройка доступа к резюме)

### Причины, по которым опрос не отображается

1. Отключена галочка «Активность»
2. Дата начала опроса ещё не наступила
3. Группа опроса скрыта в настройках модуля
4. Включена опция «Разрешить только один текущий опрос в группе»

## Предусловия

- Пользователь — администратор или редактор опросов
- Модуль установлен и активен

## Шаги

1. Перейти на главную страницу модуля «Опросы»
2. Нажать «Настройки» (⚙️) → «Добавить опрос»
3. Заполнить обязательные поля: Название, Группа
4. Настроить дополнительные параметры по необходимости
5. Сохранить опрос
6. Добавить вопросы через [[Опросы - Добавление вопросов и ответов]]

## Ожидаемый результат

Новый опрос появляется в общем списке и доступен для голосования целевой аудитории.

## Граничные случаи

- При удалении опроса все его вопросы и варианты ответов также стираются
- Переголосования не учитываются как новые голоса — голос пользователя учитывается только один раз
- В анонимных опросах обычные пользователи не видят количество голосов и страницу результатов
- Привилегированные пользователи видят результаты анонимных опросов в обезличенном виде
