# Оценка 360 - Сценарии оценки компетенций

**Модуль:** [[Module - Оценка 360]]

## Описание

Специфические сценарии подсчёта оценок: отрицательный вес индикатора, неотценённый индикатор, отрицательный балл при множественном выборе.

## Ключевые функции

### Оценка по шкале с отрицательным весом
- Индикатору задаётся вес со знаком минус
- Абсолютная сумма весов = сумма без учёта знака (|-2| + 3 + 5 = 10)
- Формула: (балл × вес) sum / (|вес| sum)
- Отрицательный вес уменьшает итоговую оценку

### Оценка по шкале с неотценённым индикатором
- Индикаторы, которые респондент не оценил, не учитываются
- Сумма весов считается только по оценённым индикаторам
- Пример: если оценены индикаторы А и Б, но не В — сумма весов = вес А + вес Б

### Оценка по баллам (множественный выбор) с отрицательным баллом
- Индикатору задан отрицательный балл (например, -2)
- Оценка от респондента = сумма баллов отмеченных индикаторов
- Отрицательный балл по компетенции = 0 (сотрудник не владеет компетенцией)
- Итоговая оценка = сумма баллов индикаторов, отмеченных ≥ 50% респондентов
- Процент респондентов для учёта индикатора настраивается в модуле (по умолчанию 50%)

## Предусловия

- Понимание базовых способов оценки (см. [[Оценка 360 - Способы оценки компетенций]])
- Настроены индикаторы с весами/баллами

## Граничные случаи

- Все индикаторы компетенции не могут быть отрицательными
- Сумма весов положительных индикаторов всегда должна быть больше суммы отрицательных
- Негативных индикаторов должно быть в половину меньше, чем положительных
- Отрицательная шкала предназначена только для индикаторной оценки
