# Обзор разделов модуля

**Модуль:** [[Module - Управление целями. Каскад]]

## Описание

Модуль «УЦ Каскад» состоит из восьми разделов. Количество отображаемых разделов зависит от прав доступа пользователя.

## Ключевые функции

> 8 разделов модуля с различными функциями.

- **Каталог целей** — список всех целей компании, начало работы с модулем
- **Процессы** — создание и настройка процессов постановки целей (справочник)
- **Контроль процессов** — запуск и остановка процессов для подразделений
- **Цели** — заполнение и согласование карточек целей (6 статусов)
- **Индивидуальные цели** — просмотр персональных целей сотрудников (только администраторы)
- **Оценки** — фиксирование прогресса в достижении целей (6 статусов)
- **Сводная** — отчет о выполнении целей по подразделениям
- **Настройки** — конфигурация модуля (только администраторы)

## Разделы модуля

### Каталог целей

- Список всех целей компании
- Добавление целей доступно только администраторам
- Сотрудники внутреннего контроля могут видеть каталог

### Процессы

- Создание и настройка процессов (квартал, полгода, год)
- Периодичность отчетов: неделя, 2 недели, месяц, квартал, полугодие
- Видна только администраторам модуля
- Запуск процессов выполняется в разделе «Контроль процессов»

### Контроль процессов

- Запуск и остановка процессов для подразделений
- Список всех подразделений с данными о процессах
- Администраторы видят все подразделения, руководители — только свои
- Запускать процессы могут только администраторы

### Цели

- Карточки целей создаются автоматически после запуска процесса
- 6 статусов: Составление → Согласует руководитель → Согласует функциональный руководитель → Согласует директор → Утверждает администратор → Утверждено
- Разная видимость в зависимости от роли

### Индивидуальные цели

- Автоматическое сохранение индивидуальных целей при согласовании
- Доступ только у администраторов модуля

### Оценки

- Карточки оценки создаются автоматически после утверждения карточки целей
- 6 статусов (аналогично карточкам целей)
- Периодичность зависит от настроек процесса

### Сводная

- Отчет о выполнении целей
- Средний прогресс по корпоративным и индивидуальным целям
- Разная видимость: администраторы и директор — все подразделения, руководители — только свои, сотрудники — нет доступа

### Настройки

- Настройка модуля «Управление целями. Каскад»
- Доступ только у администраторов модуля

## Список целей в профиле сотрудника

В профиле каждый сотрудник видит блок с утверждёнными целями. Также видеть этот блок могут:
- Руководители
- Функциональные руководители
- Администраторы
- Сотрудники внутреннего контроля

## Предусловия

- Модуль установлен и настроен
- Пользователь имеет соответствующие права доступа

## Граничные случаи

- Раздел «Цели» будет пустым для сотрудника, если в его подразделении не запущен процесс
- Раздел «Оценки» может быть пустым если карточка целей ещё не получила статус «Утверждено»
