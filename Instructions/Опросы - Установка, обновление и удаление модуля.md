# Установка, обновление и удаление модуля

**Модуль:** [[Module - Опросы]]

## Описание

Инструкция по установке, обновлению, активации купона и удалению модуля «Опросы» (ithive.polls).

## Ключевые функции

### Установка модуля

1. Перейти по ссылке: `адрес_портала/bitrix/admin/update_system_partner.php?addmodule=ithive.polls`
2. Модуль автоматически отмечен галочкой → нажать «Загрузить»
3. Подтвердить согласие с Лицензионным соглашением и Условиями конфиденциальности → «Применить»
4. Дождаться завершения установки
5. Уведомление об успешной загрузке подтверждает завершение

### Обновление модуля

1. Marketplace → Обновление решений → Список обновлений
2. Выбрать решение «Улей: Опросы»
3. Нажать «Загрузить»

Или сразу перейти: `адрес_портала/bitrix/admin/update_system_partner.php`

### Активация купона на покупку/продление

1. Marketplace → Обновление решений → Активация купона
2. Ввести номер купона → «Активировать купон»

Или: `адрес_портала/bitrix/admin/update_system.php?lang=ru`

### Активация купона на неустановленный модуль

1. Marketplace → Установленные решения → Купленные решения
2. Бургер-меню рядом с модулем → «Загрузить»

### Активация купона для модуля с истёкшей демо-версией

1. Обновление решений → Установка обновлений → «Зарегистрировать решение»

### Удаление модуля

1. Marketplace → Установленные решения → Доступные решения
2. Бургер-меню рядом с решением → «Удалить»
3. Подтвердить удаление после обновления страницы

## Предусловия

- Права на установку/обновление/удаление/активацию купона есть только у сотрудников группы «Администраторы»

## Ожидаемый результат

Модуль установлен, обновлён или удалён в зависимости от выполненной операции.
