# Группы опросов

**Модуль:** [[Module - Опросы]]

## Описание

Группировка опросов по тематикам или задачам (например, «Опросы для новых сотрудников», «Опросы о корпоративной жизни»). Работать с группами могут администраторы портала и редакторы опросов.

## Ключевые функции

- Создание и редактирование групп опросов через административную часть портала
- Символьный идентификатор — уникальное имя группы на латинице
- Группы используются для:
  - Категоризации опросов на главной странице модуля
  - Настройки отображаемых групп в настройках модуля
  - Привязки пульс-опросов
  - Кастомизации названий кнопок «Голосовать» / «Переголосовать» для каждой группы

## Предусловия

- Пользователь — администратор портала

## Шаги

1. Перейти в административную часть портала: `адрес_портала/bitrix/admin/`
2. В левом меню: «Сервисы» → «Опросы» → «Дополнительно» → «Группы опросов»
3. Нажать кнопку «Создать»
4. Указать название группы и символьный идентификатор (уникальное имя на латинице)
5. Сохранить группу
6. Выбрать группу при создании/редактировании опроса

## Ожидаемый результат

Группа создана и доступна для выбора при создании опросов.
