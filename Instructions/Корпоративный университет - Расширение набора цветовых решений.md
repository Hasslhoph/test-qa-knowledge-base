# Расширение набора цветовых решений для компетенций

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Настройка и расширение цветовых схем для иконок компетенций.

## Ключевые функции

### Как настроить цвет компетенции

- Публичная часть → карточка редактирования компетенции
- По умолчанию 20 предустановленных цветов

### Как расширить набор цветов

1. Административная часть → «Контент» → «Корпоративный университет» → «Компетенции»
2. Настройки информационного блока → вкладка «Свойства»
3. Три точки (...) у свойства «Цвет иконки» → настройки
4. Добавить новые значения:
   - **XML_ID** — цвет в HEX формате или градиент (два кода через дефис)
   - **Значение** — наименование цвета
   - **Сортировка** — порядок отображения
