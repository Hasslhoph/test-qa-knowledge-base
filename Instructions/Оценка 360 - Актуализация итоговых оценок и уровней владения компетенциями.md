# Оценка 360 - Актуализация итоговых оценок и уровней владения компетенциями

**Модуль:** [[Module - Оценка 360]]

## Описание

Возможность актуализировать итоговые оценки и уровни владения в уже завершённых оценках. Полезно при изменении логики определения итогов (например, переход с «Среднее без самооценки» на «Среднее»).

## Ключевые функции

### Зачем актуализировать
- В новых оценках итоги определяются по новой логике
- В завершённых оценках итоги остаются прежними
- Модуль позволяет обновить оценки и уровни в завершённых карточках

### Процесс актуализации
1. Раздел «Оценки» → отметить карточки со статусом «Оценка завершена»
2. Массовые действия → «Актуализировать средние значения и владение компетенциями» → «Применить»
3. Слайдер показывает сколько карточек будет актуализировано
4. Выбор колонки: «Среднее» или «Среднее без самооценки»
5. Опция актуализации владения компетенциями (необязательная):
   - Включена → уровни владения обновятся с учётом новых оценок
   - Отключена → обновятся только оценки, уровни останутся прежними
6. «Применить»

### Результат
- Оценки/уровни обновляются в отчётах
- Если оценка крайняя для сотрудника — обновляются в его профиле
- Актуализировать можно неограниченное количество раз

## Предусловия

- Права администратора или пользователя с правами на запуск оценки
- Карточки в статусе «Оценка завершена»

## Граничные случаи

- Не актуализируются карточки в статусах «На согласовании», «Сбор оценок», «Обработка оценок»
- При отключённой опции владения уровни не изменятся даже если оценки станут ниже/выше
