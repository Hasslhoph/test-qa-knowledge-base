# Интерфейс Зефир

**Модуль:** [[Module - Брендирование Битрикс24]]

## Описание

Статья описывает интерфейс модуля «Брендирование» для обновлённой версии Битрикс24 — «Зефир» (после обновления intranet 25.900.0). Модуль состоит из девяти разделов и работает только со светлыми темами портала.

## Ключевые функции

- **Настройки** — включение/выключение модуля, режим только для администратора, сброс настроек, экспорт/импорт ZIP
- **Общие** — цвет фона страницы (HEX), картинка фона (JPEG/PNG), скрытие виджетов, загрузка фавикона
- **Авторизация** — три блока: общие настройки (логотип, фон, цвет), форма авторизации, правый блок с QR-кодом
- **Виджеты** — 8 виджетов Ленты с индивидуальными настройками. **Общие настройки находятся в доработке** — стилизация доступна только через индивидуальные настройки или CSS
- **Шапка портала** — верхняя панель, левое меню, ссылки в верхнем меню, правый блок
- **Кнопки** — ссылки, зелёные кнопки, белые/прозрачные кнопки, синие кнопки (все состояния)
- **Контент** — фон важных сообщений, цвета формы с выбором значений
- **Другое** — цвет выделенной папки диска, цвет шапки виджета в задачах
- **Дополнительно** — произвольные CSS-стили (примеры: цвет логотипа Б24, скрытие логотипа, цвет иконки меню и др.)

## Предусловия

- Пользователь должен иметь доступ к модулю (администратор или группа с правами)
- У пользователя должна быть включена светлая тема портала (например «Светло-серый узор», «Цветная бумага», «Роса»)

## Граничные случаи

- На тёмных темах стили Брендирования не выводятся
- Исключение: настройки «Авторизация» и загружаемые элементы (фавикон, логотип) отображаются у всех
- Общие настройки виджетов не применяются (интерфейс в доработке) — использовать индивидуальные настройки или CSS
- CSS-стили из раздела «Дополнительно» не сбрасываются кнопкой «Сбросить все настройки»
- При импорте настроек из ZIP применяются все сохранённые состояния, включая «Выключить работу модуля»
