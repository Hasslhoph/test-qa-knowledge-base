# Установка, обновление и удаление модуля

**Модуль:** [[Module - Управление целями. Каскад]]

## Описание

Инструкция по установке, обновлению, активации купонов и удалению модуля «Управление целями. Каскад» через административную часть Битрикс24.

## Ключевые функции

> Полный жизненный цикл модуля.

- Установка модуля через список обновлений
- Обновление через Marketplace
- Активация купона на покупку/продление
- Активация купона на неустановленный модуль
- Активация купона после истечения демо-версии
- Удаление модуля

## Установка модуля

1. Перейти: `адрес_портала/bitrix/admin/update_system_partner.php?addmodule=ithive.goalsmanagement`
2. Модуль автоматически отмечен галочкой → нажать «Загрузить»
3. Подтвердить Лицензионное соглашение → «Применить»
4. Дождаться завершения установки
5. Уведомление об успешной загрузке + загрузка всех обновлений

**Права:** только администраторы портала.

## Обновление модуля

1. «Marketplace» → «Обновление решений» → «Список обновлений»
2. Выбрать «Улей: Управление целями. Каскад»
3. Нажать «Загрузить»
4. Уведомление об успешном обновлении

Или сразу: `адрес_портала/bitrix/admin/update_system_partner.php`

**Права:** только администраторы портала.

## Активация купона на покупку/продление

1. «Marketplace» → «Обновление решений» → «Активация купона»
2. Ввести номер купона → «Активировать купон»

Или сразу: `адрес_портала/bitrix/admin/update_system.php?lang=ru`

## Активация купона на неустановленный модуль

1. «Marketplace» → «Установленные решения»
2. «Купленные решения» → бургер-меню → «Загрузить»

## Активация купона после истечения демо-версии

1. «Обновление решений» → «Установка обновлений»
2. Нажать «Зарегистрировать решение»

**Права:** только администраторы портала.

## Удаление модуля

1. «Marketplace» → «Установленные решения» → «Доступные решения»
2. Бургер-меню → «Удалить»
3. Подтвердить удаление
4. Уведомление об удалении

Или сразу: `адрес_портала/bitrix/admin/update_system_partner.php`

**Права:** только администраторы портала.

## Предусловия

- Роль администратора портала для всех операций

## Граничные случаи

- При установке автоматически загружаются все обновления
- Удаление модуля удаляет все связанные данные
