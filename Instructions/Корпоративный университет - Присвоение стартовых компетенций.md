# Присвоение стартовых компетенций

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Закрепление за сотрудниками их текущих компетенций до начала развития. Два способа: через раздел «Сотрудники» и через личный профиль сотрудника.

## Ключевые функции

### Индивидуальное присвоение (раздел «Сотрудники»)

1. Раздел «Сотрудники» → бургер-меню (≡) → «Присвоить компетенции»
2. Шаг 1: выбрать компетенции → «Далее»
3. Шаг 2: указать уровень владения:
   - Синяя галочка — достаточный уровень
   - Академическая шапочка — экспертный уровень
   - Обе серые — убрать компетенцию

### Массовое присвоение

1. Отметить сотрудников галочками → «Присвоить компетенции» → «Применить»
2. Выбрать компетенции → следующий шаг
3. Указать уровни владения для каждого сотрудника → сохранить

### Присвоение через профиль сотрудника

1. Открыть профиль сотрудника → блок «Имеющиеся компетенции» → карандаш (✏️)
2. Три статуса:
   - **Нет проявления** — серая галочка
   - **Проявляет в достаточной мере** — синяя галочка
   - **Является экспертом** — академическая шапочка

### Изменение уровней владения

- Повторный клик на карандаш (✏️) → указать другой статус → сохранить
- При успешном завершении программы развития компетенции добавляются автоматически

## Предусловия

- Права: администраторы и группа HR-отдела

## Граничные случаи

- Даже если компетенция находится в развитии, её можно присвоить сотруднику
- Владение компетенциями на текущий момент бессрочное
- Экспертный уровень нельзя получить через программу развития — только вручную
