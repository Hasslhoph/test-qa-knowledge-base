# Типы программ в Корпоративном университете

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Для обучения используются программы развития — список компетенций, которые должен освоить сотрудник.

## Ключевые функции

### Программа развития

- Набор компетенций (soft и hard skills, корпоративные)
- Руководитель, чёткие сроки, список развивающих действий
- Сроки по умолчанию: от 6 месяцев до 2 лет

### Для чего используются программы развития

- Развитие для перехода на вышестоящую/смежную позицию
- Развитие в рамках текущей позиции
- Изучение отдельных компетенций (кадровый резерв)
- Саморазвитие (курсы, книги, мероприятия)

### Варианты запуска

1. Запуск по шаблону
2. Запуск на основании карьерной траектории/текущей позиции
3. Свободный выбор компетенций

### Программа адаптации

- Та же программа развития, но для онбординга новых сотрудников
- Сроки по умолчанию: от 1 до 3 месяцев
- Единственный вариант запуска — по шаблону
- Опция «Использовать программы адаптации» в настройках модуля

### Автоматический запуск программ адаптации

- При создании шаблона включить опцию автоматического запуска
- При добавлении новых сотрудников автоматически запускается программа
- Для разных подразделений — свои шаблоны

## Предусловия

- Настроенные компетенции и развивающие действия
- Созданные шаблоны программ (для запуска по шаблону)
