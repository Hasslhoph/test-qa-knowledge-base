# Вопросы для тестов

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Создание и управление вопросами для тестов в онлайн-курсах.

## Ключевые функции

### Типы вопросов

1. **Одиночный выбор** — один правильный ответ
2. **Множественный выбор** — один или несколько правильных ответов
3. **Сортировка** — расставить ответы в правильном порядке
4. **Текстовый ответ** — ответ в свободной форме
5. **Ответ картинкой** — варианты ответов в виде изображений
6. **Сопоставление** — поиск соответствий между столбцами

### Стандартные поля вопроса

- **Тип вопроса**
- **Выводится в** — привязка к курсу/главе/уроку
- **Название** (обязательное)
- **Описание**
- **Картинка** (рекомендуемая ширина ≤ 1020px)
- **Активность** — отображение в тестах
- **Обязателен правильный ответ** — неправильный ответ = тест не пройден
- **Заметка** — для внутреннего использования
- **Сортировка** — порядок вопросов
- **Баллы** — по умолчанию 10

### Список вопросов

- «Настройки» (⚙️) → «Вопросы курса» → «Список вопросов»
- Действия: Редактировать, Деактивировать/Активировать, Удалить
- Массовые действия

## Предусловия

- Права: администраторы, HR-отдел, пользователи с правами на курс

## Граничные случаи

- Текстовые вопросы невозможно проверить автоматически
- Вопрос может быть привязан только к одному элементу
- При опции «Случайный порядок вопросов» сортировка не применяется
