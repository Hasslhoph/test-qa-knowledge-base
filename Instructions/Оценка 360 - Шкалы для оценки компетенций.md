# Оценка 360 - Шкалы для оценки компетенций

**Модуль:** [[Module - Оценка 360]]

## Описание

Шкала — ключевой элемент модуля, влияющий на оценку компетенций, присвоение уровней владения, итоговые оценки. По умолчанию используется десятибалльная шкала (положительная и отрицательная).

## Ключевые функции

### Где найти настройки шкалы
- Административная часть: адрес_портала`/bitrix/admin`
- «Настройки» → «Настройки продукта» → «Настройки модулей» → «Улей: Оценка 360» → «Настройка шкалы оценок»

### Положительная шкала
- Максимальный размер — 10 баллов, минимальный — 3 балла (без учёта нуля)
- Можно сократить до 9, 8, 7, 6, 5, 4 баллов
- Размер шкалы влияет на баллы для индикаторов при балльной оценке

### Уровни владения компетенцией
1. **Не владеет** (серая галочка) — нулевая/низкая оценка
2. **Владеет на достаточном уровне** (синяя галочка) — средняя/хорошая оценка
3. **Эксперт** (академическая шапочка) — высокая оценка, может быть наставником

### Привязка баллов к уровням (10-балльная шкала)
- 1-4 балла — не владеет компетенцией
- 5-8 баллов — владеет на достаточном уровне
- 9-10 баллов — эксперт

### Отрицательная шкала
- Для оценки негативных проявлений компетенций
- Всегда дублирует положительную по количеству делений
- Активируется через отрицательный вес индикатора
- Респонденты ставят баллы со знаком «плюс», отрицательное значение появляется при умножении на вес

## Предусловия

- Права администратора для изменения настроек шкалы

## Граничные случаи

- Отрицательная шкала только для индикаторной оценки
- Все индикаторы не могут быть отрицательными
- Положительных индикаторов должно быть больше негативных
- По отрицательной шкале не ставятся отрицательные баллы (баллы всегда положительные, вес отрицательный)
- Чем меньше делений на шкале, тем менее удобна для респондента
