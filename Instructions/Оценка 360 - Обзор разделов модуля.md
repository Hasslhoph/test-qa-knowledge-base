# Оценка 360 - Обзор разделов модуля

**Модуль:** [[Module - Оценка 360]]

# Обзор разделов модуля

Оценка 360 включает в себя пять разделов: «Оценки», «Я оцениваю», «Компетенции», «Шаблоны», «О методике». В этой статье приведен краткий обзор каждого раздела и его основных функций.

Раздел «Шаблоны» отображается только у пользователей с правами доступа к запуску и настройке оценки. По умолчанию это администраторы и пользователи из группы «Отдел персонала».

![](_images-02/15.png)

## Раздел «Оценки»

Данный раздел является главной страницей модуля. В первую очередь он предназначен для хранения информации о завершенных оценках пользователя. Другими словами, здесь любой сотрудник может ознакомиться с результатами оценки своих компетенций. Для этого достаточно открыть нужную карточку в списке.

![](_images-02/image63.png)

Сотруднику доступна своя карточка только если ей присвоен статус «Оценка завершена». Незавершенные оценки по сотруднику будут скрыты от него даже при наличии максимального уровня доступа к модулю и порталу.

Обычные пользователи могут использовать раздел «Оценки» только для просмотра итогов по своим компетенциям. У привилегированных пользователей возможности шире: здесь они могут следить за сбором оценок, обрабатывать результаты опросов, просматривать итоги оценок других пользователей. Соответственно, им доступен расширенный список карточек.

## Как видят раздел привилегированные пользователи

1. **Администраторы и пользователи из группы «Отдел персонала».** В разделе «Оценки» им видны не только свои карточки с завершенной оценкой, но и карточки любых других сотрудников за все время и с любым статусом. Кроме того, этой категории пользователей доступна кнопка «Запустить оценку».

![](_images-02/image301.png)

Обратите внимание, что для доступа к этому интерфейсу администраторам и отделу персонала нужно предварительно настроить [соответствующие права.](../06-administrirovanie-modulya/01-nastroyki-modulya-otsenka-360.md)

2. **Руководители.** По умолчанию руководителям видны свои завершенные карточки, а также карточки подчиненных со статусами «На согласовании» и «Оценка завершена». Настраивать права доступа для этого не требуется.

![](_images-02/image395.png)

Всего у карточек существует четыре статуса:

1. **На согласовании.** Оценка еще не запущена, так как карточку должен согласовать руководитель оцениваемого сотрудника.

2. **Сбор оценок.** Оценка запущена, пользователи делятся обратной связью о компетенциях сотрудника.

3. **Обработка оценок.** Сроки сбора оценок завершились, респонденты больше не могут оставлять оценки.

4. **Оценка завершена.** HR-специалист завершил оценку данного сотрудника. Оцениваемому сотруднику и его руководителю отправлен отчет с результатами.

Администраторы и группа «Отдел персонала» могут открывать карточки с любым статусом, чтобы проверить, как идет процесс оценки: кто уже оценил сотрудника и каковы предварительные баллы по компетенциям.

## Раздел «Я оцениваю»

В этом разделе все пользователи видят список карточек, которые они должны оценить (включая свои собственные). Здесь у карточек может быть два статуса:

1. **Заполнена.** Означает, что пользователь уже поставил какие-либо оценки в карточке.

2. **Отказ.** Означает, что пользователь отказался оценивать сотрудника.

3. **Ожидание.** Означает, что пользователь еще не поставил оценки сотруднику.

Обратите внимание, что изменить ответы можно любой момент до окончания срока оценки.

![](_images-02/image139.png)

Карточка исчезает из списка, когда срок оценки по данному сотруднику завершается. То есть когда в разделе [«Оценки»](#razdel-otsenki) у карточки появляется статус «Обработка оценок».

Если у пользователя есть карточки, которые он еще не оценил, то в разделе «Я оцениваю» и в левом меню модуля будут гореть красные индикаторы с соответствующим количеством неоцененных карточек.

![](_images-02/image70.png)

## Раздел «Компетенции»

Данный раздел доступен всем пользователям. Здесь составляется и хранится система компетенций, принятая в компании. Именно по этим компетенциям выполняется оценка.

Если помимо Оценки 360 на портале установлен модуль «Корпоративный университет», то раздел «Компетенции» в Оценке 360 отображаться не будет. Раздел можно будет найти в Корпоративном университете — через него производится управление компетенциями, категориями и индикаторами в рамках обоих модулей.

![](_images-02/image25.png)

В этом разделе любой сотрудник может увидеть текущий уровень развития своих компетенций. Если по данному сотруднику есть хотя бы одна завершенная оценка, в разделе он увидит:

1. **Актуальные оценки по конкретным компетенциям.** Оценки по компетенциям, взятые из самой недавней завершенной карточки.

2. **Индикаторы освоения компетенций.** Иконки в виде серой галочки, синей галочки и академической шапочки (см. скриншот ниже). Автоматически присваиваются сотруднику при получении нужной оценки по компетенциям.

3. **Общие данные по завершенным оценкам.** Список завершенных оценок и количестве компетенций, которые в них входят. По клику на дату оценки сотрудник может перейти в отчет о своей [завершенной оценке](../05-provedenie-otsenki/05-rezultaty-otsenki-obzor-itogovogo-otcheta.md).

![](_images-02/image90.png)

При необходимости любой сотрудник может воспользоваться поиском с фильтрами, чтобы найти нужные ему компетенции.

![](_images-02/image51.png)

Здесь же привилегированные пользователи могут добавлять/удалить индикаторы и настраивать способы их оценки. Более подробно о работе с компетенциями написано в [отдельной статье](../04-podgotovka-k-otsenke/01-dobavlenie-kompetentsiy-i-kategoriy.md).

## Карточка компетенции

При нажатии на компетенцию откроется карточка с детальной информацией о данной компетенции. В самом верху карточки находится описание компетенции и данные о сотрудниках, которые ее освоили.

![Нажмите, чтобы отредактировать изображение](_images-02/image171.png)

Блок с информацией об освоении компетенции состоит из двух частей:

1. **Обладают данной компетенцией.** Числовой индикатор показывает, сколько сотрудников владеют данной компетенцией на достаточном уровне для выполнения своей работы.

![](_images-02/image419.png)

2. **Эксперты в данной компетенции.** Числовой индикатор показывает, сколько сотрудников владеют данной компетенцией на экспертном уровне. Этому уровню владения компетенцией соответствует иконка академической шапочки.

![](_images-02/image94.png)

В нижней части карточки находится информация об индикаторах данной компетенции, если они указаны. Здесь пользователь может увидеть свои актуальные итоговые оценки по каждому индикатору.

![](_images-02/image408.png)

## Раздел «Шаблоны»

Этот раздел доступен только пользователям с расширенными правами доступа к модулю. Он используется для создания готовых наборов компетенций, по которым будет проводиться оценка.

Например, HR-менеджер может создать шаблон «Оценка руководителей проектов» и добавить в него те компетенции, которыми должны владеть сотрудники в этой должности:

![](_images-02/image459.png)

В итоге, при запуске оценки для руководителей проектов, HR-менеджеру не придется вручную проставлять нужные компетенции в карточке. Достаточно будет выбрать шаблон, из которого они подтянутся автоматически.

![](_images-02/image443.png)

## Раздел «О методике»

Справочный раздел, где собраны ответы на частые вопросы о методике «360 градусов» и работе в модуле. С материалами раздела может ознакомиться любой пользователь вне зависимости от его уровня доступа к корпоративному порталу.

При необходимости текст в данном разделе можно изменить. О том, как это сделать написано в отдельной [статье](https://dev.1c-bitrix.ru/learning/course/index.php?COURSE_ID=43&LESSON_ID=2560). Обратите внимание, что для редактирования описания нужны знания в области HTML-верстки.
