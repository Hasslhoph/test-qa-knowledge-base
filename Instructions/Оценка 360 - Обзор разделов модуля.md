# Оценка 360 - Обзор разделов модуля

**Модуль:** [[Module - Оценка 360]]

## Описание

Оценка 360 включает пять разделов: «Оценки», «Я оцениваю», «Компетенции», «Шаблоны», «О методике». Раздел «Шаблоны» отображается только у пользователей с правами на запуск и настройку оценки.

## Ключевые функции

### Раздел «Оценки»
- Главная страница модуля
- Хранение информации о завершённых оценках пользователя
- Обычные пользователи видят только свои карточки со статусом «Оценка завершена»
- Привилегированные пользователи (администраторы, HR) видят все карточки с любыми статусами
- Руководители видят свои карточки и карточки подчинённых со статусами «На согласовании» и «Оценка завершена»

### Статусы карточек
1. **На согласовании** — оценка не запущена, ожидает согласования руководителем
2. **Сбор оценок** — оценка запущена, респонденты оставляют обратную связь
3. **Обработка оценок** — сроки сбора завершились, респонденты больше не могут оценивать
4. **Оценка завершена** — HR завершил оценку, отправлен отчёт с результатами

### Раздел «Я оцениваю»
- Список карточек, которые пользователь должен оценить
- Статусы: «Заполнена», «Отказ», «Ожидание»
- Ответы можно менять до окончания срока оценки
- Красные индикаторы показывают количество неоценённых карточек

### Раздел «Компетенции»
- Система компетенций, принятая в компании
- Поиск с фильтрами для нахождения нужных компетенций
- Отображение актуальных оценок, индикаторов освоения, данных по завершённым оценкам
- Если установлен «Корпоративный университет» — раздел скрыт (управление через него)

### Раздел «Шаблоны»
- Доступен только привилегированным пользователям
- Создание готовых наборов компетенций для оценки
- Автоматическое подтягивание компетенций из шаблона при запуске оценки

### Раздел «О методике»
- Справочный раздел с ответами на вопросы о методике «360 градусов»
- Доступен всем пользователям
- Текст можно изменить (требуются знания HTML)

## Предусловия

- Модуль установлен и настроен
- Пользователь имеет соответствующие права доступа

## Граничные случаи

- Раздел «Шаблоны» виден только администраторам и группе «Отдел персонала»
- Раздел «Компетенции» не отображается при наличии модуля «Корпоративный университет»
- Незавершённые оценки скрыты от обычных сотрудников даже при максимальном уровне доступа
