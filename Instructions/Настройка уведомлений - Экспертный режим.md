# Экспертный режим

**Модуль:** [[Module - Настройка уведомлений]]

## Описание

В разделе «Экспертный режим» можно включить или отключить данный режим для выбранных пользователей: конкретных сотрудников, подразделений и/или рабочих групп. Доступ к данным настройкам есть только у привилегированных пользователей.

По умолчанию в ленте Битрикс24 выводятся уведомления о задачах текущего пользователя. Когда экспертный режим включён, такие уведомления не видны в ленте.

## Ключевые функции

- **Включение/отключение экспертного режима** для выбранных пользователей, подразделений и рабочих групп
- **Массовая настройка** — можно применить настройки сразу для группы сотрудников или отдела
- **Индивидуальная настройка** — можно настроить экспертный режим для конкретного сотрудника
- **Ограничение доступа** — при необходимости доступ к настройкам экспертного режима можно ограничить в настройках модуля

## Предусловия

- Модуль установлен и включён
- Пользователь имеет права на настройку уведомлений
- В настройках модуля опция «Экспертный режим» включена (галочка стоит)
- Версия Битрикс24 — до intranet 25.900.0 (обновление «Зефир»)

## Шаги настройки

### Шаг 1. Перейдите в раздел «Экспертный режим»

Раздел доступен в публичной части модуля (левое меню Битрикс24 или настройки мессенджера → «Настроить для всех»).

### Шаг 2. Выберите пользователей

Выберите пользователей, для которых нужно включить или отключить экспертный режим. Можно выбрать:
- Отдельного пользователя
- Нескольких пользователей
- Подразделения
- Рабочие группы

### Шаг 3. Примените настройки

Установите или снимите экспертный режим для выбранных пользователей и примените настройки.

## Граничные случаи

- **Критично:** Экспертный режим доступен в Битрикс24 только до версии intranet 25.900.0, т.е. до обновления «Зефир». Начиная с этого обновления данный режим не используется и уведомления о задачах по умолчанию не выводятся в ленте. В обновлении «Зефир» включение/отключение экспертного режима не поддерживается
- Если в настройках модуля галочка «Экспертный режим» не стоит, доступ к настройке экспертного режима в публичной части отсутствует
- При включённом экспертном режиме уведомления о задачах пользователя не дублируются в ленте новостей
