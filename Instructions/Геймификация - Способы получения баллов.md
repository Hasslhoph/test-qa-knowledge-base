# Геймификация - Способы получения баллов

**Модуль:** [[Module - Геймификация]]

## Описание

Пять путей, которые позволяют сотрудникам естественным путём зарабатывать внутреннюю валюту Геймификации.

## Ключевые функции

- **Баллы за рейтинг** — за призовые места в автоматических (авто) и ручных (вручную) рейтингах; по умолчанию 3 призовых места, 300 баллов
- **Баллы за дни рождения и срок работы** — настраиваемое начисление за знаковые даты (каждый год, каждые 5 лет и т.д.)
- **Баллы за закрытые задачи** — настраиваемые параметры: модерация, задачи самому себе, повторное начисление, вовремя/просроченная задача
- **Баллы за активити в Бизнес-процессах** — начисление за выполнение нужной активити (выставление счёта, отправка документов)
- **Баллы «Спасибо»** — ежемесячно 50 баллов по умолчанию; можно перечислять другим через: Ленту событий, Доску почета, свой профиль, профиль другого сотрудника

## Предусловия

- Модуль установлен и настроен
- Настроены категории «Спасибо» (иначе сотрудники не смогут отправлять благодарности)

## Шаги (перечисление баллов «Спасибо»)

1. Откройте слайдер «Сказать спасибо» (из Ленты, Доски почета, профиля)
2. Выберите сотрудника-получателя
3. Выберите бейдж (категорию «Спасибо») — соответствует корпоративным ценностям
4. При необходимости отрегулируйте количество баллов ползунком (если разрешено)
5. Добавьте комментарий к благодарности
6. Нажмите «Сказать спасибо»

## Ожидаемый результат

Сотрудник получает баллы на счёт, в Ленте событий появляется сообщение о благодарности.

## Граничные случаи

- Если у сотрудника не хватает баллов «Спасибо» для бейджа — бейдж окрашен в серый цвет
- При попытке перечислить без достаточных баллов — появляется уведомление
- По умолчанию лимит 5-15 баллов за раз (настраивается для каждой категории)
- Баллы «Спасибо» нельзя потратить на себя — только передать другим
- Сообщения о начислениях за задачи и бизнес-процессы появляются только в «Операциях», не в «Ленте событий»
