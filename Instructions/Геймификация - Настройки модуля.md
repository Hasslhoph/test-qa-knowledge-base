# Геймификация - Настройки модуля

**Модуль:** [[Module - Геймификация]]

## Описание

Основные настройки модуля «Геймификация» в административной части портала. Модуль включает 10 вкладок настроек.

## Ключевые функции

### Вкладка «Общие настройки»
- Время пересчета рейтингов (по умолчанию 00:00)
- Настройки инфоблоков (не рекомендуется менять)
- Права доступа к публичной части модуля
- Ответственные за бюджет/списание баллов (обязательно)
- Ответственные за исполнение заказа (второй этап согласования, опционально)
- Сопровождающие исполнение заказа (наблюдатель, опционально)
- Модераторы магазина бонусов
- Таблица рейтингов: опубликовано, активность, автоначисление, сортировка, название, папка логики, коэффициент веса, описание, иконка, символьный код, подразделения

### Вкладка «Рейтинги»
- Выпадающий список всех рейтингов с индивидуальными настройками
- Количество призовых мест (1-30), сумма баллов (по умолчанию 300)
- Количество выводимых позиций (по умолчанию 10)
- Количество призеров предыдущих месяцев (по умолчанию 10)
- Настройки детальной информации
- Пересчет рейтинга (с опцией «не записывать в БД»)
- Индивидуальные веса параметров для каждого из 7 рейтингов с формулами расчёта

### Вкладка «Связи»
- Выбор рейтинга для отображения в личном кабинете «Корпоративного университета» ([[Module - Корпоративный университет]])

### Вкладка «Валюта»
- Название валюты (иконка — первая буква, если не загружена отдельная)
- Строка формата для вывода валюты (сокращённое название)
- Иконка валюты (PNG/JPG/SVG) — перебивает букву
- Склонение названия валюты

### Вкладка «Транзакции»
- Права доступа к начислению/списанию баллов
- Права на просмотр чужих транзакций
- По умолчанию — только группа «Администраторы»

### Вкладка «Настройка уровней»
- Названия уровней, иконки, баллы для достижения

### Вкладка «Спасибо»
- Ежемесячное начисление баллов «Спасибо» (вкл/выкл)
- Количество начисляемых баллов в месяц (по умолчанию 50)
- Уровни подразделений для оповещений
- Таблица категорий «Спасибо»: сортировка, название, связь с благодарностями, стоимость, изменение количества баллов, описание

## Предусловия

- Права администратора портала
- Доступ к административной части (`/bitrix/admin/`)

## Шаги

1. Перейдите в административную часть → Настройки → Улей: Геймификация
2. Выберите нужную вкладку
3. Внесите изменения
4. Сохраните

## Ожидаемый результат

Настройки модуля применены, изменения отражены в публичной части портала.

## Граничные случаи

- Настройки рейтингов едины для всех сайтов (департаментов)
- Права и ответственные настраиваются отдельно для каждого сайта
- Удалить можно только рейтинг без транзакций
- Если никакие категории «Спасибо» не созданы — сотрудники не могут отправлять благодарности
- Поле «Связь с благодарностями» является рудиментным (не используется)
