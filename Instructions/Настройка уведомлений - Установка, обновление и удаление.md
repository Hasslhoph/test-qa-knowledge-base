# Установка, обновление и удаление модуля Настройка уведомлений

**Модуль:** [[Module - Настройка уведомлений]]

## Описание

Инструкция по установке, обновлению, активации купона и удалению модуля «Настройка уведомлений для всех» (`ithive.imsettingsforall`) на портале Битрикс24.

## Ключевые функции

- Установка модуля через административную часть портала
- Обновление модуля через Marketplace
- Активация купона на покупку/продление
- Удаление модуля через Marketplace

## Предусловия

- Права группы пользователей «Администраторы» для всех операций
- Доступ к административной части портала Битрикс24

## Установка модуля

### Шаг 1. Перейдите по ссылке

Откройте: адрес_портала`/bitrix/admin/update_system_partner.php?addmodule=ithive.imsettingsforall`

Ссылка ведёт в административную часть портала на вкладку «Список обновлений».

### Шаг 2. Модуль будет автоматически отмечен

Модуль «Настройка уведомлений для всех» будет автоматически отмечен галочкой. Нажмите кнопку **«Загрузить»**. Откроется окно с Лицензионным соглашением.

### Шаг 3. Подтвердите соглашение

Подтвердите согласие с Лицензионным соглашением и Условиями конфиденциальности, затем нажмите кнопку **«Применить»**. Установка модуля начнётся автоматически.

### Шаг 4. Дождитесь завершения установки

### Шаг 5. Установка завершена

На экране появится уведомление об успешной загрузке. Также будут загружены все обновления.

## Обновление модуля

### Способ 1: Через Marketplace

1. В левом меню административной части выберите раздел **«Marketplace»**
2. Перейдите в пункт **«Обновление решений»**
3. Нажмите на вкладку **«Список обновлений»**
4. Выберите решение «Улей: Настройка уведомлений для всех»
5. Нажмите на кнопку **«Загрузить»**

### Способ 2: По прямой ссылке

Перейдите на вкладку «Список обновлений» по ссылке: адрес_портала`/bitrix/admin/update_system_partner.php`

При успешном обновлении появится соответствующее уведомление.

## Активация купона на покупку/продление

### Шаг 1-3. Перейдите к активации купона

1. В левом меню административной части выберите раздел **«Marketplace»**
2. Перейдите в пункт **«Обновление решений»**
3. Выберите вкладку **«Активация купона»**

Или перейдите по ссылке: адрес_портала`/bitrix/admin/update_system.php?lang=ru`

### Шаг 4. Введите купон

Введите номер купона и нажмите кнопку **«Активировать купон»**.

### Активация купона на неустановленный модуль

1. В разделе «Marketplace» выберите пункт **«Установленные решения»**
2. В списке «Купленные решения» нажмите на бургер-меню рядом с модулем, затем на кнопку **«Загрузить»**

### Активация купона для модуля с истёкшим сроком демоверсии

1. Перейдите в раздел **«Обновление решений»**
2. Во вкладке «Установка обновлений» нажмите **«Зарегистрировать решение»**

## Удаление модуля

### Шаг 1-2. Перейдите к установленным решениям

1. В левом меню административной части выберите раздел **«Marketplace»**
2. Перейдите в пункт **«Установленные решения»**

Или перейдите по ссылке: адрес_портала`/bitrix/admin/update_system_partner.php`

### Шаг 3-4. Удалите модуль

3. Перейдите в блок **«Доступные решения»**
4. Слева от выбранного решения откройте бургер-меню и нажмите кнопку **«Удалить»**

### Шаг 5-6. Подтвердите удаление

5. Подтвердите удаление модуля после обновления страницы
6. Страница будет ещё раз обновлена, и вы увидите сообщение, что модуль удалён

## Граничные случаи

- Права на установку, обновление, активацию купона и удаление модуля есть **только** у группы пользователей «Администраторы»
- При активации купона на неустановленный модуль необходимо сначала загрузить модуль из «Купленных решений»
- При истёкшем сроке демоверсии необходимо зарегистрировать решение во вкладке «Установка обновлений»
