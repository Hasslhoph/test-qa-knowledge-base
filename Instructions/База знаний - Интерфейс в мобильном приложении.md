# База знаний - Интерфейс в мобильном приложении

**Модуль:** [[Module - База знаний]]

## Описание

Отображение Базы знаний в стандартном мобильном приложении Битрикс24. Мобильное приложение не имеет функционала администрирования и наполнения справочников.

## Ключевые функции

- Просмотр списка разделов и статей
- Просмотр статей
- Создание публичных ссылок на статьи
- Комментирование статей
- Просмотр файлов в статье

## Предусловия

- Установленное мобильное приложение Битрикс24
- Права на чтение статей

## Шаги

### Просмотр разделов и статей

1. Откройте мобильное приложение Битрикс24
2. Перейдите в пункт меню «База знаний»
3. Просматривайте список разделов и статей

### Просмотр статьи

1. Кликните на нужную статью
2. Просматривайте содержимое статьи

### Создание публичной ссылки

1. Откройте статью
2. Используйте опцию «Поделиться» для создания публичной ссылки

### Комментарии

1. Откройте статью
2. Прокрутите вниз до блока комментариев
3. Оставьте комментарий

### Просмотр файлов

1. Откройте статью
2. Просматривайте прикреплённые файлы

## Ожидаемый результат

Пользователь может просматривать разделы, статьи, комментарии и файлы через мобильное приложение.

## Граничные случаи

- Мобильное приложение не имеет функционала администрирования
- Мобильное приложение не имеет функционала наполнения справочников (создание/редактирование статей)
- Уникальные просмотры через мобильное приложение засчитываются отдельно от десктопных
