# Закрепление места за сотрудником

**Модуль:** [[Module - Карта офиса + Бронирование]]

## Описание

Инструкция по закреплению рабочих мест за сотрудниками. Закреплённые места становятся недоступны для бронирования другими пользователями.

## Ключевые функции

- Закрепление рабочего места за одним сотрудником
- Открепление сотрудника от места
- Визуальное отличие закреплённых мест (серый цвет по умолчанию)
- Изменение цвета закреплённого места в настройках модуля

## Предусловия

- Права на закрепление есть у администраторов, модераторов карты офиса и офис-менеджеров
- Карта кабинета загружена и содержит рабочие места

## Как закрепить место

1. Открыть карту офиса
2. Навести курсор на нужное место
3. В блоке «Закреплено за» нажать на иконку карандаша
4. Выбрать сотрудника из списка

**Важно:** закрепить место можно одновременно только за одним сотрудником.

## Как открепить/перезакрепить

- **Перезакрепить:** нажать на иконку карандаша и выбрать другого сотрудника
- **Открепить:** нажать на крестик

## Внешний вид закреплённого места

- По умолчанию закреплённые места окрашены в серый цвет
- Цвет можно изменить в настройках модуля (вкладка «Настройки стилей»)
- На карте закреплённые места отмечены серым цветом в фильтре по дате

## Граничные случаи

- Закреплённое место недоступно для бронирования никому, кроме закреплённого сотрудника
- Можно закрепить только за одним сотрудником одновременно
- Цвет закреплённого места настраивается глобально для всех закреплённых мест
