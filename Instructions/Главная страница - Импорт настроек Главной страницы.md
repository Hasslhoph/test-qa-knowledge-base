# Импорт настроек Главной страницы

**Модуль:** [[Module - Главная страница]]

## Описание

Инструкция по экспорту и импорту настроек Главной страницы для временного изменения конфигурации или переноса настроек между порталами.

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- **Экспорт настроек** вкладок «Компоненты» и «Источники данных»
- **Импорт настроек** из файла
- **Сохранение текущей конфигурации** перед временными изменениями
- **Перенос настроек** между порталами

## Предусловия

- Права администратора
- Настройки модуля: `/bitrix/admin/settings.php?lang=ru&mid=ithive.homepage`

## Шаги

### Экспорт настроек

1. Настройки модуля → вкладка «Импорт/Экспорт настроек»
2. Скачайте файл с текущими настройками

### Импорт настроек

1. Настройки модуля → вкладка «Импорт/Экспорт настроек»
2. Загрузите файл с настройками

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- **Что импортируется:** настройки вкладок «Компоненты» и «Источники данных» (только значения, не файлы)
- **Что НЕ импортируется:** фотографии, новости, сервисы, другой контент; настройки компонентов через режим правки (шаблоны баннеров, виджеты)
- **Источники данных:** при импорте выгружаются как настройки сайта, так и общие настройки инфоблоков
- Если на целевом портале отсутствуют ИБ и папки с такими же названиями, соответствующие опции не будут заполнены
- Перед импортом убедитесь, что идентичные ИБ и папки есть на целевом портале
