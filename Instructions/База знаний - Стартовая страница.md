# База знаний - Стартовая страница

**Модуль:** [[Module - База знаний]]

## Описание

Стартовая страница — корневой раздел Базы знаний. Слева отображаются разделы первого уровня, справа — список последних добавленных статей. Администраторы могут добавить описание на стартовую страницу.

## Ключевые функции

- Отображение разделов первого уровня и последних статей
- Кнопка возврата на стартовую страницу из любого раздела
- Добавление описания на стартовую страницу (только для администраторов)
- Возможность отключить вывод статей на стартовой странице через настройки модуля

## Предусловия

- Права администратора для настройки стартовой страницы

## Шаги

### Просмотр стартовой страницы

1. Кликните на пункт меню «База знаний»
2. Слева — разделы первого уровня, справа — последние статьи
3. Для возврата на стартовую страницу из раздела — кнопка в правом верхнем углу

### Добавление описания на стартовую страницу

1. Перейдите на стартовую страницу (`адрес_портала/knowledgebase/`)
2. Нажмите «Настройки» (⚙️) > «Настройка стартовой»
3. В открывшемся слайдере добавьте описание с помощью визуального редактора
4. Сохраните изменения

## Ожидаемый результат

Описание отображается на стартовой странице сразу после сохранения.

## Граничные случаи

- Вывод статей на стартовой странице можно отключить в настройках модуля
- Права на настройку стартовой страницы есть только у администраторов портала
