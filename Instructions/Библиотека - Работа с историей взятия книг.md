# Работа с историей взятия книг

**Модуль:** [[Module - Библиотека]]

## Описание

Инструкция по корректировке истории взятий и возвратов бумажных книг в административной части портала. Администратор может редактировать и удалять записи из истории.

## Ключевые функции

- Просмотр общего списка сотрудников, которые брали и возвращали книги
- Удаление действий из истории (например, ошибочное взятие)
- Редактирование записей: книга, сотрудник, дата взятия, плановая дата возврата, фактическая дата возврата

## Предусловия

- Права администратора портала
- Модуль «Библиотека» установлен и активен

## Шаги

### Просмотр истории

1. Перейдите в административную часть
2. «Контент» → «Highload-блоки» → «История кто брал книги»

### Удаление записи из истории

1. Нажмите бургер-меню (≡) слева от нужного пункта
2. Выберите «Удалить»

### Редактирование записи

1. Нажмите «Изменить» у нужной записи
2. Доступные поля для редактирования:
   - **Книга** — наименование книги
   - **Кто взял** — ID сотрудника
   - **Когда взял** — дата взятия
   - **Вернет до** — плановая дата возврата
   - **Когда вернул** — фактическая дата возврата
3. Сохраните изменения

## Граничные случаи

- Потребность в редактировании возникает, например, когда сотрудник взял книгу по ошибке
- История взятий и возвратов также доступна в карточке книги по ссылке «Смотреть историю»
