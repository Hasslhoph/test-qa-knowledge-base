# Привязка позиций к сотрудникам

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Процесс ручного присвоения позиций сотрудникам через раздел «Сотрудники». Позиция также может быть присвоена автоматически по результатам прохождения программы развития.

## Ключевые функции

### Раздел «Сотрудники»

- Доступен всем пользователям портала
- Информация о компетенциях и программах развития каждого сотрудника
- Просмотр чужих карточек: руководители, администраторы, группа HR-отдела, наблюдатели

### Индивидуальное прикрепление позиции

1. Раздел «Сотрудники» → бургер-меню (≡) → «Привязать к позиции»
2. Выбрать позицию из списка (только одну)
3. Результат:
   - Если нет позиции — прикрепляется выбранная
   - Если есть позиция — заменяется на выбранную
   - Если запущена программа развития — позиция прикрепляется даже при незавершённом ИПР

### Массовое прикрепление позиций

1. Отметить сотрудников галочками → «Привязать к позиции» → «Применить»
2. Выбрать позицию → следующий шаг
3. Проверить предварительный результат → сохранить

### Где посмотреть позицию сотрудника

1. Раздел «Сотрудники» (колонка позиции)
2. Личный кабинет КУ (если не указана — выводится должность)
3. Раздел «Карьерные траектории» (подпись «Это вы»)
4. Раздел «Мониторинг развития» (колонка позиции)

## Предусловия

- Права на привязку: администраторы и группа HR-отдела
- Созданные позиции в схеме карьерных траекторий
