# Управление таймингом рассылки

**Модуль:** [[Module - Библиотека]]

## Описание

Инструкция по изменению периодичности и точного времени отправки email-уведомлений о книжных новинках. По умолчанию рассылка запускается каждый понедельник в 7:00.

## Ключевые функции

- Изменение точного времени отправки уведомлений через настройки модуля
- Настройка периодичности рассылки через Агента Битрикс24
- Интервал по умолчанию: 604800 секунд (1 неделя)

## Предусловия

- Права администратора портала
- Подписка на книжные новинки активна

## Шаги

### Изменение времени отправки

1. Перейдите: `адрес_портала/bitrix/admin`
2. «Настройки» → «Настройки модулей» → «Улей: Библиотека»
3. Перейдите на вкладку «Подписка»
4. В поле «Время запуска (по понедельникам)» укажите нужное время
5. Сохраните изменения

### Настройка периодичности рассылки

1. Перейдите: `адрес_портала/bitrix/admin`
2. «Настройки» → «Настройки продукта» → «Агенты»
3. В поле «Модуль» введите «ithive.library»
4. Из списка выберите `ITHive\Library\Library::Agent();`
5. Нажмите бургер-меню (≡) → «Изменить»
6. Настройте поля:
   - **Дата и время следующего запуска** — конкретная дата и время следующей рассылки
   - **Интервал (сек.)** — периодичность отправки (по умолчанию 604800 сек = 1 неделя)
7. Сохраните изменения

## Граничные случаи

- Дата следующего запуска рассчитывается автоматически исходя из заданного интервала
- С помощью настройки даты можно задать конкретный день недели для следующей рассылки
- Агенты — базовая технология Битрикс24
