# Пульс-опросы

**Модуль:** [[Module - Опросы]]

## Описание

Пульс-опросы — короткие и регулярные исследования (1-3 вопроса) для оперативного получения обратной связи от сотрудников. Доступны в двух форматах: всплывающее окно на каждой странице портала и виджет на Главной странице.

## Ключевые функции

### Окно пульс-опроса

- Всплывающее поп-ап окно на каждой странице портала
- Пользователь может закрывать окно неограниченно, но оно появляется до тех пор, пока сотрудник не проголосует
- Для создания необходимо:
  1. Опрос должен входить в группу пульс-опросов
  2. Активирована галочка «Обязательный опрос» при создании/редактировании
- В настройках модуля можно указать адреса страниц, где окно пульс-опросов не должно выводиться

### Виджет пульс-опроса

- Небольшой интерактивный блок в правой части Главной страницы
- Требует установленный модуль [[Module - Главная страница]]
- Числовой индикатор показывает количество активных опросов
- При отсутствии активных пульс-опросов виджет не отображается

### Управление виджетом

- **Создать новый** — слайдер добавления опроса (по умолчанию в группе «Пульс-опросы»)
- **Редактировать** — слайдер редактирования текущего опроса
- **Показать результаты** — страница с результатами
- **Выгрузить Excel** — скачивание результатов
- **К списку опросов** — переход на домашнюю страницу модуля

## Предусловия

- Модуль «Опросы» установлен
- Для виджета: модуль [[Module - Главная страница]] установлен
- Для создания опросов: права администратора или редактора
- Для настройки виджета: права администратора портала

## Шаги (создание всплывающего окна)

1. Создать группу пульс-опросов или использовать существующую
2. Создать опрос и выбрать группу пульс-опросов
3. Активировать галочку «Обязательный опрос»
4. Сохранить опрос

## Шаги (создание виджета)

1. Добавить опрос в группу пульс-опросов
2. Перейти в настройки Главной страницы: `адрес_портала/bitrix/admin/settings.php?lang=ru&mid=ithive.homepage`
3. Во вкладке «Компоненты» найти таблицу «Правый блок»
4. Поставить галочку напротив «Пульс-опросы»
5. Сохранить настройки

## Ожидаемый результат

Пульс-опросы отображаются в виде всплывающих окон на страницах портала и/или в виде виджета на Главной странице.

## Граничные случаи

- С обновлением 8.0.0 окно пульс-опросов выводится на всех страницах портала и больше не зависит от модуля «Главная страница»
- Если опрос анонимный, страница результатов недоступна обычным пользователям
