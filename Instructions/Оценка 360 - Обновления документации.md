# Оценка 360 - Обновления документации

**Модуль:** [[Module - Оценка 360]]

## Описание

Файл содержит историю изменений документации модуля «Оценка 360». Ведётся хронология добавления новых статей, обновления текста и скриншотов в соответствии с выходами новых версий модуля.

## Ключевые функции

- Фиксация новых статей в документации
- Отслеживание обновлений текста и скриншотов существующих статей
- Привязка изменений к версиям модуля (7.0.0 — 10.4.0)
- Ссылки на изменённые статьи документации

## Предусловия

- Доступ к репозиторию документации модуля
- Понимание структуры инструкции (разделы user-guide)
- Осведомлённость о выходящих обновлениях модуля

## Шаги

1. Открыть файл «Обновления документации».
2. Просмотреть записи в хронологическом порядке (сверху — самые свежие).
3. Ознакомиться с типом изменения (новая статья, обновлён текст, обновлены скриншоты).
4. Перейти по ссылке в изменённую статью для просмотра актуального содержания.

## Ожидаемый результат

Пользователь и автор документации всегда в курсе последних изменений в инструкции к модулю «Оценка 360».

## Граничные случаи

- Если запись об изменении отсутствует, документация могла обновляться без отдельного упоминания — рекомендуется свериться с историей версий модуля.
- Некоторые записи содержат множественные изменения в рамках одной даты — каждое изменение выделено отдельным блоком.
