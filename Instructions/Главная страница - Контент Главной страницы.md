# Контент Главной страницы

**Модуль:** [[Module - Главная страница]]

## Описание

Инструкция по наполнению контентом Главной страницы на втором сайте (или основном).

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- **Баннеры:** создание, перенос в папку сайта
- **Новости компании/СМИ:** создание статей
- **Сервисы и заявки:** создание подразделов и сервисов
- **Фотоальбом:** создание альбомов и загрузка фото
- **Видеоальбом:** загрузка видео с привязкой к папке
- **Лидеры продаж:** создание разделов и добавление сотрудников
- **Кадровые изменения:** стандартный раздел Битрикс24 (общий для всех сайтов)
- **Живая лента:** стандартный раздел (автоматически для каждого сайта)
- **Виджеты:** стандартные виджеты Битрикс24, виджеты Улья, виджеты Главной

## Предусловия

- Созданные и указанные папки разделов для сайта ([[Главная страница - Добавление модуля на второй сайт]])
- Права администратора или привилегированного пользователя

## Шаги

### Баннеры

1. На Главной в режиме правки: шестерёнка → «Создать»
2. Заполните поля баннера
3. Перейдите в админку: «Контент» > «Главная страница» > «Баннеры»
4. Перенесите баннер в папку второго департамента

### Новости

1. На Главной → раздел «Новости компании» или «Новости СМИ»
2. Нажмите «Добавить новость»
3. Заполните поля и сохраните

### Сервисы и заявки

1. Админка: «Контент» > «Главная страница» > «Сервисы и заявки» → папка сайта
2. «Добавить раздел» → создайте подраздел
3. Откройте раздел → «Добавить сервис»
4. Заполните поля, привяжите к разделу второго сайта

### Фотоальбом

1. Админка: «Контент» > «Фотогалерея» > «Фотогалерея пользователей» → папка сайта
2. «Добавить альбом» → создайте тематический альбом
3. На Главной → «Фотоальбом» → «Загрузить фото»

### Видеоальбом

1. На Главной → «Видеоальбом» → «Добавить видео»
2. Заполните поля, привяжите к папке второго сайта

### Лидеры продаж

1. Админка: «Контент» > «Главная страница» > «Лидеры продаж» → папка сайта
2. «Добавить раздел» → создайте тематический раздел
3. Откройте раздел → «Добавить сотрудника»

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- Баннеры сохраняются в корне инфоблока — их нужно переносить в папку сайта
- Кадровые изменения и Живая лента — общие для всех сайтов, настройки не требуются
- Стандартные виджеты Битрикс24 не требуют настройки для второго сайта
- Виджеты «Ссылки» и «Социальные сервисы» требуют добавления хотя бы одной ссылки в папку сайта
