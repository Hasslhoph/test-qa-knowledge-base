# Карточка оценки

**Модуль:** [[Module - Управление целями. Каскад]]

## Описание

Карточка оценки — инструмент для фиксирования прогресса в достижении целей. Создается автоматически при наступлении отчетного периода. У сотрудника может быть несколько карточек оценок в течение одного процесса.

## Ключевые функции

> Фиксация и отслеживание прогресса по целям.

- Автоматическое создание по расписанию (неделя, 2 недели, месяц, квартал, полугодие)
- 6 статусов: Составление → Согласует руководитель → Согласует функциональный руководитель → Согласует директор → Утверждает администратор → Утверждено
- Указание процента выполнения и комментария
- Отслеживание динамики (предыдущий прогресс)
- Экспорт списка оценок (Excel, печать, почта)

## Периодичность оценок

- **Неделя** — дедлайн: пятница
- **2 недели** — дедлайн: пятница
- **Месяц** — дедлайн: последний рабочий день месяца
- **Квартал** — дедлайн: последний рабочий день квартала
- **Полугодие** — дедлайн: последний рабочий день полугодия

## Обзор карточки оценки

### Статусы карточки

1. **Составление** — комментарий и процент прогресса
2. **Согласует руководитель** — проверка прямым руководителем
3. **Согласует функциональный руководитель** — проверка функциональным руководителем
4. **Согласует директор** — проверка руководителем на уровень выше
5. **Утверждает администратор** — проверка и утверждение
6. **Утверждено** — комментарии и прогресс нельзя изменить, прогресс попадает в сводный отчет

### Информация о карточке

- **Оцениваемый** — пользователь, для которого создана карточка
- **Данные о процессе** — дедлайны, сроки, корректировки
- **Прогресс по целям:**
  - Шкала «Время процесса» — автоматически заполняется
  - Шкала «Прогресс» — среднее значение по всем целям

### Блок фиксирования результатов

Поля при заполнении:
1. **Процент выполнения** — процент достижения (обязательно)
2. **Комментарий сотрудника** — отчет о проделанной работе (обязательно, только для владельца)
3. **Предыдущий прогресс** — процент из предыдущей карточки

## Заполнение карточки

- Указать процент по каждой цели
- Оставить комментарий о проделанной работе
- «Сохранить оценку» или «Отправить на согласование»
- Модуль высчитывает среднее значение прогресса

### Особенности

- Каскадированный прогресс учитывается только в карточке сотрудника
- Руководитель заполняет прогресс по каскадированным целям самостоятельно
- Поле «Комментарий сотрудника» доступно только владельцу карточки

## Повторная оценка

- Для каждого периода создается новая карточка
- Новая карточка создается на основе текущей при наступлении даты согласования
- Пример: годовой процесс с квартальными отчетами = 4 карточки оценки

## Вкладка «История»

- Все действия с карточкой
- Этапы согласования
- Комментарии

## Экспорт списка оценок

- **В Excel** — файл xls с отображенными столбцами
- **Распечатать** — печать в табличном формате
- **Отправить на почту** — xls на почту пользователя

## Предусловия

- Утвержденная карточка целей
- Наступивший отчетный период

## Граничные случаи

- Согласующие могут менять процент выполнения (обязательно с комментарием)
- Согласующие могут отправить карточку вместо сотрудника (без комментария сотрудника)
- При каскадировании прогресс учитывается только в карточке получателя
