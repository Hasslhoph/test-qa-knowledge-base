# Настройки модуля «Корпоративный университет»

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Основные настройки модуля в административной части портала. Пять вкладок: Общие, Уведомления, Пользовательские функции, Развивающие действия, Доступ.

## Ключевые функции

### Вкладка «Общие»

1. **Настройка доступа к публичной части** — отделы/сотрудники для каждого сайта
2. **Редактор Сайты24** — визуальный редактор для курсов
3. **Публичный доступ к курсам** — ссылка `/pub/learning/`
4. **Отзывы у курсов** — возможность оставлять отзывы и прикреплять результаты
5. **Информация о компетенциях на странице курса** — блоки компетенций и программ
6. **Компетенции и программа в отдельном слайдере** — пункт «Университет» в профиле
7. **Вывод новостей** — раздел «Новости» в КУ
8. **Программы адаптации** — онбординг с меньшими сроками
9. **Кто может запускать программы адаптации** — группы пользователей
10. **Кто может запускать программы развития** — группы пользователей
11. **Кто может завершать карточки развития** — роли: Сотрудник, Руководитель, HR-отдел, Администраторы
12. **Количество незавершённых программ** — от 1 до 5 (по умолчанию 2)
13. **Кто может менять сроки развивающих действий** — роли с доступом
14. **Группа HR-отдела** — пользователи с максимальным набором прав
15. **Группа модераторов мероприятий**
16. **Группа для назначения курсов**
17. **Группа для отчётов по обучению**
18. **Настройки инфоблоков** — не рекомендуется менять
19. **Ограничение функциональности при большом количестве попыток** — для порталов >1000 пользователей

### Вкладка «Уведомления»

1. **Регулярные** — раз в месяц о самооценке
2. **Финальные** — по окончании срока программы
3. **При создании** — о запуске программы развития/онбординга
4. **При оценке** — после оценки сотрудника/руководителя
5. **При назначении** — о назначении онлайн-курса
6. **Развивающие действия** — о новом действии и приближающемся дедлайне

### Вкладка «Пользовательские функции»

- Данные для корректной работы КУ (не рекомендуется менять)

### Вкладка «Развивающие действия»

- Автоматическое добавление в участники события календаря

### Вкладка «Доступ»

- Группы пользователей с доступом к настройкам модуля
