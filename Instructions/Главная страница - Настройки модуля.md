# Настройки модуля

**Модуль:** [[Module - Главная страница]]

## Описание

Основные настройки модуля «Главная страница» в административной части портала. Включают 5 вкладок: Компоненты, Источники данных, Новости, Кастомные стили, Импорт/Экспорт настроек.

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- **Вкладка «Компоненты»:** настройка отображения блоков и виджетов на Главной, шаблон модуля (Default/Custom)
- **Вкладка «Источники данных»:** настройка инфоблоков для каждого сайта, выбор альбомов фото/видео
- **Вкладка «Новости»:** постмодерация комментариев, модераторы, ограничения символов, избранное, важные новости, группировка по месяцам, доступ к лайкам/просмотрам
- **Вкладка «Кастомные стили»:** загрузка CSS-архива для кастомного дизайна
- **Вкладка «Импорт/Экспорт настроек»:** скачивание/загрузка настроек

## Предусловия

- Права администратора портала
- Доступ к `/bitrix/admin/settings.php?lang=ru&mid=ithive.homepage`

## Шаги

### Вкладка «Компоненты»

1. Шаблон модуля: Default (стандартный) или Custom (кастомные CSS-стили)
2. Центральный блок: Активность, Сортировка, Название, Вывести в левом меню
3. Правый блок (виджеты): Активность, Сортировка, Название, Код виджета
4. Добавить виджет: создание собственного виджета

### Вкладка «Новости» — Модерация комментариев

1. Режим модерации: «Постмодерация» для включения
2. Модераторы: выбор сотрудников/групп
3. Шаблон уведомления: текст уведомления о новых комментариях

### Вкладка «Новости» — Дополнительно

1. Ограничение символов в заголовке
2. Ограничение символов в анонсе
3. Избранное: вкл/выкл
4. Блок «Важные новости»: показать/скрыть
5. Группировка по месяцам: вкл/выкл
6. Доступ к списку лайков
7. Доступ к списку просмотров

### Вкладка «Кастомные стили»

1. Создайте файл `.css`, положите в папку `custom`
2. Сохраните папку в архив `custom.zip` (структура: `custom.zip > custom > .css`)
3. Загрузите архив во вкладке «Кастомные стили»
4. Во вкладке «Компоненты» выберите «Custom»

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- Настройки «Источники данных» не рекомендуется менять при одном сайте
- При кастомных стилях архив должен содержать и дефолтные стили модуля
- Дефолтные стили: `/bitrix/css/ithive.homepage/templates/default` (`homepage.css`, `style.css`)
