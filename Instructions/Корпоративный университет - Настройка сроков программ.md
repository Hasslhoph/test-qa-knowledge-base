# Настройка сроков программ

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Настройка сроков программ развития и адаптации через административную часть портала.

## Ключевые функции

### Сроки по умолчанию

- **Программа адаптации**: 1-3 месяца
- **Программа развития**: 6-24 месяцев

### Как изменить сроки

1. Административная часть → «Контент» → «Корпоративный университет» → «ИПР»
2. Настройки инфоблока → вкладка «Свойства»
3. Три точки (...) → «Настройки свойства инфоблока»:
   - Для онбординга: «ИПР — Период выполнения ПИС»
   - Для развития: «Период выполнения ИПР»
4. Добавить новые значения:
   - **XML_ID** — количество месяцев
   - **Значение** — текст в поле «Сроки»
   - **Сортировка** — порядок отображения

## Предусловия
