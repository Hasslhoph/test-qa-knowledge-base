# HelpDesk - Установка, обновление и удаление

**Модуль:** [[Module - HelpDesk]]

## Описание

Процедуры установки, обновления, активации купона и удаления модуля ithive.helpdesk. Все операции требуют прав группы «Администраторы».

## Ключевые функции

### Установка модуля
1. Перейти: `/bitrix/admin/update_system_partner.php?addmodule=ithive.helpdesk`
2. Модуль автоматически отмечен → кнопка «Загрузить»
3. Подтвердить Лицензионное соглашение и Условия конфиденциальности → «Применить»
4. Дождаться завершения установки
5. Уведомление об успешной загрузке

### Обновление модуля
- Marketplace → Обновление решений → Список обновлений → «Улей: Helpdesk» → «Загрузить»
- Или: `/bitrix/admin/update_system_partner.php`

### Активация купона
- Marketplace → Обновление решений → Активация купона → ввести номер → «Активировать»
- Или: `/bitrix/admin/update_system.php?lang=ru`

**Для неустановленного модуля:**
- Marketplace → Установленные решения → Купленные решения → бургер-меню → «Загрузить»

**Для модуля с истёкшей демоверсией:**
- Обновление решений → Установка обновлений → «Зарегистрировать решение»

### Удаление модуля
1. Marketplace → Установленные решения → Доступные решения
2. Бургер-меню → «Удалить»
3. Подтвердить удаление
4. Уведомление об удалении

## Предусловия

- Права группы «Администраторы»

## Граничные случаи

- При активации купона на неустановленный модуль — дополнительная загрузка из «Купленных решений»
- При истёкшей демоверсии — требуется регистрация решения
