# Legacy. Загрузка дизайна на портал

**Модуль:** [[Module - Брендирование Битрикс24]]

## Описание

Инструкция по выгрузке стилей из Figma и загрузке их на корпоративный портал Битрикс24 для дизайна «Legacy». Требуется десктопное приложение Figma и специальный плагин от Улья.

## Ключевые функции

### Установка плагина

1. Скачать плагин с Яндекс.Диска: `Hive.Design - Branding 2.0.0`
2. В десктопном Figma: `Plugins > Development > Import plugin from manifest`
3. Выбрать файл `manifest.json` из скачанной папки
4. Плагин появится в меню как «Hive.Design — Branding»

### Выгрузка и загрузка

1. В Figma: `Plugins > Development > Hive.Design — Branding`
2. В блоке «Стили» выбрать макет → нажать «Сгенерировать»
3. В блоке «Экспорт Bitrix» нажать «Сконвертировать» → «Скачать ZIP»
4. На портале: открыть `/changeportaldefaultheme/`
5. Нажать шестерёнку (⚙️) → в поле «Импорт настроек» загрузить ZIP-файл
6. Новый дизайн сразу отобразится на портале

## Предусловия

- Дизайн создан в макете Figma
- Установлен десктопный плагин Hive.Design — Branding
- Пользователь имеет доступ к модулю «Брендирование»

## Граничные случаи

- Плагин работает только в десктопном приложении Figma (не в браузере)
- ZIP-файл содержит все настройки, включая состояние модуля (включён/выключен)
- При импорте настройки применяются немедленно
