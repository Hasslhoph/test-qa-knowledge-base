# О продукте

**Модуль:** [[Module - Управление целями. Каскад]]

## Описание

Стартовая страница инструкции к модулю «Управление целями. Каскад». Модуль автоматизирует постановку целей подразделениям и контроль за достижением этих целей. Целеполагание в модуле реализовано по принципу каскадирования.

## Технические данные

- Модуль совместим с PHP версией 8.1 и выше
- Модуль совместим с СУБД: MySQL, PostgreSQL
- Модуль поддерживает английский язык
- Модуль поддерживает многодепартаментность
- Функционал модуля доступен через современный десктопный браузер

**Инструкция актуальна для версии модуля:** 8.1.0
**Последняя актуализация:** Январь 2026

## Ключевые функции

> Основные возможности модуля, описанные в документации.

- Автоматизация постановки целей по принципу каскадирования
- Интеграция с оргструктурой и задачами Битрикс24
- Многоэтапное согласование целей и отчетов
- Контроль прогресса достижения целей
- Сводные отчеты по подразделениям
- Гибкий механизм управления целями (квартал, полгода, год)
