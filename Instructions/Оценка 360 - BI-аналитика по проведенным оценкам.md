# Оценка 360 - BI-аналитика по проведенным оценкам

**Модуль:** [[Module - Оценка 360]]

## Описание

Загрузка CSV-файла с данными об оценках в BI-конструктор Битрикс24 для построения интерактивных дашбордов с диаграммами и графиками.

## Ключевые функции

### Выгрузка CSV-файла
- Раздел «Оценки» → шестерёнка (⚙️) → вариант экспорта:
  - «Экспорт оценок по компетенциям» — без индикаторов
  - «Экспорт оценок по компетенциям с индикаторами» — с индикаторами (только для шкалы)
- Выбор видов оценок: Средние, Средние без самооценки, Самооценка, Оценка руководителя, Итоговая
- Включить все колонки списка перед выгрузкой

### Подготовка CSV-файла
1. Изменить формат ячеек «Прогресс» с процентного на общий
2. Переименовать колонки компетенций (max 32 символа) с помощью формул Excel
3. Формула обрезает окончания «- Самооценка», «- Оценка руководителя», «- Итоговая оценка» и добавляет суффиксы «_S», «_R», «_I»
4. Удалить формулы, оставить обычный текст
5. Опционально: добавить префиксы «K_» для компетенций, «IN_» для индикаторов

### Загрузка в BI-конструктор
- «BI-конструктор» → «Рабочее место аналитика» → «Создать» → «Из CSV-файла»
- Кодировка: UTF-8, Разделитель: Точка с запятой
- Настройка форматов данных: дата, деньги, число
- Настройка типов колонок: десятичное число, целое число, дата и время
- Названия колонок автоматически транслитерируются

### Создание дашборда
- Создать группу (папку) → создать отчёт (дашборд)
- Настройки датасета: переименование столбцов кириллицей
- Настройки дашборда: имя, URL, доступ, цветовая палитра

### Типы графиков
1. **Карточка с показателями** — количество оценок, сотрудников, максимальная оценка
2. **Карточка с линией тренда** — динамика оценок по компетенции
3. **Сводная таблица** — оценки по компетенциям в разрезе подразделений
4. **Столбчатая диаграмма** — динамика оценок в разрезе подразделений
5. **Диаграмма «Радар»** — оценки по индикаторам в разрезе подразделений

### Кросс-фильтрация
- Фильтр, применённый к одному графику, применяется ко всем
- Можно добавлять собственные фильтры (временной интервал и др.)

## Предусловия

- Права на выгрузку оценок
- BI-конструктор Битрикс24 доступен на портале
- CSV-файл подготовлен и отредактирован

## Граничные случаи

- Подготовка файла — трудозатратный процесс
- Названия колонок датасета не могут быть длиннее 32 символов
- При большом количестве компетенций и индикаторов файл может содержать сотни колонок
- Для радарной диаграммы нужно вручную указать min/max значения шкалы
