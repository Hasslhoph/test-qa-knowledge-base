# Оценка 360 - BI-аналитика по проведенным оценкам

**Модуль:** [[Module - Оценка 360]]

# BI-аналитика по проведенным оценкам

Модуль позволяет выгружать данные об оценках сотрудников в CSV-формате. Полученный CSV-файл можно загрузить в [BI-конструктор Битрикс24](https://helpdesk.bitrix24.ru/open/18897300/), чтобы затем составить интерактивный дашборд с диаграммами и графиками.
В данной статье мы рассмотрим как импортировать CSV-файл в BI-конструктор и получить следующую аналитику:

- Количество сотрудников и оценок (карточка с показателями)

- Максимальная оценка по компетенции (карточка с показателями)

- Динамика оценки по каждой компетенции (карточка с линией тренда)

- Средние оценки по компетенциям в разрезе подразделений (сводная таблица)

- Динамика оценок по компетенциям в разрезе подразделений (столбчатая диаграмма)

- Средние оценки по индикаторам в разрезе подразделений (диаграмма «Радар»)

Пример дашборда, который включает в себя всю перечисленную аналитику: 

![](_images-09/13.png)

![](_images-09/14.png)

{% note warning %}

После экспорта CSV-файла его нужно будет вручную отредактировать, чтобы подготовить к загрузке в BI-конструктор. Кроме того, созданный в BI-конструкторе датасет также нужно будет настраивать вручную. Работа с файлами — трудозатратный процесс, на который потребуется выделить время.

{% endnote %}

## Выгрузка CSV-файла

Чтобы выгрузить данные об оценках сотрудников перейдите в раздел модуля «Оценки» и нажмите на шестеренку (⚙️). В выпадающем меню выберите вариант экспорта:

- **Экспорт оценок по компетенциям.** Excel или CSV-файл с оценками только по компетенциям, без детализации по индикаторам.

- **Экспорт оценок по компетенциям с индикаторами.** Excel или CSV-файл с оценками по компетенциям и их индикаторам. Обратите внимание, что оценки по индикаторам выгружаются только для компетенций, которые оцениваются по шкале.

![](_images-09/image132.png)

В рамках данной статьи мы будем использовать CSV-файл с компетенциями и индикаторами. Перед выгрузкой рекомендуется включить для отображения все колонки списка.

![](_images-09/image260.png)

На следующем шаге модуль предложит выбрать, какие именно оценки по компетенциям и индикаторам нужно выгрузить. Для выбора доступно 5 вариантов, можно указать несколько:

- Средние значения

- Средние значения без самооценки

- Самооценка

- Оценка руководителя

- Итоговая оценка

![](_images-09/image429.png)

{% note warning %}

Чем больше значений выбрано для выгрузки, тем больше колонок с оценками будет содержать CSV-файл. Например, если в оценке присутствуют 20 компетенций, в каждой из которых по 10 индикаторов, и для выгрузки выбраны все 5 видов оценок, то в файле будет следующее количество колонок: 20×5 + 20×10×5 = 1100. Этот момент важно учитывать, так как данные из файла нужно будет редактировать до и после импорта в BI-конструктор. Количество колонок влияет на трудозатратность этого процесса.

{% endnote %}

В рамках данной статьи мы выгрузим три вида оценки: Самооценку, Оценку руководителя и Итоговую оценку.

![](_images-09/image249.png)

Когда оценки выбраны, нажмите «Выполнить» > «Скачать файл экспорта». Файл с выбранными оценками будет загружен.

![](_images-09/image138.png)

## Формат ячеек в колонке «Прогресс»

Для начала в колонке «Прогресс» нужно изменить формат ячеек с процентного на общий. Кликните на ячейку Е2 и зажмите Ctrl + Shift + **⇩** (Cmd + Shift + **⇩** на Mac), чтобы выделить весь остальной столбец. Кликните по столбцу правой кнопкой и выберите «Формат ячеек» > «Общий».

![](_images-09/image162.png)

![](_images-09/image392.png)

В результате процентные значения в столбце будут переведены в десятичные.

![](_images-09/image143.png)

## Переименование колонок компетенций и индикаторов

Следующий шаг — переименовать колонки с компетенциями и индикаторами таким образом, чтобы количество символов в названии не превышало 32 символа. В BI-конструкторе название колонок датасета (файла с данными) не может быть длиннее этого значения. Переименовать ячейки можно вручную, но в данной статье мы рассмотрим ускоренный способ с использованием формул.

Правой кнопкой кликните на ячейку А1 и нажмите «Вставить», чтобы в строке 1 появилась новая пустая строка.

![](_images-09/image121.png)

В ячейку L1 (самая первая колонка с названием компетенции) вставьте формулу для переименования столбцов. В зависимости от версии Excel используйте одну из двух формул:

1. **Для версий Excel старше 2021 года, а также для Гугл-таблиц:**

```text
=LET(
    txt; L2;
    self; " - Самооценка";
    boss; " - Оценка руководителя";
    final; " - Итоговая оценка";
    check_self; ПРАВСИМВ(txt; ДЛСТР(self)) = self;
    check_boss; ПРАВСИМВ(txt; ДЛСТР(boss)) = boss;
    check_final; ПРАВСИМВ(txt; ДЛСТР(final)) = final;
    prefix_len; ЕСЛИ(check_self; ДЛСТР(txt)-ДЛСТР(self);
                ЕСЛИ(check_boss; ДЛСТР(txt)-ДЛСТР(boss);
                ЕСЛИ(check_final; ДЛСТР(txt)-ДЛСТР(final); ДЛСТР(txt))));
    prefix_short; ЛЕВСИМВ(txt; МИН(30; prefix_len));
    suffix; ЕСЛИ(check_self; "_S"; ЕСЛИ(check_boss; "_R"; ЕСЛИ(check_final; "_I"; "")));
    ЕСЛИ(suffix=""; txt; prefix_short & suffix)
)
```

2. **Для любой версии Excel, а также для Гугл-таблиц (универсальная):**

```text
=ЕСЛИ(
  ЕСЛИ(ПРАВСИМВ(L2;ДЛСТР(" - Самооценка"))=" - Самооценка";"_S";ЕСЛИ(ПРАВСИМВ(L2;ДЛСТР(" - Оценка руководителя"))=" - Оценка руководителя";"_R";ЕСЛИ(ПРАВСИМВ(L2;ДЛСТР(" - Итоговая оценка"))=" - Итоговая оценка";"_I";"")))="";
  L2;
  ЛЕВСИМВ(L2;МИН(30;ЕСЛИ(ПРАВСИМВ(L2;ДЛСТР(" - Самооценка"))=" - Самооценка";ДЛСТР(L2)-ДЛСТР(" - Самооценка");ЕСЛИ(ПРАВСИМВ(L2;ДЛСТР(" - Оценка руководителя"))=" - Оценка руководителя";ДЛСТР(L2)-ДЛСТР(" - Оценка руководителя");ЕСЛИ(ПРАВСИМВ(L2;ДЛСТР(" - Итоговая оценка"))=" - Итоговая оценка";ДЛСТР(L2)-ДЛСТР(" - Итоговая оценка");ДЛСТР(L2))))))
  &ЕСЛИ(ПРАВСИМВ(L2;ДЛСТР(" - Самооценка"))=" - Самооценка";"_S";ЕСЛИ(ПРАВСИМВ(L2;ДЛСТР(" - Оценка руководителя"))=" - Оценка руководителя";"_R";ЕСЛИ(ПРАВСИМВ(L2;ДЛСТР(" - Итоговая оценка"))=" - Итоговая оценка";"_I";"")))
)
```

Формула проверяет название каждого столбца и обрезает это окончание, оставляя только первые 30 символов названия. Вместо названий «- Самооценка», «- Оценка руководителя», «- Итоговая оценка», формула добавляет короткий суффикс: «_S» для самооценки, «_R» для оценки руководителя и «_I» для итоговой оценки. Если название не содержит ни одного из этих окончаний, оно остается без изменений.

После вставки формулы в ячейку L1 вы увидите, что название столбца было сокращено. Теперь данную формулу нужно применить ко всем последующим столбцам с названиями компетенций и индикаторов.

![](_images-09/image114.png)

Для этого скопируйте ячейку L1 через Ctrl+C. В левом верхнем углу в поле «Имя» введите диапазон от L1 до последней колонки в таблице, где есть компетенция/индикатор. В рамках примера последней колонкой является FH1, соответственно диапазон будет таким: L1:FH1.

![](_images-09/image9.png)

Нажмите Enter, чтобы выделить весь указанный диапазон. Затем нажмите Ctrl+V чтобы вставить формулу во все выделенные ячейки. В результате, названия всех колонок с индикаторами и компетенциями будут сокращены.

![](_images-09/image112.png)

Скопируйте названия ячеек A2-K2 и вставьте их в ячейки выше, А1-K1 соответственно.

![](_images-09/image519.png)

На следующем шаге нужно удалить формулы из строки 1 и оставить там обычный текст. Для этого выделите строку 1 и нажмите Ctrl+C. Кликните правой кнопкой на выделенную строку и выберите «Вставить значения». В результате вся строка будет очищена от формул.

![](_images-09/image285.png)

Выделите строку 2 и удалите ее полностью, так как теперь в ней нет необходимости.

![](_images-09/image305.png)

## Дополнительные действия с колонками

В колонках CSV-файла не отмечено, какие именно наименования являются компетенциями, а какие — индикаторами. При необходимости вы можете вручную дописать к названиям компетенций символ «K_», а к названиям индикаторов «IN_». После импорта в BI конструктор это поможет быстрее отделять компетенции от индикаторов, чтобы строить нужные графики.

![](_images-09/image283.png)

## Загрузка CSV-файла в BI-конструктор

Все графики в дашбортах BI-конструктора основаны на датасетах, т.е. на наборах данных. Датасет автоматически создается на основе загруженного CSV-файла.

Чтобы создать датасет, перейдите в раздел «BI-конструктор» > «Рабочее место аналитика».

![](_images-09/image66.png)

Нажмите «Создать» > «Из CSV-файла».

![](_images-09/image262.png)

В открывшемся слайдере загрузите ранее подготовленный CSV-файл.

![](_images-09/image195.png)

Убедитесь, что в поле «Кодировка» указано значение «UTF-8», а в поле «Разделитель» — «Точка с запятой». Если все действия выполнены верно, то в блоке предпросмотра вы увидите колонки и значения из своего файла.

![](_images-09/image2.png)

Обратите внимание, что в BI-конструкторе названия колонок датасета автоматически переводятся на латиницу — это требование системы. При загрузке датасета переименовать колонки нельзя, но их можно будет изменить позднее в настройках, о чем написано [ниже](#nastroyki-dataseta-i-dashborda).

Кликните на поле «Форматы данных». Укажите корректные значения для следующих полей:

1. **Дата:** DD.MM.YYYY

2. **Дата и время:** Свой формат данных > DD.MM.YYYY hh:mm

3. **Деньги:** 12345,67

4. **Число:** 1,23

![](_images-09/image298.png)

Прокрутите файл ниже. В блоке «Свойства датасета» задайте ему название, например «assessment_2023_2025». По желанию укажите описание датасета.

![](_images-09/image64.png)

Перейдите к блоку «Настройка колонок датасета». Здесь укажите корректный тип ячеек для следующих колонок:

- **PROGRESS:** Десятичное число

- **OTSENIVAYUSHCHIE:** Целое число

- **KOMPETENTSIY:** Целое число

- **DATA_SOZDANIYA:** Дата и время

- **DATA_ZAVERSHENIYA:** Дата и время

![](_images-09/image297.png)

Для всех имеющихся колонок с компетенциями индикаторами выставите тип «Десятичное число». Данный процесс может занять время если в файле большое количество компетенций и индикаторов.

![](_images-09/image407.png)

Из-за транслитерации названия некоторых колонок могут увеличиться и превысить ограничение в 32 символа. Такие колонки будут выделены красным, их можно переименовать прямо на слайдере создания датасета.

![](_images-09/image26.png)

Когда типы ячеек будут указаны, нажмите «Проверить файл на ошибки» в правом верхнем углу слайдера. Если ошибок не выявлено, файл можно сохранить.

![](_images-09/image317.png)

![](_images-09/image1.png)

Датасет будет создан и появится в общем списке. После этого можно переходить к созданию дашборда и графиков.

![](_images-09/image474.png)

![](_images-09/image115.png)

## Создание дашборда

Перейдите в раздел «BI-конструктор» и нажмите «Создать» > «Группа». Группа — это папка для хранения отчетов одной тематики. Дайте группе название, например «HR-аналитика» и сохраните изменения.

![](_images-09/image34.png)

![](_images-09/image446.png)

Откройте папку и нажмите «Создать». Назовите новый отчет (дашборд). Например «Сводная по Оценке 360». Нажмите «Сохранить».

![](_images-09/image223.png)

![](_images-09/image43.png)

В результате откроется новый дашборд. Это рабочая область для добавления различных графиков.

![](_images-09/image306.png)

## Настройки датасета и дашборда

Прежде чем добавлять в дашборд графики, стоит настроить датасет и дашборд чтобы упростить дальнейшую работу с графиками. Откройте созданный дашборд и в верхнем правом углу нажмите «Редактировать».

![](_images-09/image177.png)

На странице редактирования перейдите в раздел «Датасеты». Выберите нужный дашборд из списка и нажмите «Редактировать» в строке.

![](_images-09/image461.png)

В открывшемся окне перейдите во вкладку «Столбцы». Здесь показан список всех столбцов из ранее загруженного CSV-файла.

![](_images-09/image505.png)

Нажмите на иконку плюса (+) рядом с нужным столбцом и в поле «Метка» введите название этого столбца кириллицей. Переименуйте все нужные столбцы — благодаря этому при создании графиков на них будут отображаться русскоязычные названия.

![](_images-09/image469.png)

Далее перейдите во вкладку «Дашборды». Найдите нужный дашборд в списке и нажмите «Редактировать» в строке.

![](_images-09/image386.png)

Здесь доступны следующие настройки для редактирования:

1. **Имя.** Название дашборда

2. **Читаемый URL.** Кастомный адрес дашборда

3. **Доступ.** Поле для указания сотрудников, которые могут редактировать дашборд и графики на нем. Подробнее о настройке прав написано [здесь](https://helpdesk.bitrix24.ru/open/25556500/).

4. **Цвета.** Цветовая палитра, которая будет применяться ко всем графикам и диаграммам в данном дашборде.

![](_images-09/image259.png)

## Создание графиков

## Карточка с показателями

Первый вид аналитики, который мы рассмотрим — это карточка с показателями. В рамках данного примера создадим три карточки: с общим количеством оценок, с количеством уникальных сотрудников и с максимальной полученной оценкой.

![](_images-09/image44.png)

Откройте созданный дашборд в режиме редактирования и в верхнем правом углу нажмите «+» > «График»

![](_images-09/image137.png)

![](_images-09/image227.png)

Выберите ранее загруженный датасет, на основе которого нужно будет построить аналитику. В списке графиков выберите «Карточка» и нажмите «Создать график».

![](_images-09/image247.png)

В результате откроется страница формирования графика. Здесь нужно указать какие столбцы использовать для получения той или иной аналитики.

В поле «Мера» перенесите столбец «Сотрудник. В качестве агрегатной функции укажите «COUNT». Это означает, что система будет высчитывать количество записей в колонке «Сотрудник».

![](_images-09/image29.png)

В поле «Фильтры» перетащите столбик «Дата завершения» и удалите значение ««Дата создания».

![](_images-09/image65.png)

Нажмите «Создать график». Вы увидите карточку с количеством количеством всех запущенных оценок.

![](_images-09/image390.png)

В левом верхнем углу задайте имя графика, например «Всего оценок». Дополнительно в разделе «Кастомизация» можно дать краткое пояснение к значению в поле «Подзаголовок». Здесь же можно увеличить или уменьшить шрифты.

![](_images-09/image187.png)

В левом верхнем углу нажмите «Сохранить» и подтвердите сохранение графика в нужном дашборде.

![](_images-09/image471.png)

В результате в дашборде появится первая карточка с аналитикой.

![](_images-09/image242.png)

Через редактирование дашборда карточку можно изменить в размере.

![](_images-09/image513.png)

Аналогичным способом можно создать карточку с количеством сотрудников, на которых запущены оценки, а также с максимальной полученной оценкой.

**Параметры для карточки «Всего сотрудников»:**

- **Мера:** столбец «Сотрудник»

- **Агрегатная функция:** `COUNT_DISTINCT` (считать только уникальные значения)

- **Фильтры:** Дата завершения

![](_images-09/image82.png)

**Параметры для карточки «Максимальная оценка»:**

- **Мера:** столбец с итоговой оценкой по любой компетенции, например «Клиентоориентированность (I)»

- **Агрегатная функция:** `MAX` (найти максимальное значение)

- **Фильтры:** Дата завершения

## Сводная таблица

Рассмотрим, как добавить в дашборд сводную таблицу с оценками по компетенциям в разрезе подразделений.

Выберите нужный датасет и тип графика «Сводная таблица. Нажмите «Создать график».

![](_images-09/image8.png)

На странице создания графика укажите следующие параметры:

### Вкладка «Данные»

1. **Строки.** Значения, которые должны быть указаны в строках таблицы. В рамках примера нас интересуют оценки в разрезе подразделений, поэтому перетащим сюда столбец «Подразделения».

![](_images-09/image103.png)

2. **Меры.** Данные, которые будут отображаться в столбцах таблицы. Так как нас интересуют оценки по компетенциям, в качестве мер (столбцов таблицы) укажем столбцы с итоговыми оценками по компетенциям. Например «Клиентоориентированность (I)», «Проактивность (I)» и т.п. Вы можете указать здесь любое нужное вам количество столбцов с компетенциями, столько же столбцов будет иметь таблица. В поле «Агрегатная функция» у всех мер укажите `AVG` (среднее).

При необходимости в меню настройки меры вы можете отредактировать метку, чтобы сделать название столбца с компетенцией более понятным.

![](_images-09/image101.png)

3. **Фильтры.** Укажите здесь значение «Дата завершения».

![](_images-09/image287.png)

4. **Функция агрегирования.** Опция влияет на то, какие значения будут выводиться в столбце с итогами, если он выведен (см. п.5). Например, можно вывести сумму значений по строке, среднее арифметическое, количество значений и т.п. Опцию можно заполнить по желанию. В рамках текущего примера установим здесь значение «Среднее» чтобы получить среднее арифметическое по строке.

5. **Показать общий итог по строкам.** Если галочка установлена, то в таблице появляется столбец «Итого», где подсчитано значение по строкам (вычисляемое значение указывается в п. 4). В настройке выше мы указали функцию «Среднее», поэтому в столбце будет выводиться среднее арифметическое оценок по компетенциям.

Поставьте галочку, если в таблицу требуется добавить колонку с итогами. Так, если в таблица включает компетенции только из одного раздела (Soft Skills или Hard Skills), то среднее арифметическое по строке будет равно оценке по данному разделу компетенций.

![](_images-09/image75.png)

В левом верхнем углу укажите название таблицы и нажмите «Создать график». Готовая таблица отобразится в рабочей области. При необходимости вы можете отредактировать внешний вид таблицы, чтобы сделать ее более наглядной. Для этого перейдите во вкладку «Кастомизация».

### Вкладка «Кастомизация»

Здесь доступны следующие параметры:

1. **Формат значения.** Опция влияет на то, в каком формате в таблице будут показаны оценки по компетенциям. В рамках текущей таблицы эта опция позволяет управлять количеством символов после запятой. Выберите одно из следующих значений:

- **,.1f (12345.432 =&gt; 12,345.4)** — 1 знак после запятой.

- **,.2f (12345.432 =&gt; 12,345.43)** — 2 знака после запятой.

- **,.2f (12345.432 =&gt; 12,345.432)** — 3 знака после запятой.

![](_images-09/image116.png)

2. **Сортировка строк по/Сортировать столбцы по.** Принцип сортировки строк и столбцов в таблице: по алфавиту (А-Я, Я-А), по возрастанию/убыванию значений (оценок).

![](_images-09/image160.png)

3. **Условное форматирование.** Опция позволяет окрашивать оценки в столбцах к красный, желтый и зеленый в зависимости от того высокая оценка или низкая.

![](_images-09/image332.png)

Нажмите на иконку плюса (+) чтобы создать условие для окрашивания и заполните следующие поля:

- **Столбец.** Столбец в таблице, в котором нужно окрашивать оценки.

- **Цветовая схема.** Цвет, в который нужно окрашивать оценки. Доступно три цвета:

  - Успех (зеленый)

  - Оповещение (желтый)

  - Ошибка (красный)

- **Оператор и целевое значение.** Диапазон значений, которые должны окрашиваться в нужный цвет. Например, если указать >3, то в колонке будут окрашены все значения начиная с 3.1.

![](_images-09/image213.png)

Чтобы оценки в колонке окрашивались в разные цвета, для данной колонки нужно добавить сразу несколько условий. Например, добавим для колонки «Управление конфликтами» такие два условия:

- \>2, успех

- ≤ 2, оповещение

Теперь все оценки больше 2 будут окрашиваться в данной колонке в зеленый цвет. А оценки, меньше и равные 2 будут окрашиваться в желтый.

![](_images-09/image263.png)

Находясь в дашборде таблицу в любой момент можно отредактировать: изменить строки и столбцы, кастомизировать внешний вид. Нажмите на три точки в правом верхнем углу таблицы и выберите «Редактировать график». Страница редактирования графика полностью дублирует страницу создания.

![](_images-09/image98.png)

## Диаграмма «Радар»

Рассмотрим создание радарной диаграммы. Такая диаграмма может показывать оценки по индикаторам компетенций в разрезе подразделений.

Выберите нужный датасет и тип графика «Радар». Нажмите «Создать график».

![](_images-09/image105.png)

На странице создания графика укажите следующие параметры:

### Вкладка «Данные»

1. **Название графика.** В названии графика укажите название конкретной компетенции, оценки по индикаторам которой нужно будет проанализировать. Например, «Клиентоориентированность».

2. **Измерения.** Аналитический срез, для которого нужно построить график. Перенесите в данное поле столбец «Подразделения».

3. **Меры.** Данные, которые нужно анализировать для построения графика. Так как нас интересуют оценки по индикаторам компетенции «Клиентоориентированность», в качестве мер укажем колонки с итоговыми оценками по индикаторам данной компетенции. Например, «Своевременно реагирует (I), Ориентирован на комфорт клиента (I)» и т.п. В поле «Агрегатная функция» укажем «AVG» (среднее).

![](_images-09/image142.png)

В настройках меры вы можете переименовать метку, чтобы при наведении на график выводилось понятное пояснение.

![](_images-09/image363.png)

4. **Фильтры.** Укажите здесь столбец «Дата завершения».

5. **Лимит строк.** Максимальное количество строк из датасета, которое должно быть обработано для составления графика. В данном примере количество строк равно количеству подразделений, по которым нужно построить аналитику. Укажите в данном поле то количество строк, которое равно или больше кол-ва подразделений в загруженном CSV-файле. Например, если подразделений 44, то укажите 50 строк.

![](_images-09/image129.png)

Нажмите на кнопку «Создать график» и перейдите во вкладку «Кастомизация». Обратите внимание на пункт «Настроить меры».

![](_images-09/image45.png)

6. **Настроить меры.** Данная опция позволяет получить корректную картину оценок по компетенциям на радарной диаграмме.

![](_images-09/image130.png)

По умолчанию для создания радарной диаграммы система принимает за максимальное и минимальное значение шкалы максимальную и минимальную оценку из имеющихся в датасете. Например, если самая минимальная оценка по индикатору 1.6, а максимальная — 3.3, то система построит диаграмму для шкалы от 1.6 до 3.3 и автоматически распределит все прочие оценки между этими делениями. Чтобы это сделать укажите для каждой меры (индикатора) максимальное и минимальное значение шкалы, по которой ее оценивали. В рамках текущего примеры оценка сотрудников производилась по 4-балльной шкале, поэтому для каждой меры укажем значения 0 и 4.

### Редактирование дашборда

Когда графики созданы, их расположение в дашборде можно менять через редактирование. Каждый график меняется в размере и может быть передвинут.

![](_images-09/image133.png)

![](_images-09/image238.png)

### Кросс-фильтрация

Благодаря тому, что графики созданы на основании одного датасета, в них поддерживается функция кросс-фильтрации. Это значит, что фильтр, примененный к одному графику, применяется ко всем.

Например, по клику на любое подразделение в сводной таблице, во всех остальных графиках также будет применен фильтр по этому подразделению. В итоге, на графиках будут показаны данные только по данному подразделению.

![](_images-09/image340.png)

![](_images-09/image7.png)

Кроме того, в дашборд можно добавить собственные фильтры. Для этого в левой панели нажмите на шестеренку (⚙️) > «Добавить или изменить фильтры».

![](_images-09/image311.png)

В открывшемся окне можно создавать различные типы фильтров.

![](_images-09/image398.png)

В качестве примера создадим фильтр типа «Временной интервал», чтобы просматривать графики за нужный нам период. Для удобства добавим для фильтра значение по умолчанию, чтобы фильтр автоматически показывал аналитику за нужный нам отрезок времени.

![](_images-09/image378.png)

Теперь в левой панели дашборда есть фильтр, который применяется ко всем созданным нами графикам.

![](_images-09/image55.png)
