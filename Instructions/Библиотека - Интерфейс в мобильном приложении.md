# Интерфейс в мобильном приложении

**Модуль:** [[Module - Библиотека]]

## Описание

Инструкция по использованию модуля «Библиотека» в мобильном приложении Битрикс24. Основные возможности модуля доступны на мобильных устройствах, за исключением функций администрирования.

## Ключевые функции

- Просмотр электронного каталога
- Скачивание файлов с электронными и аудиокнигами
- Контроль взятия и возврата бумажных экземпляров
- Предложение своих книг в Библиотеку
- Подписка на рассылку о новых книгах
- Оценки и отзывы на книги

## Предусловия

- Установлено мобильное приложение Битрикс24
- Модуль «Библиотека» установлен на корпоративный портал

## Граничные случаи

- Через мобильное приложение **нельзя** перейти в административную часть портала
- Добавление и модерация книг осуществляются только через десктопную версию Битрикс24
- Настройки модуля доступны только через десктопную версию
