# База знаний - О продукте

**Модуль:** [[Module - База знаний]]

## Описание

База знаний — электронное хранилище полезной информации для сотрудников. Позволяет хранить документы, инструкции, памятки, шаблоны, реквизиты и любые другие данные, которые сотрудники используют в работе. Выполняет функцию информационной поддержки, позволяя систематизировать материалы на удобной для компании основе.

## Ключевые функции

- Хранение внутренних материалов компании
- Систематизация по тематикам, отделам, видам документов
- Гибкие настройки доступа к просмотру и добавлению контента
- Совместимость с PHP 8.1+, MySQL, PostgreSQL
- Поддержка английского языка
- Многодепартаментность
- Доступ через десктопный браузер и мобильное приложение Битрикс24

## Предусловия

- Установленный модуль `ithive.knowledgebase`
- Корпоративный портал Битрикс24

## Шаги

1. Перейдите в пункт меню «База знаний» на корпоративном портале
2. На стартовой странице отображаются разделы первого уровня (слева) и последние добавленные статьи (справа)
3. Кликните на раздел для просмотра списка статей
4. Кликните на статью для просмотра содержимого

## Ожидаемый результат

Пользователь видит список разделов и статей, может просматривать содержимое в соответствии с правами доступа.

## Граничные случаи

- Без прав на чтение — разделы и статьи не отображаются
- Мобильное приложение не имеет функционала администрирования и наполнения
