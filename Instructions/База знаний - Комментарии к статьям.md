# База знаний - Комментарии к статьям

**Модуль:** [[Module - База знаний]]

## Описание

Комментирование статей с возможностью прикрепления файлов и оставления реакций. Авторам статей автоматически приходят уведомления о новых комментариях.

## Ключевые функции

- Комментирование статей
- Прикрепление файлов к комментариям
- Реакции на комментарии
- Уведомления авторам о комментариях
- Редактирование и удаление комментариев
- Отображение количества комментариев в колонке списка
- Отключение комментариев на уровне статьи
- Глобальное отключение комментариев на уровне модуля

## Предусловия

- Права на чтение статей
- Комментарии не отключены в настройках статьи или модуля

## Шаги

### Добавление комментария

1. Откройте статью
2. В блоке комментариев внизу страницы введите текст
3. При необходимости прикрепите файл
4. Отправьте комментарий

### Отключение комментариев на уровне статьи

1. При создании/редактировании статьи поставьте галочку «Запретить комментарии»
2. Сохраните статью

### Глобальное отключение комментариев

1. Перейдите в административную часть: `адрес_портала/bitrix/admin/`
2. «Настройки» > «Настройки продукта» > «Настройки модулей» > «Улей: База знаний»
3. Поставьте галочку «Отключить функционал комментирования статей»
4. Сохраните изменения

## Ожидаемый результат

Комментарии добавлены/отключены. Автор получает уведомления о новых комментариях.

## Граничные случаи

- При отключении комментариев на уровне статьи блок комментариев скрывается полностью (даже существующие)
- При глобальном отключении комментариев: поле «Запретить комментарии» исчезает, блок комментариев не отображается, колонка с количеством комментариев пропадает
- При повторном включении комментариев все существующие комментарии снова видны
- Для отключения комментариев нужны права не ниже «Добавление»
