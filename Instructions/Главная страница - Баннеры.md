# Баннеры

**Модуль:** [[Module - Главная страница]]

## Описание

Карусель баннеров в верхней части Главной страницы — информационные изображения и/или видео для привлечения внимания сотрудников к важным анонсам.

## Ключевые функции

> Перечисли основные функции/сценарии. Критично для AI-генерации чек-листов.

- **5 шаблонов баннеров:** Картинка, Картинка с текстом, Видео растянутое, Видео с текстом справа, Видео с текстом слева
- **Настройки блока:** выбор шаблона компонента (banners_sparkle / banners), количество баннеров (до 20), сортировка, поведение картинок
- **Таймер обратного отсчёта** на баннерах типа «Картинка с текстом»
- **Таргетирование** — показ баннеров определённым категориям сотрудников
- **Временной промежуток активности** — начало и окончание показа
- **Показ только новым сотрудникам** (в течение 30 дней)
- **Редактирование, деактивация и удаление** баннеров через слайдер управления

## Предусловия

- Права администратора или привилегированного пользователя с полным доступом к инфоблоку «Баннеры»
- Включённая административная панель (режим правки)

## Шаги

### Создание баннера

1. В режиме правки наведите курсор на блок с баннерами и кликните на шестерёнку (⚙️)
2. Нажмите «Создать» и выберите один из 5 шаблонов
3. Заполните поля:
   - **Название** (обязательное)
   - **Картинка/Видео** (JPEG, PNG / mp4, mpg, avi, wmv, mpeg, mpe, flv)
   - **Активность** (галочка для отображения на Главной)
   - **Начало/Окончание активности** (временной промежуток)
   - **Сортировка** (порядок в карусели)
   - **Время показа** (секунды, по умолчанию 10)
   - **Показывать только новым сотрудникам** (опционально)
   - **Ссылка сноски** (куда переходить при клике)
   - **Права доступа** (таргетирование)
4. Для шаблона «Картинка с текстом» дополнительно: заголовок, подзаголовок, кнопка, таймер обратного отсчёта
5. Сохраните изменения

### Редактирование/удаление

1. На слайдере управления нажмите бургер-меню (≡) рядом с нужным баннером
2. Выберите: Редактировать / Деактивировать/Активировать / Удалить

### Деактивация блока с баннерами

1. Перейдите в настройки модуля → вкладка «Компоненты»
2. В таблице «Центральный блок» снимите галочку в строке «Слайдер»
3. Сохраните изменения

## Граничные случаи

> Edge cases, ограничения, известные проблемы.

- Максимальная ширина баннера: 1229px
- При выборе «Растягивать по ширине, сохраняя высоту» изображение обрезается и центрируется
- При выборе «Растягивать по ширине и пропорционально увеличивать высоту» изображение не обрезается
- Если высота картинок отличается, карусель выглядит беспорядочно — рекомендуется готовить картинки одинакового размера
- Баннеры доступны по прямым ссылкам даже при скрытии из левого меню
