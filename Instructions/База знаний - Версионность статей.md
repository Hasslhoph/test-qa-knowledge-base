# База знаний - Версионность статей

**Модуль:** [[Module - База знаний]]

## Описание

Функционал сохранения до 50 версий для каждой статьи. Пользователи могут просматривать сохранённые версии, пользователи с правами на редактирование — сохранять, восстанавливать и удалять версии.

## Ключевые функции

- Сохранение до 50 версий на статью
- 4 режима работы версионности
- Просмотр версий всеми пользователями с правом просмотра статьи
- Восстановление версий (заменяет текущую опубликованную статью)
- Удаление версий (безвозвратно, не влияет на текущую статью)
- Настройка лимита версий на статью
- Настройка по умолчанию из раздела

## Предусловия

- Включённый функционал версионности в настройках модуля
- Права на редактирование для сохранения/восстановления/удаления версий

## Шаги

### Настройка версионности

1. Перейдите в административную часть: `адрес_портала/bitrix/admin/settings.php?lang=ru&mid=ithive.knowledgebase`
2. Выберите режим версионности:
   - **Использовать, создавать ли версию — из настроек раздела** — галочка по умолчанию НЕ активна, в разделе можно включить
   - **Использовать, создавать ли версию — исходное значение «Да»** — галочка по умолчанию активна
   - **Использовать, создавать ли версию — исходное значение «Нет»** — галочка по умолчанию НЕ активна
   - **Не использовать** — функционал отключён и скрыт
3. Установите **Количество хранимых версий на статью** (от 1 до 50)

### Сохранение версий

1. При создании/редактировании статьи включите галочку «Сохранить текущую версию статьи в истории версий»
2. Сохраните статью
3. В версии сохраняются: заголовок, содержание, визуальный редактор, файлы, привязка к разделам, теги

### Просмотр версий

1. Откройте статью
2. В нижней части слайдера перейдите на вкладку «Версии»
3. В списке версий можно настроить колонки (шестерёнка)
4. Кликните на бургер-меню (≡) для просмотра версии

### Восстановление версии

1. В списке версий: бургер-меню (≡) > «Восстановить»
2. ИЛИ на слайдере просмотра версии: кнопка восстановления
3. Версия заменит текущую опубликованную статью и станет основной

### Удаление версии

1. В списке версий: бургер-меню (≡) > «Удалить»
2. ИЛИ на слайдере просмотра версии: кнопка удаления
3. Версия безвозвратно удаляется, текущая статья не изменяется

## Ожидаемый результат

Версии статей сохраняются, просматриваются, восстанавливаются и удаляются в соответствии с правами.

## Граничные случаи

- При превышении лимита версий удаляется самая ранняя версия
- В версии НЕ хранятся: оповестить сотрудников, запретить публичную ссылку, запретить комментарии, галочка версионности
- При отключении версионности ранее созданные версии сохраняются и станут доступны при повторном включении
- Вкладка «Версии» видна всем пользователям с правом просмотра статьи
