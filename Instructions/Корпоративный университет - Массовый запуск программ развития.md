# Массовый запуск программ развития

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Массовый запуск программ развития через раздел «Сотрудники». Для выбранных сотрудников запускается одна и та же программа.

## Ключевые функции

### Права на массовый запуск

- Администратор
- Группа HR-отдела

⚠️ Обычные сотрудники и руководители имеют права только на одиночный запуск

### Варианты массового запуска

1. **По шаблону программы развития/адаптации** — доступен
2. **Свободный выбор компетенций** — доступен
3. **На основе карьерной траектории** — НЕ доступен

### Массовый запуск по шаблону

1. Раздел «Сотрудники» → отметить галочками → «Запустить программу»
2. Поля: Шаблон, Как определять руководителя, Отправить на согласование, Наблюдатели, Срок, Целевая должность/позиция, Следующая программа, Компетенции

### Массовый запуск со свободным выбором

- Без выбора шаблона — HR-специалист сам выбирает компетенции
- Поля: Как определять руководителя, Отправить на согласование, Наблюдатели, Срок, Компетенции

### Шаг 2: Настройка и проверка данных

**Фильтры:**
- Исходное выбранное количество сотрудников
- Сотрудники с имеющимися компетенциями
- Будет запущено
- Не будет запущено

**Причины блокировки запуска:**
- Уже есть программа по такому же шаблону
- Превышен лимит программ (по умолчанию 2)
- Сотрудник уволен
- Вручную отменён запуск

## Предусловия

- Права: администраторы и группа HR-отдела
- Созданные шаблоны программ

## Граничные случаи

- При массовом запуске невозможно выбрать для каждого сотрудника свою программу
- На втором шаге можно изменить руководителей, убрать освоенные компетенции, отменить запуск
