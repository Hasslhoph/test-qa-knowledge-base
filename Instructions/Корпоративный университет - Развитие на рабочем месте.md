# Развитие на рабочем месте

**Модуль:** [[Module - Корпоративный университет]]

## Описание

Раздел для развивающих действий, отличных от других типов обучения: видео, статьи, подкасты, внешние курсы, задания.

## Ключевые функции

### Типы развивающих действий

- Просмотр обучающих видео и фильмов
- Изучение статей в интернете
- Прослушивание подкастов
- Прохождение внешних онлайн-курсов
- Чтение книг с фиксацией выводов
- Выполнение обучающих заданий (обратная связь, дневник саморефлексии)

### Права доступа

- **Сотрудники** — общедоступные записи + свои записи
- **Администраторы и HR-отдел** — все активные записи

### Создание развивающего действия

- Опция «Создать» доступна: администраторы, HR-отдел, пользователи с правами на инфоблок
- Поля:
  - **Название** — конкретное или общее
  - **Описание / цель развития** — цель и польза выполнения
  - **Привязка к сотрудникам** — автоматически при добавлении из карточки
  - **Компетенции** — обязательные / рекомендуемые

### Настройка интерфейса

- Колонки: «Дата создания», «Кто создал», «Дата изменения», «Кто изменил»
- Колонки «Обязательно для» / «Рекомендуется для»

### Просмотр и редактирование

- «Открыть», «Отредактировать», «Удалить» через бургер-меню
- Удаление: HR-отдел, администраторы, «Полный доступ»
- Изменения отражаются только на будущих карточках

## Предусловия

- Права на создание: администраторы, HR-отдел, пользователи с правами на инфоблок
