# Оценка 360 - Настройки модуля

**Модуль:** [[Module - Оценка 360]]

## Описание

Основные настройки модуля выполняются в административной части портала. Шесть вкладок: Общие настройки, Шаблоны уведомлений, Настройка шкалы оценок, Настройка индикаторных оценок, Настройка прав доступа к отчёту, Настройка шаблона отчёта.

## Ключевые функции

### Вкладка «Общие настройки»

**Доступ к публичной части**
- Выбор отделов/сотрудников, которые видят модуль в публичной части
- Для каждого сайта можно настроить свой доступ

**Запуск и завершение оценки**
- Кол-во дней на сбор оценок (по умолчанию 14, min 7, max 30)
- Проверка данных перед созданием карточек
- Возможность завершить оценку досрочно
- Обязательность самооценки сотрудника
- Оповещение HR об истечении сроков

**Комментарии к компетенциям**
- Обязательность комментария для неоценённых компетенций
- Обязательность комментария при оценке ≤ N (по умолчанию 5)
- Обязательность комментария при оценке ≥ N (по умолчанию 9)

**Права на запуск и выгрузку**
- Группы пользователей для запуска/обработки/завершения оценки
- Группы пользователей для выгрузки оценок в CSV

**CRM и коллеги**
- Приглашение контактов из CRM к оценке
- Раздел soft/hard-компетенций
- Лимиты коллег и подчинённых для привлечения

**Прочие настройки**
- Воздержаться от оценки («0 - нет оценки»)
- Минимальный % заполненных оценок для сохранения (по умолчанию 85%)
- Лимит карточек при массовом запуске (по умолчанию 50)
- Вывод описания компетенций в окне оценивающего

**Права доступа к компетенциям**
- Группа HR-отдела для создания/редактирования компетенций

**Описание шкалы**
- Скрыть блок с описанием шкалы на слайдере оценки

### Вкладка «Шаблоны уведомлений»
- 7 типов уведомлений: CRM e-mail/SMS, согласование, приглашение, окончание срока, ознакомление сотруднику/руководителю

### Вкладка «Настройка шкалы оценок»
- Положительная и отрицательная шкала
- Количество делений, описание значений
- Уровни владения компетенцией

### Вкладка «Настройка индикаторных оценок»
- Процент пользователей для выбора индикатора (по умолчанию 50%)
- Описание значения «Нет проявления»

### Вкладка «Настройка прав доступа к отчёту»
- Анонимность данных: оценки, комментарии, профили респондентов
- Настройки по умолчанию для всех карточек

### Вкладка «Настройка шаблона отчёта»
- Описание отчёта и шкалы
- Автоматическое выставление итоговых оценок
- Цветовая индикация (пороги 1 и 2, HEX-коды цветов)
- Названия компетенций на диаграмме
- Лимиты сильных/слабых сторон, слепых пятен, скрытых возможностей
- Резюме: заголовок, обязательность
- Объединение анонимных колонок
- Скрытие проигнорировавших/отказавшихся респондентов

## Предусловия

- Права администратора портала

## Граничные случаи

- Настройки инфоблоков заполняются автоматически, не рекомендуется менять
- Сумма отрицательных весов не может быть больше суммы положительных
- Пороги цветовой индикации автоматически меняются при изменении размера шкалы
